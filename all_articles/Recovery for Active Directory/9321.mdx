## Question

How do I migrate Netwrix Recover for Active Directory v2.6 to a new server?

## Answer

To migrate NRAD v2.6 to a new server, please do the following:

* Get the location of the NRAD Database (SQLSERVER & DB name

  _OPTIONAL_: Take screenshots of each configuration tabs in the Web Console
* Download and install NRAD v2.6 on the new server (verify the file is unblocked before installing)

* During the install, connect to the previous NRAD Database

* After the reinstall is complete, open the Web Consol

  _OPTIONAL_: If screenshots were taken, compare them to the new configuration tabs
