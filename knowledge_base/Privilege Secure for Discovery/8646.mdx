# Symptom

When trying to login to the web interface, the following message is displayed:

Invalid Token

# Cause

The Google Authenticator TOTP token supplied is incorrect or is out of sync with the time server.

# Resolution

IMPORTANT: In order to perform troubleshooting steps, you must have the Administrator role

Troubleshooting steps:

* Ensure that your device time matches global UTC/GMT time, reference sites

  https://www.timeanddate.com/worldclock/timezone/ut

  https://www.time.gov/
* Confirm that the correct Google Authenticator token was used when attempting logon

* Log in using the Local Account

* Navigate to the __Configure__ > __Users & Groups__ pane.

* Reset the token and log in again. Read more in the Configuration – Resetting MFA Tokens ? v2.20 article.
