Issue: Summary:
This Hotfix addresses two primary issues:

  1. Threats created via investigations not properly evaluating hosts used in the rule builder filter controls.
  2. The ransomware job can fail if the Threat Manager server does not have internet access.
This update also has some minor performance improvements and quality of life improvements.
Behavior before Fix:
  • Custom Threats created via investigations could sometimes improperly evaluate hosts.
  • Ransomware Job could fail if the Threat Manager server does not have an internet connection
  • AD Sync performance impacted when a large number of contacts are stored in Active Directory
  • SSL Configuration susceptible to failure when invalid characters are copy/pasted with the thumbprint
Behavior after Fix:
  • Custom Threats created via investigations properly evaluate hosts.
  • Ransomware job works appropriately in "offline" / non-internet connected environments
  • Improved performance of AD Sync when a large number of contacts are stored in Active Directory
  • SSL Configuration handling for invalid characters in the thumbprint ID of the certificate
