Issue: Summary: Database event maintenance has an exception
Behavior before Fix:
When user press Save button application must validate that each selected item must have Retention Period and Operation set to some values. During this validation it was possible that Data Set for grid is not filled with a data yet which caused an exception
Behavior after Fix:
Added logic to validate if Data Source is filled with a data before execution validation for each row.
