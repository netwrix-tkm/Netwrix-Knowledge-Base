Hotfix Patch 2.2.204 for Threat Manager 2.2

Summary: This Hotfix addresses an issue with the Threat Manager UI and Action Steps when configuring threats and when executing threat response.

Issue: Summary:
This Hotfix addresses an issue with the Threat Manager UI and Action Steps when configuring threats and when executing threat response.
Behavior before Fix:

  • Rare Instances of 500 Errors when creating investigations and utilizing the Threat Configuration Page
  • Actions could fail to access Threat Manager when certain SSL configurations are enabled
  • Occasional failures to manually execute an action step when logged in as a non-admin user
Behavior after Fix:
  • Additional Error Handling to prevent issues when saving investigations and Threat Configuration changes
  • Resolved issues with Action Step PowerShell and SSL
  • Improved Threat Response UI / API Error Handling to resolve manual threat response execution error
