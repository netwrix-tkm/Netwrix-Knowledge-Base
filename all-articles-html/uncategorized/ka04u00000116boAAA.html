HTTP error 404.3 in Password Manager

I recieve 404.3 error message on Password manger portal. The details say: The page you are requesting cannot be served because of the extension configuration. If the page is a script, add a handler. If the file should be downloaded, add a MIME map.
The error means that even though the files are in place and accessible, IIS service cannot process them correctly, because the required extensions are not installed.
To resolve the issue install ASP and ASP.NET (3.5) IIS features. Refer to the KB2125 for detais on how to enable IIS features.
