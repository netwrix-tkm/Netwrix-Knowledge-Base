Hotfix: SIWindowsAgent_5.2.0.256

Summary: SI FS Activity for SA Policy Issue - SI Resolves FQDN to Shortname

Issue: Summary:
SI FS Activity for SA Policy Issue - SI Resolves FQDN to Shortname
Behavior before Fix:
When configuring 'File Statem StealthAUDIT' policy SI Agent creates corresponding section in the SBTFileMon.ini file with HOST parameter set to NetBIOS name of the computer.
Behavior after Fix:
When configuring 'File Statem StealthAUDIT' policy SI Agent creates corresponding section in the SBTFileMon.ini file with HOST parameter set to FQDN name of the computer if it is available. If FQDN is not available then it will use NetBIOS name as it was before the fix.
