'Error in Active Directory Action Module: Cannot insert the value NULL into column 'joinRowGUID'

Summary: 'Error in Active Directory Action Module: Cannot insert the value NULL into column 'joinRowGUID'

Issue: Cause: This can happen when the input table has a column named "rowGUID" that contains a NULL value. It should not happen via normal StealthAUDIT operation, as the GUIDs are generated automatically by SA Core for every query and are not allowed to be null. If the "rowGUID"column is missing completely, then SA defaults to an empty GUID which will still work. As such, this is generally caused by changes to the tables outside of standard SA operation.

�

Instructions: Resolution: Fix the schema by populating GUIDs where NULL values exist.
