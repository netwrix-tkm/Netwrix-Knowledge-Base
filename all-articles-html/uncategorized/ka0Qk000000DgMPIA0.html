Hotfix: SAM_5.0_032

Summary:

SAM has stopped monitoring Netapp

Behavior Before Fix:

SAM netapp service crashes when SAM cannot find external engine to connect to. "Enable and Connect" requires external engine name to be "StealthAUDITEngine".

Behavior After Fix:

SAM now logs error and does not crash when external engine is named differently.
