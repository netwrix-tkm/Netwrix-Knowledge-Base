Summary:

SI Agent - add tracking / metrics for cache hits / misses

Behavior before Fix:

New metrics need to be added to troubleshot RBC issues

Behavior after Fix:

Added two counters for internal queues in the agent:

AccountInfoBySid

ByHostName

Each of these counter logs on WARN level the number of attempts which were resolved and picked from queue cache for accounts or computers.

ImMemoryCacheCounter 'ComputerByHostNameCounter': Attempts to query: 2. Picked from the cache: 1
