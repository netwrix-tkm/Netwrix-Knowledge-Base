Enrollment failed: cannot save secret

When enrolling a user, the following error occurs: Enrollment failed: cannot save secret
Password Manager is trying to create a secrets.bin file in the Password Manager installation directory. The account used for the Password Manager service does not have proper access writes to the Password Manager installation directory.
  1. Provide the Password Manager service account rights with Modify and/or Write to the Password Manager installation directory.
  2. Verify that the account is a domain admin and a local administrator on the Netwrix server where Password Manager is installed.
  3. Verify the Administrators group on the Netwrix server has Modify/Write access to the Password Manager installation directory.
