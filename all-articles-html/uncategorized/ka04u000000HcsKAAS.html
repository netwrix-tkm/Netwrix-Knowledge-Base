Summary: Active Directory group lockdown policy not blocking consistently

Issue: Summary:� Active Directory group lockdown policy not blocking consistently
Before fix: it happens because list of objects for this policy is too big. Semicolon separated list of values in blocking rule can have not more than 1024 characters but in this policy it is more than 7000 characters. Therefore this list passed to AdMonitor as truncated and therefore some groups are blocked and some are not.
After fix: Increased buffer size of INI-parser to 30MB
