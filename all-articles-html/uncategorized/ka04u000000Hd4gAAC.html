Hotfix: AIC_7.2_SP1_007

Summary: Deleted Items in AIC should not be searchable

Issue: Summary:
Deleted Items in AIC should not be searchable

  • Issue: Delete objects are searchable in the AIC, and if you select one, the AIC will throw an error: "Unable to find the ObjectSid in AD".
  • Fix: Now deleted trustees won't be searchable in the main page search bar (or in the ownership page, for groups), they won't autocomplete, and if the user just types the full name no action will be done and no error thrown.
