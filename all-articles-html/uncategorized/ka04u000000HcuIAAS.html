Summary: SI Agent Causing Failing Backups

Issue: Summary: SI Agent Causing Failing Backups
Behavior before Fix:
When Lsass.exe audit mode is enabled SI Agent 'thinks' that lsass is protected and tries to unprotect it to be able to inject into lsass and protect is back with very high restrictions for signature level which make impossible to load any Micsosoft library by lsass process.
Behavior after Fix:

  1. Now SI Agent to decide if lsass is protected in addition to 'IsProceted' flag will also analyze Protection.Type value and will assume lsass is not protected if Protection.Type is '0'.
  2. If lsass IS protected then SI Agent will restore original signature level values and as a result will set lsass protection parameters to original state.
