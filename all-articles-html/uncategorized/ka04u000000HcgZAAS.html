Hotfix: SA_10.0_072

Summary:

SMPC-10960 \ Escalations 56968: Report issue showing 1000 lines

Issue/Request: In reports configuration, closing the dialog by pressing X instead of OK, was resetting the 'Limit maximum number of displayed rows to' option to 1000 rows. Also, unchecking this option and then selecting OK, was setting the value to a grayed out 0.

Fix/Enhancement: Now hitting Cancel or X in the report configuration dialog, will keep the above setting with the saved value. Also, when unchecking that option and hitting OK, the previously set limit value will show grayed out, and report will generate with unlimited rows.
