Choose Alternate Database Location for StealthRECOVER

Summary: Use a simple command line script to modify the StealthRECOVER database location.

�

Issue: The basic StealthRECOVER 1.2 installation process does not include functionality to modify the default database location.� Following the directions below, you can utilize the .exe to change the database location using a simple command line script.


Instructions: The default save location for the SR database files is:
C:\ProgramData\STEALTHbits\StealthRECOVER\db *** Note it's ProgramData, not Program Files. ***
�
In order to change this location, complete the following:

  1. Copy the StealthRECOVER.exe to the server desktop.
  2. Run the following command script as Administrator.
    1. StealthRECOVER.exe /I DatabaseFolder=[PATH]
    2. Ensure [PATH] (without brackets) reflects the folder destination of the desired database location.

�
I found this to be useful as one of my clients wanted to move all SR files off of the C: drive and onto a dedicated SR drive.� This information is now in the SR documentation.
