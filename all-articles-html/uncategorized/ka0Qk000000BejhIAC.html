Error 26201: Error Installing Package

IMPORTANT: This is a work-in-progress article. The product R&D team has yet to review some or all of the information below.

Symptom

While installing PingCastle on Windows Server, the installer generates the following error message:

Error 26201
The installer has encountered an unexpected error installing this package

Cause

These errors may be caused by any one of the following:

  • Installation process started without Local administrator rights

  • The account used to perform installation does not belong to the appropriate AD groups
  • Group policies or security software blocking the installation
  • SQL server is not configured properly

NOTE: This is important information to highlight.

Resolutions

To address these causes, refer to the following steps.

  • Configure antivirus exclusions in your environment. Refer to the following article for additional information: Antivirus Exclusions for Netwrix Auditor.

  • Refer to the following steps in case excluding folders from the antivirus scans didn't help:

    1. Review the registry permissions for the affected SID, and compare them with the unaffected user SID in the same server.

    2. Review the list of users in Computer\HKEY_LOCAL_MACHINE\SOFTWARE\Microsoft\WindowsNT\CurrentVersion\ProfileList\.
