Summary: Agent fails when Syslog enabled

Issue: Summary:
Agent fails when Syslog enabled
Behavior Before Fix:
When syslog buffer file was locked, this would cause service to crash.
Behavior After Fix:
When syslog buffer file is locked, we now handle this gracefully and service doesn't crash anymore.
