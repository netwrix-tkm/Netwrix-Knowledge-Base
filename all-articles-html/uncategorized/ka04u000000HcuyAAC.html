Hotfix: AIC_7.1_SP1_007

Summary: Issue: Before there was no means to copy content from the AIC reports' grids. Fix: A new option for copying content to the clipboard was added to the 'Options' menu in Resource Audit. - Issue: Trustee Audit / Permissions reports was showing deleted resources. Generally clients don't want to see permissions for things that no longer exist. - Fix: Deleted resources (DeletedUSN not Null) are now filtered out of the report.

Instructions:

  1. Uninstall current version of AIC if it is already installed
  2. Install version of AIC included with this hotfix
  3. (For SMPC-1131: Copy/Paste support): In Resource Audit navigate to any report that has the Options menu, select the desired rows to be copied, click on the Options menu: Options > Options > Copy Selected Rows, this will copy the selected rows' contents to the clipboard.
