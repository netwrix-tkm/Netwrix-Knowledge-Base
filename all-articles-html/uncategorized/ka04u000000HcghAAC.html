Issue: Summary: Capital One Enhancement - Support use of domain name in additional agents filter of FS policy

SI supports choosing servers by Host name in Additional Agents filter.
Behavior after Fix:
SI supports choosing servers by Domain name in Additional Agents filter.
In this case SI will send this policy too all agents in the domain without need to select each server manually
