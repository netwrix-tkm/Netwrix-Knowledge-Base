Hotfix: AIC_9.0_005

Summary: SMPC-8015: AIC - Support Processing Entitlement Review Changes on Deleted Users. . Issue/Current behavior: A membership entitlement review where a member of the reviewed group had been deleted after the review was responded by the owner, was popping up an error when the Admin accepted the changes "An application error has occurred please close this window and look at the application log file". . Fix/Enhancement: Now changes are processed gracefully even if the user has been deleted. No error are reported.
