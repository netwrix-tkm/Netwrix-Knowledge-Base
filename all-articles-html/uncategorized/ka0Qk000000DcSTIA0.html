Hotfix Patch 2.5.438 for Threat Manager 2.5

Description:

This hotfix addresses or implements the following:

  • Resolved some compatibility issues with SI 7.2
  • Improved Investigation performance for large data sets
  • Issues with Ransomware Job failing on Non-Internet connected servers
  • Issues with Ransomware (and other threats) not respecting their threat detection configuration settings
  • Issues with Ransomware Threat Rollup
  • Issues with Exporting Playbooks to JSON
  • Issues with Kerberoasting not resolving perpetrator accounts
