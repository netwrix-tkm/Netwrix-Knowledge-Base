Issue: Summary:
Summary: This hotfix corrects an error in the Web Console when using the Vault service to store profiles..
Issue/Request: When using the Vault service to store a profile, users would not see any published reports in the Web Console.
Fix/Enhancement: Published reports will be visible in the Web Console when using the Vault Service to store profile information. Note that the Web Server Service must be running under a Access Analyzer administrative account, and not as a Local Service.
