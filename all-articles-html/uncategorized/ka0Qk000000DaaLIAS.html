Support Process - Enabling Netwrix License Creation

Note requires Hub 7.0.1.22 or above


1. Add regkey if not present:

HKEY_LOCAL_MACHINE\SOFTWARE\NNT

2. Add string value:

Name: LC
Data: YpYnVpbGRpbmchKClTUEFDRTIwMTU=

3. Copy the NNT.Hub.Licensing.dll file (which is attached to this page, use the 7.2.1.19 version) into the Gen7 hub bin folder, typically at: C:\inetpub\wwwroot\Change Tracker Generation 7 Hub\bin

4. Open command prompt window as we will need to add the LicenseCreate permission to the admin user in mongo using the following commands:

> cd "C:\Program Files\NNT Change Tracker Suite\Gen7\MongoDB\bin"

> mongo.exe

> use NNTHubService

> db.UserAuth.update( { UserName: "admin", RefId: 0 }, { $addToSet : { Permissions: "LicenseCreate" }})

5. After this restart IIS, login as the admin user and the admin > licenses page should show a �Create New License� section


Extra: If you want to add multiple permissions, this is the query:

db.UserAuth.update( { UserName: "tom", RefId: 0 }, { $addToSet : { Permissions: { $each: [ "EventManage", "PlannedChangeManage" ] } }})
