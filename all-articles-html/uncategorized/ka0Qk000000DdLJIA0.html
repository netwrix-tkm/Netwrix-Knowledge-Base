How To: Remove Delete option from the GroupID Portal

Applies To:

GroupID 11

Business Scenario:

We have Users who are members of many critical groups and use the GroupID portals on a daily basis. We do not want them to accidentally delete any groups that may lead to issues within our organization. Is there a way to disable/remove the option to delete groups so that only administrators have the right to perform this action, and not the users?

Solution:

Yes, it is possible by making changes to the design of the GroupID portals. We can hide the option to delete from users accessing the groups.

Please follow the steps provided below to implement the restrictions based on the security roles:

Steps:

  1. In the GroupID Admin Center, go to the Applications tab in the left navigation bar.
  2. Click on the Ellipses button for the portal you wish to make changes to, and then click on Settings.

  1. Once the portal settings open, click on the Identity Store name under Design Settings for which you want to make the design changes.

  1. Click on the Toolbars button and select �Groups� from the Select Toolbar Type dropdown list.
  2. Select �Delete� from the list that appears corresponding to the selected toolbar and click on the "Pencil� icon.

  1. A new window will appear. Locate the �Visibility Role� dropdown list and change the value from �Users� to �Administrator� security role. When "Users" is selected, it allows the visibility of the button to every role that has a priority level greater than Users. If you want to disable this feature for everyone, simply select "Never," and the delete button will be hidden from all the Security Roles available within your environment.

  1. Once the changes are made, make sure that you save the changes by clicking on the "Save" icon in the Designs tab.

  1. Refresh/relaunch the GroupID Portal and verify by going into the properties of any group. The Delete option will no longer be available in the toolbar.

Similarly, if you want to hide the delete option from the Group Search or My Groups, select the �Groups Search� toolbar type in the step 3 and repeat steps 4 to 7.
