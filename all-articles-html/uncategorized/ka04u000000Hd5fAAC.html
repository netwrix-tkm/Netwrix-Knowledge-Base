Summary: Fix for incorrect users being collected when Domain groups are used as SQL logins.

Issue: Summary:
Fix for incorrect users being collected when Domain groups are used as SQL logins.
Behavior Before Fix:
When a Domain group is used as a SQL login, audit collection was using a non-unique identifier to identify the actions of the members of that group.� This caused the collected audits for all users in that group to show up as the first user who logged in from that group.
Behavior After Fix:
The collected audit now shows the correct member of the domain group.
