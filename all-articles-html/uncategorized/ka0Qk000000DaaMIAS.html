Hotfix: Patch 2.2.199 for Threat Manager 2.2

Summary: This Hotfix addresses an issue with the Access Analyzer Unified Interface when logging into Threat Manager.

Issue: Summary:
This Hotfix addresses an issue with the Access Analyzer Unified Interface when logging into Threat Manager.

  • Due to Updates to the various authentication components in Threat Manager 2.2, logins via the Access Analyzer unified interface will often fail to properly authenticate.
Behavior after Fix:
  • Full Login experience with the Access Analyzer Unified Login integration is restored.
