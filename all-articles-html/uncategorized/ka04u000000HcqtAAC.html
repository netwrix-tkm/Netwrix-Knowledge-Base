Hotfix: SIWinConsole_6.0.0.420

Summary: Stealth Intercept | discrepancies between data

Issue: Behavior before Fix:
When you open Event Details window it not only shows selected event but also allows you to navigate to other events using mouse scroll-wheel and arrow buttons at the button of the window. Therefore, depending on how data source is sorted and grouped in the grid view the order of events can change and sometimes will not correspond to the order of events in the card view.
Behavior after Fix:
Code is modified to generate new data source for Card View which contains entries sorted by index in the grid view instead of old approach when these two views shared the same data source.
Changes since Last Hotfix:
