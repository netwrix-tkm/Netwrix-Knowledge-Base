Comments:
SA_ADInventory_Exceptions script uses computers view samaccount name (netbios name) to cross reference host list.
If the host list is in format FQDN then this does not match the join to the computer view, and the data is not inserted into the AD Exceptions table.
I was able to work around by adding an "or" operator to the join for Dnshostname on computer view which would match the FQDN.
