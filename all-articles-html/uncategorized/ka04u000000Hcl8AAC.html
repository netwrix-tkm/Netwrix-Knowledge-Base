Hotfix Patch 2.1.320 for StealthDEFEND 2.1

Summary: This Hotfix addresses an issue with the AD Sync utilizing an excessive amount of memory. Customers experiencing problems with the StealthDEFEND Active Directory Service and memory usage should apply this patch.

Issue: Summary:
This Hotfix addresses an issue with the AD Sync utilizing an excessive amount of memory. Customers experiencing problems with the StealthDEFEND Active Directory Service and memory usage should apply this patch.
Behavior before Fix:

  • Active Directory Service utilizes and does not release large amounts of ram even in some small/medium-sized environments.
Behavior after Fix:
  • Resolved "Memory-Leak like" behavior of the Active Directory Service
