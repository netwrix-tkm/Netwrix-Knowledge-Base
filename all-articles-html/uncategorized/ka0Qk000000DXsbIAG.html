How To: Change the Display Name Format to First.Last in GroupID Portal

Applies To:

Netwrix GroupID 11 � Portal

Business Scenario:

By default, GroupID formats a user's display name as Last, First during user creation through the portal. However, in some environments, there may be a requirement to use the First.Last format instead.

This article explains how to make this customization by modifying the underlying JavaScript file used by the GroupID Portal.

Solution:

To change the display name format, you need to make a code-level modification to one of the portal�s script files.

Note: This change should be applied carefully and ideally tested in a non-production environment first. Always back up original files before making any modifications.

Steps to Modify the Display Name Format:

  1. Navigate to the following directory on your GroupID server:
    C:\Program Files\Imanami\GroupID 11.0\GroupIDPortal\Inetpub\<PortalName>\Web\wwwroot\Scripts

  2. Locate the files named:

    • CreateWizard.js

    • CreateWizard.min.js

  3. Rename the file CreateWizard.min.js to CreateWizard.min.backup.js.

  4. Open CreateWizard.js in Notepad++.

  5. Search for the function named getFullName, or go directly to line 392 in Notepad++.

  6. Replace the existing function:

    function getFullName(firstNameText, lastNameText) {

    var comma = ",";

    if (firstNameText === "" || lastNameText === "") {

        comma = "";

    }

    return lastNameText + comma + firstNameText

}

    with the following version:

    function getFullName(firstNameText, lastNameText) {

    var comma = ".";

    if (firstNameText === "" || lastNameText === "") {

        comma = "";

    }

    return firstNameText + comma + lastNameText

}

  7. Save the file.

  8. Rename CreateWizard.js to CreateWizard.min.js.

  9. Open Command Prompt as Administrator and run the following command:

    iisreset

  10. Once the IIS reset is complete, log in to the portal and try creating a new user.
    The Display Name will now appear in the First.Last format.
