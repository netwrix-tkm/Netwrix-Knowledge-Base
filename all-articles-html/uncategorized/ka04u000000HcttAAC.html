Writing PID into Lock Files in 8.0

Summary: When running a job or scheduled task in StealthAUDIT, open the Running.lck file in the Job's folder to view the Process ID (PID) associated with the job/task.

Issue: Sometimes customers might run multiple scheduled tasks at the same time. If one of the processes hang, then it can be difficult to know which one to kill.

Instructions:

  1. Open Task Manager and click on the processes tab, if you don't have a PID column add it by clicking view > select columns.
  2. Open the jobs folder for the task/job that is running. There is a lock (.lck) file named "running".
  3. Open it in notepad to view what the PID associated with the job/task is, and kill that process using task manager.
