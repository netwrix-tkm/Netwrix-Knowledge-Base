Hotfix: SA_8.1_106

Summary: Currently, StealthAUDIT does not support using TLS v1.2 for communicating with a database server. As this standard is set to replace earlier versions of TLS in the coming years, customers are starting to transition to 1.2. We are adding compatibility with the TLS 1.2 standard to SA v8.0 and all subsequent versions.
