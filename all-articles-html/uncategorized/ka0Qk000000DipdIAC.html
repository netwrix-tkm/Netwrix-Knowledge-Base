Issue: Title:
Hotfix Patch 2.0.435 for Threat Manager 2.0
Summary:
This Hotfix addresses a variety of issues relating but not limited to:

  • Additional SAM Event Handling
  • High Risk Permission Threat & Threat Response Improvements
  • Active Directory Sync Performance Improvements
  • Active Directory Sync Error Handling
  • Missing Client Information in SIEM/Action Response
Behavior before Fix:
  • In some rare cases events from SAM would not be processed by Threat Manager due to some unique variability in the client data.
  • Some rare instances of the high risk permission threat would not gather the correct events or handle threat response properly.
  • Active Directory Performance issues when syncing Token Groups
  • Active Directory Stability issues on minor network interruption, and certain AD data formats
  • SIEM / Playbooks for AD Threats would sometimes be missing client information
Behavior after Fix:
  • DEFEND properly processes SAM events in different data types in.
  • High Risk Permission threat handles more unique scenarios.
  • Greatly increased performance for the Active Directory Sync.
  • Additional error handling and logging for the Active Directory Sync.
  • SIEM / Playbook Responses for AD Threats will properly translate outgoing client information
