Does Change Tracker integrate with ServiceNow?

Problem

ServiceNow is a cloud provided IT Service Management solution based around the ITIL framework. Organisations use ServiceNow, sometimes refereedï¿½to as 'Snow', for change management and so an integration into Change Tracker, in order to make Change Tracker aware of environment changes is an obvious link.

Solution

Yes!! Change Tracker has a REST API which can be used to integrate with ServiceNow. Note: Currently (9th June 2017) NNT has not integrated with ServiceNow. While absolutely possible any integration work would be a service lead engagement involving the NNT development team.
