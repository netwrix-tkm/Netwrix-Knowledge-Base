Hotfix: SA_7.1_034

Summary: Add configurable option to determine whether to zip emailed reports With the release of 7.1 GA we changed the default behavior with emailing reports to always zip. This fix changes that default behavior to a configurable option to zip pdf/csv files once they get larger than a certain threshold.
