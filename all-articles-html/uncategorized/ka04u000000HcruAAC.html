Hotfix: SD_1.2_011

Summary: This Hotfix addresses an issue with the Threat Configuration and Actions page when logged in as an Active Directory Login. When accessing the application with an Active Directory login, it has been discovered that some of the permissions tied to the role can occasionally be lost by the application. This causes some page elements such as Threat Configuration and Action Groups to be prevented from loading for users with properly scoped access.
