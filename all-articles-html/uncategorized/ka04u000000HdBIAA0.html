ITSM - REST API CALLS MADE

This article contains a catalog of API calls made by our service to the ITSM products. There are many ITSM products which can be synchronized with Change Tracker - they all make similar but subtly different REST calls.

The color-coded parts are sourced either from the configuration, or calculated by the service as it runs.
