How To: Hide Synchronize, Reports, and Entitlements Tabs from the Portal

Applies To:

Netwrix Directory Manager 11 � Portal

Business Scenario:

In the Directory Manager User Portal, the Synchronize, Reports, and Entitlements tabs are displayed on the navigation bar. Organizations that do not want to provide users access to these options can hide them.

Directory Manager allows administrators to set an access level for each option on the navigation bar, determining which user roles can view specific links.

In this article, we will walk through the process of hiding the Reports, Entitlements, and Synchronize tabs from the navigation bar.

Solution:

To restrict access to these tabs, you need to modify their access level settings in the Admin Center.

Steps to Hide the Tabs from the Navigation Bar:

  1. Go to the Admin Center.

  2. Navigate to Applications.

  3. Open the Settings of the Portal where you want to apply the changes.

  1. Go to Design Settings and expand Identity Store.

  2. Select Navigation Bar.

  3. Click the Dropdown List and select External Links.

  1. Edit Entitlements.

  2. In the Access Level list, select a security role:

    • The Entitlements link will be visible to users of this role and any roles with a higher priority.

    • To hide the link from all users, select Never.

    • If you want only Administrators to see it, choose Administrator (users in roles below Administrator will not have access).

  1. Click OK and Save the changes.

  2. Log in to the Portal to verify the changes.

  1. Repeat the same process for the Reports and Synchronize tabs:

    • Edit each one individually.

    • Change the Access Level setting as needed.

Once configured, users without the required access level will no longer see the Reports, Entitlements, and Synchronize tabs in the portal.�
