Hotfix: SA_9.0_001

Summary: Summary: This hotfix corrects an issue where the SCRIPT data collector cannot be found even though it is present in the installation. Issue/Request: Trying to run a job that uses the SCRIPT data collector results in an error "The 'SCRIPT' data collector has not been installed.' even though the SCRIPT DC is present in the installation. Fix/Enhancement: Running Jobs including the SCRIPT data collector will not produce an error.
