Scope FSAA SDD Scans by File Type

Summary: FSAA SDD Scans can be scoped by file type to improve performance

Issue: FSAA scans can be scoped by file type, which can be particularly useful when doing things like OCR scans. Currently, this can only be done through the job xml, but a UI will be added with SAFS-2426.
�

Instructions: Modify the SelectedTypesList parameter in the job xml to remove particular file types or whole key values.

DOCUMENTS*.doc;*.docx;*.msg;*.odt;*.pages;*.rtf;*.wpd;*.wps;*.abw;*.bib;*.dotx;*.eml;*.fb2;*.fdx;*.gdoc;*.lit;*.sig;*.sty;*.wps;*.wpt;*.yml;*.tex;*.pdf;

SPREADSHEETS*.csv;*.xlr;*.xls;*.xlsx;*.gsheet;*.nb;*.numbers;*.ods;*.qpw;*.sdc;*.wks;*.xlsb;*.xltm;*.xltx;*.aws;*.fods;*.ots;*.rdf;*.sxc;*.uos;*.xlsm;
