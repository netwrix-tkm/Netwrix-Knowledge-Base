Summary: FSAA - Misleading error message being thrown when retry is successful.

Issue: Summary: �FSAA - Misleading error message being thrown when retry is successful.
Before Fix: When a host failed to start on one of the proxies in the proxy list but ended up succeed on another, FSAA would still throw an error message for that host failing to start on a proxy.
After fix: FSAA will only throw an error message for a host after it has exhausted all the proxies in host list.
