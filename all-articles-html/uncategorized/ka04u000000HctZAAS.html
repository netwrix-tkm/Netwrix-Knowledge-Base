Hotfix: SIEntManager_4.1.0.568_SI-3311

Summary: Object Collection with Groups does NOT report on Group Object Changes

Issue: Title: SIEntManager_4.1.0.568_SI-3311
Summary: �Object Collection with Groups does NOT report on Group Object Changes
Before fix: When user adds some entry to the collection which wasn't linked to any policy and then modifies this collection, then Enterprise Manager will not notify Agent about changes in the collection. So, if after that this collection will be linked to some policy then this policy will start working correctly only after next re-registration of the Agent during which Agent will get updated collection.
After fix: Enterprise Manager will always send notifications to Agent about modified or newly created collection even if this collection is not linked to any policy.
