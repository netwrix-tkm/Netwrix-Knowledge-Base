Hotfix Patch 2.2.249 for Threat Manager 2.2

Summary: This Hotfix addresses several issues with Threat Manager, primarily regarding reducing false positives threats and improving the Event Message service performance and memory utilization.

Issue: Summary:
This Hotfix addresses several issues with Threat Manager, primarily regarding reducing false positives threats and improving the Event Message service performance and memory utilization.

Issues Addressed:

  • False Positive Threat Reduction - when specific Sub Domain configurations are used, Exposed Admin Credentials and DCSync threats can trigger inadvertently.
  • Event Message Service - Memory Utilization Fixes
  • Event Message Service - Threat Insertion and General Performance Improvements
  • Investigations - Removing File Rules from Rule Builder can invalidate Investigation
  • Threat Exclusions - Excluded Threats and Threat Events permanently stored in the database which could lead to database bloat when many exclusions are in place.

Lee Berg
Affected Versions:
