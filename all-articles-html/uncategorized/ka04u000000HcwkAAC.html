Instructions:

  1. Unblock the hotfix zip file in the windows property dialog, if an unblock button exists there.�
  2. Close all instances of StealthAUDIT (check task manager under processes for all users)
  3. Stop the "StealthAUDIT Web Server" service.
  4. Update StealthAUDIT.exe in the %SAInstallDir%
  5. Update all Web files in your %SAInstallDir%Web folder
  6. Update all PrivateAssemblies in your %SAInstallDir%PrivateAssemblies folder.
  7. Update all Templates in your %SAInstallDir%Templates folder.
  8. Restart the "StealthAUDIT Web Server" service.
