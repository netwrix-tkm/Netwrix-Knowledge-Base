Hotfix: SA_8.1_092

Summary: Core: Exporting data to csv should correctly escape all characters in report exports

Issue: Summary: Core: Exporting data to csv should correctly escape all characters in report exports
Before fix: Data exported to CSV preserved newline characters, which created display problems in Excel.
After fix: Newlines will be preserved within cells in Excel. Text viewers may treat these differently (e.g. Notepad does not preserve newlines at all).
