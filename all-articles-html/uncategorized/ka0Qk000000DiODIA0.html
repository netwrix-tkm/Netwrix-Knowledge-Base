Hotfix: SA_10.0_074

Summary:

SMPC-11316 \ Escalations 58158: SQL Best Practices report, "Publish report failed with error message: Cannot save a PDF with no pages"

Issue/Request: When a job XML file shows the charts color in name format as opposed to hexadecimal format, emailing the report in PDF format was failing.

Fix/Enhancement: Now the PDF document is created and attached to the email for the above scenario. In addition, if the report configuration is resaved, the color format will be changed to hexadecimal.
