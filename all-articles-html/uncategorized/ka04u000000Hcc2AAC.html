SI-4469 60748 - Multiple agents go into status "unknown"

SI-4404 58894 - domain controllers were inactive in SI

Behavior before Fix:

Agent EM communication was based on a mix of asynchronous and synchronous code which also required locking of streaming grpc calls

Behavior after Fix:

Agent EM communication is based on asynchronous code only and queuing of streaming grpc calls instead of locking, which should result in much better performance in case of hundreds of connected agents
