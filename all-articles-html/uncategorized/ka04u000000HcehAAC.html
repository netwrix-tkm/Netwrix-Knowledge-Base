Hotfix: SA_10.0_088

Summary: Dynamic Host List Aren't being Recreated

Issue/Request: When deleting dynamic host lists from Host Management, running the job interactively/scheduled was not recreating it, and hosts were not updated.

Fix/Enhancement: Dynamic Host lists (with auto-recreate option enabled) are correctly being recreated and hosts updated when job is run interactively or through scheduled tasks. As part of the changes only 1 dynamic host list can be created per results table.
