Instructions: http://downloads.stealthbits.com/access/files/SAHotFixes/7.0/SA_7.0_SP1_018.zip

  1. Install "Microsoft Visual C++ 2005 Service Pack 1 Redistributable Package MFC Security Update" on the SA console
  2. Drop included modules into the DC folder onto your SA console, and into the Proxy install directory on any machines that are running the proxy server.
  3. If you're customer uses FSDFS, they must update the necessary jobs as included in this hotfix. If they do not use this, then it is not required to update these jobs. Confirm there are no customizations on the affected jobs before updating with included files. Once confirmed, update the job files into the respective job directories.
