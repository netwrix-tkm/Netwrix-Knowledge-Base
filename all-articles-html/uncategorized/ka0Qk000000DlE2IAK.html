How To Reset Access to SI Console

Situation

A customer has lost control of the Threat Prevention Console and cannot access the configuration menu.

Solution

VIA SSMS - Reset SecurityDescriptor to null: 
update [NVMonitorConfig].dbo.Folder
set securityDescriptor = NULL
WHERE [Folder].[name] = 'Policies';
On SI Host:
  • Restart SI Enterprise Manager Service
  • Use a local Admin account to Launch SI Console - This user will be assigned as the only SI Administrator
  • Navigate to Configuration > Users
  • Add additional Administrators to prevent loss of access
