Summary: Enterprise Manager service stopped unexpectedly

Issue: Summary: Enterprise Manager service stopped unexpectedly
Behavior before Fix:
If one user is changing something (saved policy, or collection, or folder, or template, ...) and different user is starting new WinConsole instance or closing WinConsole at the same time then exception can be raised from the ForeignUpdateNotify function because list of WinConsole clients changed during going through the list.
Behavior after Fix:
Instead of going through the list now code in this place changed to go through items in the copy of this list. Also added exception handling into this function to avoid application crash in case if any other exception will happen.
