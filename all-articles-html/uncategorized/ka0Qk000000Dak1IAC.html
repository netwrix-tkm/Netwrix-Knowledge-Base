Hotfix Patch 2.2.213 for Threat Manager 2.2

Summary: This Hotfix addresses a variety of issues with Threat Manager Active Directory Sync, as well as implemented improved system outage handling and database installer artifact cleanup

Issue: Summary:
This Hotfix addresses a variety of issues with Threat Manager Active Directory Sync, as well as implemented improved system outage handling and database installer artifact cleanup
Issues Addressed

  • Additional Cleanup of unneeded PSQL artifacts post-install
  • Additional Fixes for (event_user violates foreign key constraint "fk_event_user_primary_group_token") issue
  • Implementation of PSQL Health Monitoring to "Reset" PSQL connection in the event of extended loss of communication between Threat Manager and Database.
