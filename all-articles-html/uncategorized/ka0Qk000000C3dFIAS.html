How to enable collection of more than 100 items in a report

IMPORTANT: This is a work-in-progress article. The product R&D team has yet to review some or all of the information below.

Question

How to enable collection of more than 100 items in PingCastle Enterprise and Pro?

Answer

To enable collection of more than 100 items in PingCastle Enterprise and Pro, follow these steps:

  1. Navigate to PingCastle Pro/Enterprise directory and open the appsettings.json file

  2. Add the following key in the JSON file

"__comment_MaxNumberUsersInHtmlReport": "Defines the maximum number of users displayed in the HTML report",

"MaxNumberUsersInHtmlReport": 1000,

You can adjust the value 1000 to any desired limit.
