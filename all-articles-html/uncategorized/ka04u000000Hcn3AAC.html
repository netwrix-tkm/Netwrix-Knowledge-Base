Issue: Ticket: Store ownership information if info not available in ADI
Issue: Cross-domain ownership cases where owners' AD information is not scanned, was not being supported by AIC, this caused the Owner information to show as 'Unresolved' in the AIC owners grid, entitlement reviews and access request workflows were not possible because of this.
Fix: A new AIC table SA_AIC_ActiveDirectoryPrincipals was created to store owners' AD data which was not scanned. The table gets populated by either assigning resource owners in the Manage Resource Ownership page or assigning roles to trustees in the configure console.
