[UPDATE]- For how many computers can I use File Tracing and File Shadowing?

The maximum number of endpoints for which you can enable File Tracing and File Shadowing is not limited in the Administration Interface of Endpoint Protector. However, we recommend you enable File Tracing and File Shadowing for up to 15% of your appliance total endpoint capacity, as this will slow your appliance simply from too many logs being sent. If you have questions how to optimize this feature, please contact our support by opening a support ticket.
