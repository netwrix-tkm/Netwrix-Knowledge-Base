Hotfix: SA_8.1_018

Summary: Attempts to start a SQL Sensitive Data scan on a Access Analyzer console with restrictive UAC policies was resulting in the scans not starting because the process was unable to create an SDD folder under the StealthAUDIT\Jobs\SA_CommonData\SQLDC directory.

Issue: Summary:
Attempts to start a SQL Sensitive Data scan on a Access Analyzer console with restrictive UAC policies was resulting in the scans not starting because the process was unable to create an SDD folder under the StealthAUDIT\Jobs\SA_CommonData\SQLDC directory.

Related Errors:
Failed to initialize new scan session: Unknown error
Access Denied for folder SQLDC
With this fix the Access Analyzer should be able to create the SDD folder and the scan runs successfully.
