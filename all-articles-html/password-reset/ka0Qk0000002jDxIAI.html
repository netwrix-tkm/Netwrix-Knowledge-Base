How to Configure Website Accessibility Using Server Hostname Only?

! This article is currently under review. Some of the information below may not be up to date.

Question

How to get Netwrix Password Reset to redirect a site to its hostname?

Answer

Follow the steps below to configure Netwrix Password Reset to redirect a site to its hostname:

  1. Open IIS Manager.

  2. Select Default web site and click Advanced Settings... in the right pane.

    NPR1

  3. Change the Application Pool from DefaultAppPool to Netwrix Password Reset.

    NPR2

  4. Change Physical Path from %SystemDrive%\inetpub\wwwroot to %SystemDrive%\inetpub\wwwroot\pwreset.

    NPR3

  5. Click OK, then double-click the Default Document icon in the middle pane.

    NPR4

  6. Click Add... in the right pane, then enter en_default.htm and press Enter.

    NPR5

  7. Double-click Handler Mappings and select ISAPI-dll.

  8. click Edit Feature Permissions... in the right pane.

    NPR7

  9. Check Execute, then OK.

Your default page when inputting Hostname or IP address should now be your Netwrix Password Reset site
