Export Not Available for DSAR Searches

Symptom

Once the DSAR search is completed, the Export button is greyed out.

Cause

Export of DSAR reports requires an output location to be specified.

Answer

In administrative web console, navigate to Data Analysis > DSAR. In the Settings tab, specify the output location in the corresponding field. Refer to the following article for additional information on DSAR settings: DSAR Settings.
