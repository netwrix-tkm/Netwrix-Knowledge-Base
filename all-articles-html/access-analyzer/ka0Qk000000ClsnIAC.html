Built-in Users Group Permissions

Question

During the security assessment in the Netwrix Access Analyzer environment, the use of the Users group (S-1-5-32-545) was noted for Access Analyzer services and folders. Does Access Analyzer require the use of the Users group?

Answer

The use of the Users group is not required�Access Analyzer implements the Users group by default. You can modify the permissions for the installation folder to exclude the group. Refer to the following path for the default installation folder for Access Analyzer:

C:\Program Files (x86)\STEALTHbits\StealthAUDIT\
