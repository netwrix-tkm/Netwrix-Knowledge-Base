Overview

This article lists the steps to perform the Access Information Center (AIC) upgrade.

Upgrade

Refer to the following steps:

  1. Download the latest version of Netwrix AIC from the Netwrix Customer Portal in My Products � Netwrix.

    IMPORTANT: Unblock the installer�in Properties, select the Unblock checkbox and save the changes.

  2. To upgrade the AIC application to a newer version, simply run the new AccessInformationCenter.msi executable. You do not need to uninstall the existing version. See the Install the Access Information Center topic for additional information.

  3. Hold Shift, right-click the latest AIC installer, and select Copy as path. In an elevated Command Prompt instance, paste the path and run it to start the installation.

  4. Complete the installation.

Post-Upgrade

Refer to the following steps to complete the AIC upgrade:

  1. If you have SSL implemented, alter the BindingURL node of the AIC configuration file to work with HTTPS. Refer to the following example:

    <add key="BindingUrl" value="http://+:82" />

    • If you have SSO implemented, alter the AuthAllowWindowsAuthentication node of the AIC configuration file to allow SSO. Refer to the following example:

      <add key="AuthAllowWindowsAuthentication" value="True" />

    • Refer to the following default path for the AIC configuration file:

      C:\Program Files\STEALTHbits\Access Information Center

  2. Have a user with the correct permissions log into the AIC to confirm the operability.

  3. Verify the AIC Configuration settings to match the settings saved during the preparation phase.

  4. Verify that the built-in admin for the AIC is either disabled or that the password has been changed.
