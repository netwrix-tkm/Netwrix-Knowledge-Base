Overview

This article lists the steps to perform the Access Analyzer upgrade.

Upgrading Access Analyzer Console

Refer to the following steps to upgrade the Access Analyzer Console:

  1. Download the latest version of Netwrix Access Analyzer Management Platform from the Netwrix Customer Portal in My Products � Netwrix.

    IMPORTANT: Unblock the installer�in Properties, select the Unblock checkbox and save the changes.

  2. Verify no Access Analyzer jobs are running. Review Task Manager > Details to verify.

  3. Uninstall the Access Analyzer Console either via Control Panel or Apps & Features.

  4. Hold Shift, right-click the Console installer, and select Copy as path. In an elevated Command Prompt instance, paste the path and run it to start the installation.

  5. Complete the installation and launch the Console instance�this should start the upgrade wizard.

    IMPORTANT: If the upgrade wizard does not start, run the following lines in a PowerShell instance to launch the upgrade wizard:

    cd $Env:SAINSTALLDIR
.\StealthAUDIT.EXE /Upgrade

  6. Review the upgrade conflicts�click View Conflicts to review the conflicts. Select one from the Changes window list and click Undo.

  7. In the Solution Set Files tab, click Advanced to verify the Active Directory solution, File System solution, and any other solution group with a custom structure in the job tree are set to Upgrade in Place.

  8. Proceed with the upgrade wizard to complete the setup.

Upgrading Sensitive Data Discovery

IMPORTANT: If licensed and used in the environment, Upgrade the Add-on after you upgraded the Access Analyzer Console.

Refer to the following steps to upgrade the Sensitive Data Discovery Add-on (SDD):

  1. Download the latest SDD version from the Netwrix Customer Portal in My Products � Netwrix. Transfer the installer to the Access Analyzer Console server.

    IMPORTANT: Unblock the installer�in Properties, select the Unblock checkbox and save the changes.

  2. Uninstall the older SDD version.

  3. Hold Shift, right-click the SDD installer, and select Copy as path. In an elevated Command Prompt instance, paste the path and run it to start the installation. Complete the installation.

Upgrading File System Proxy

IMPORTANT: If used in the environment, upgrade the FSSA Proxy after you upgraded the Access Analyzer Console.

If used for the Sensitive Data discovery and audit, upgrade the FSAA Proxy after upgrading the SDD Add-on.

Refer to the following steps to upgrade the File System Proxy Service (FSAA Proxy):

  1. Download the latest FSAA Proxy version from the Netwrix Customer Portal in My Products � Netwrix. Transfer the installer to the target Proxy servers.

    IMPORTANT: Unblock the installer�in Properties, select the Unblock checkbox and save the changes.

  2. In the target Proxy server, open Services and review the account used to run the StealthAUDITFSAA service.

  3. Uninstall the older FSAA Proxy version.

  4. Hold Shift, right-click the up-to-date Proxy installer, and select Copy as path. In an elevated Command Prompt instance, paste the path and run it to start the installation. Complete the installation.
