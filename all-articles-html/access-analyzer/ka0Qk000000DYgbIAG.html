Overview

This article lists the steps to complete the Access Analyzer upgrade procedure.

Instructions

Refer to the following steps to complete the Access Analyzer Console upgrade:

  1. After the Netwrix Upgrade Wizard finishes the upgrade, open the Access Analyzer Console.

  2. Verify the global settings to match the initial settings.

  3. If licensed for the File System auditing, run the File System\0-CreateSchema job. The job updates the schema for the tables, views, and functions used by the rest of the File System Solution.

  4. Run the .Active Directory Inventory group from the Schedules node to confirm the operability.

  5. Verify the published reports configuration matches the backed up version. Migrate the <AppSettings> node, if necessary�refer to the default configuration path:

    %SAInstallDir%Web\WebServer.exe.config

  6. If you are using the Windows authentication method to connect to the SQL database, set up the Netwrix Access Analyzer Web Server service to log on as a service account with the db_owner permissions for the Access Analyzer database.

  7. Have a user with correct permissions login to the Published Reports site to confirm the operability.
