Tracking Enterprise Auditor Console Logons

Question

Does Netwrix Enterprise Auditor track console logons and logoffs?

Answer

Enterprise Auditor logs the events in the dedicated StealthAUDIT event log located in the following path:

%SystemRoot%\System32\Winevt\Logs\StealthAUDIT.evtx

Review the Event IDs 1000 and 1003 to learn more about accounts accessing the Enterprise Auditor console.
