Access Analyzer console does not open with error code 5 access is denied.

Symptom

When opening the Access Analyzer console the user receives the message "System Error. Code: 5. Access is denied." while checking schemas.

�

Cause

The current user does not have sufficient permissions to the %SAInstallDir% folder. User must have minimum of modify access to the folder and all child objects.

�

Resolution

Add the user or group to the security permissions on the %SAInstallDir% folder and ensure that they have at least modify permissions.
