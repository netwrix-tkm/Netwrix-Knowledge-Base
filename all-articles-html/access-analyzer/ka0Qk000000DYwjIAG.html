Failed to Create FSAdapter Error in Netwrix Access Analyzer

Symptom

Netwrix Access Analyzer prompts the following errors:

Failed to create FSAdapter (80070002): The system cannot find the file specified
Error during processing: Failed to create FSAdapter

Cause

Your installed versions of Access Analyzer and the Sensitive Data Discovery Add-On are incompatible.

  • Access Analyzer v11.6.0.59 and higher require Sensitive Data Discovery Add-On v11.6.0.10 and higher.

  • The Sensitive Data Discovery Add-On v11.6.0.10 and higher require Access Analyzer v11.6.0.59 and higher.

The incompatibility is caused by an outdated component.

Resolution

Update both Access Analyzer and the Sensitive Data Discovery Add-On to their latest versions.
