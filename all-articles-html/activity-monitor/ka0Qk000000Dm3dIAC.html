Hotfix: SAM_6.0_020

Summary:
  1. MS SAM Crashed a DC in relation to the ConfigurationAgent.exe casing "A critical system process died"
  2. SAM Linux agent showing outdated in console; Linux Agent fails due to Cert error
  3. Nasuni Security Vulnerability on port 4493
Behavior Before Fix:
  1. While trying to configure windows firewall, SAM agent crashed windows firewall which caused full system to crash.
  2. No events reported for linux.
  3. SAM's Nasuni Monitoring supported insecure / null encryptions.
Behavior After Fix:
  1. Implemented caching of results of queries to the windows firewall to limit our communication.
  2. Updated SAM linux auditd rules config to specify order.
  3. SAM's Nasuni Monitoring has been updated to not allow insecure / null encryptions.

  • 101947: SAM Console can't connect to the Isilon 8.1
  • 102406: Update OpenSSL to 1.0.2zd (1.1.1n - 17.03.2022)
  • 104752: 6.0 upgrade fail
