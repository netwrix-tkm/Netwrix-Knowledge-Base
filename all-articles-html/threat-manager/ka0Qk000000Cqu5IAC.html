How to Reset System Administrator Account

Overview

This article covers the steps to reset the password for the system administrator account in Netwrix Threat Manager.

Instructions

Refer to the following steps to reset the system administrator account password in Threat Manager:

  1. Find the psqlbuddy_v3.zip file at the bottom of this article. Download and extract the contents of the archive on the Threat Manager server.

  2. Run the psqlbuddy_v3.ps1 file in elevated PowerShell.

    cd %target_path%
.\psqlbuddy_v3.ps1

  3. Specify the PSQL.exe path depending on the installation path. Refer to the following default path for the latest Threat Manager versions:

    C:\Program Files\STEALTHbits\Postgres14\bin\psql.exe

  4. In the Select Operation drop-down list, select the Reset StealthDEFEND Admin Password V2.8+ option.

  5. Click the Execute Operation button�the script prompts a pop-up to input a database password. Submit the default password�StealthDEFEND1234!!

    NOTE: The prompt will remain blank while typing the password. Ensure the password is manually entered. Copying and pasting the password may result in input errors.

  6. Review the output. The following line stands for a successful reset of the password:

    Output: Delete from "AspNetUsers" where user_name = 'ADMIN';  DELETE 1

  7. Log out of Threat Manager�it should prompt a pop-up to set a new administrator password.
