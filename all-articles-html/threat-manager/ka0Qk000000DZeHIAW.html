Hotfix Patch 2.7.1336 for Threat Manager 2.7 SP1

Summary:
In this hotfix we fixed various issues relating to StealthDefend 2.7 SP1:
  • Updated Python to version 3.10.4
  • Rewrite action steps that were affected by recent schema updates
  • Fixed an issue where password spraying was not being detected
  • Fixed several issues with playbooks not triggering
  • Fixed and optimized memory usage problems for Threat Manager services
  • Updated dotNet to 5.0.15
  • Fixed several minor UI bugs
  • Fixed detection issues with Golden Tickets that were causing false positives
  • Fixed a syntax issue for the CEF SIEM template
Submitted by:
Greg Connors
Affected Versions:
Prior versions of 2.7 SP1
