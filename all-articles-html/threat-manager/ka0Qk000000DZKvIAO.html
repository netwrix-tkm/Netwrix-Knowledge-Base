Hotfix Patch 2.7.1544 for Threat Manager 2.7 SP1

Summary:
In this hotfix we fixed various issues relating to StealthDefend 2.7 SP1:
  • Updated PostgreSQL to 10.22 due to vulnerabilities in the previous version
  • Fixed an issue with summary numbers not loading on the main DEFEND page
  • Fixed a Ransomware issue relating to file rename events
  • Updated the Close SMB session action step to not require WinRM
  • Updated the End RDP session action step to not require WinRM
  • Optmizied memory usage for Threat Manager services
Submitted by:
Greg Connors
Affected Versions:
Prior versions of 2.7 SP1
