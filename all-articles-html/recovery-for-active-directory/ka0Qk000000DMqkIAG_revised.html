How to Install and Upgrade Netwrix Recovery for Active Directory

Overview

This article provides step-by-step instructions for installing and upgrading Netwrix Recovery for Active Directory to the latest version (v2.6). It also outlines the components included in the installation and their default locations.

Instructions

  1. Download the NetwrixRecovery_Setup.exe file from the official Netwrix Recovery for Active Directory website.
  2. Run the NetwrixRecovery_Setup.exe on the Netwrix Recovery for Active Directory server to upgrade to the newer version of the product.
  3. Follow the on-screen prompts to complete the installation or upgrade process.
  4. By default, the installation location is C:\Program Files\Netwrix\Recovery for Active Directory\. If you wish to change the installation location, click Change... during the setup process.

Components

Netwrix Recovery for Active Directory consists of the following components:

  • Netwrix Recovery Server: The core server component responsible for recovery operations.
  • Netwrix Recovery Web Console: Installed on the Recovery Application Server and used to perform the operations the application offers.
  • RSAT Extension: Installed automatically on the Recovery Application Server to enable additional functionality.

Related Query

"Which version of .NET is required for Netwrix Recovery for Active Directory?"

Answer: The installation does not require ASP.NET Core or .NET Framework as dependencies. Ensure your server meets the general system requirements outlined in the Recovery for Active Directory documentation.
