Error: SmtpNotification.comhost.dll Failed to Register

Symptom(s)

When installing or upgrading Netwrix Recovery for Active Directory, you encounter the following error message:

SmtpNotification.comhost.dll Failed to Register

Cause(s)

This error occurs because your version of .NET does not meet the requirements to install the latest version of Netwrix Recovery for Active Directory. The installer attempts to download a compatible version of .NET but may fail if the system cannot connect to the internet or if access is blocked.

Resolution(s)

Follow these steps to resolve the issue:

  1. Ensure that your system meets the following prerequisites:
    • Membership in the local Administrators group.
    • The Log on as a service right, which can be configured in the Local Security Policy under Local Policies > User Rights Assignment > Log on as a service.
    • SQL Server permissions for the domain account being used.
  2. Manually download and install the following components:
    • ASP.NET Core 8.0.6
    • Windows Desktop Runtime 8.0.6
  3. Launch the Recovery Configuration Utility to update the SQL Server Authentication password:
    1. Navigate to the installation directory: ...Netwrix\Recovery for Active Directory\.
    2. Double-click Recovery_Config.exe to open the utility.
    3. Enter the updated SQL Server Authentication credentials in the Login and Password fields.
  4. If upgrading from version 2.5 to 2.6, follow these steps:
    1. Run the NetwrixRecovery_Setup.exe executable as an administrator to launch the Recovery for Active Directory Setup wizard.
    2. Follow the on-screen instructions to complete the upgrade process.
  5. To install the RSAT Extension, locate the installer at ...Netwrix\Recovery for Active Directory\RSAT Extension\NetwrixRecovery_RSAT_Extension.msi and follow the installation steps.
