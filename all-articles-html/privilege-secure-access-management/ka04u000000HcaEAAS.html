How to Customize the RDP Template

Summary

Additional configuration settings for SbPAM RDP sessions can be configured by modifying the�rdp_template.txt�file on an SbPAM server. Modifications to this file will affect the default RDP settings for all SbPAM users, and the file includes such settings as:
  • Fullscreen Resolution (screen mode id:i:value)
  • Smart Sizing (smart sizing:i:value)
  • Font Smoothing (allow font smoothing:i:value)
  • Smart Card Redirection (redirectsmartcards:i:value)
  • Disable Themes (disable themes:i:value)
  • Disable Wallpaper (disable wallpaper:i:1)
  • Audio Mode (audiomode:i:value)
A full list of supported RDP settings can be found in the following Microsoft documentation -�Supported Remote Desktop RDP file settings | Microsoft Docs
