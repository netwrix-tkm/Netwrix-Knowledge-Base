How to Upgrade from SbPAM v3.6 to Netwrix Privilege Secure v4.0

NOTE: On all upgrades, the previous version will be uninstalled automatically prior to the installer running. In addition, a reboot will occur following any new install or upgrade to V 4.0 or higher.
  1. Following the upgrade and reboot, log into the Netwrix Privilege Secure UI as an administrative user and verify that the services reregister.
  2. Ensure that services are all green under Service Nodes. Refresh if necessary as they update.
  3. Restart the Proxy service (this will also restart dependent services) via services.msc.
    The upgrade is now complete.
IMPORTANT: It is recommended to open each managed domain under Resources and run a test to verify that the service account is correct. This is to ensure that the service account password was unaffected during the upgrade. If this happens, re-enter the service account password and retest to verify operation.
