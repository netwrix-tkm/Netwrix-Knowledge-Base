Hotfix: SbPAM 3.6CU1 (3.6.3107)

Summary

3.6.3107 includes several fixes and performance improvements.

Enhancements

  • Ability to delegate management of access policies to specific users in your organization. This allows SbPAM Administrators to assign a custom role to users to allow those users to manage a specific access policy for a select group of users, groups, resources and activities.
  • SignalR backplane to improve UI responsiveness in HA and web service scale out environments.
  • RDP files are now prefixed with server names, making it easier to identify server based on Window title.
  • WinRM Disabled option for resources to bypass attempts to connect to remote resources using WinRM, improves session connection times.
