How to: Configure Security questions for an Identity Store

Applies To:

GroupID 11

Summary:

GroupID provides a pool of pre-defined security questions. The pool can be modified by adding or removing questions from it.

Each identity store in GroupID has its own local security questions pool. Admin users can add questions to a local pool from the GroupID pool bank.

Enrollment and Authentication via Security Questions:
GroupID provides multiple authentication types for multifactor authentication and two-factor authentication. Security questions are one of them.
When an identity store is created, four questions from the question pool bank are automatically added to the new identity store�s local pool. Questions can be added and removed from the local pool.
To enroll in GroupID via security questions, identity store users have to select a specified number of questions from the local pool and provide answers. For authentication, users have to provide the same answers to the questions they enrolled with.

  2. In the Admin portal, click on Settings.
