How to Replace the Password Center Portal Logo

Applies To

Directory Manager 11.1 ï¿½ Directory Manager Portal

Question

Can you replace the logo on the Password Center portal in Netwrix Directory Manager (formerly GroupID)?

Answer

Yes, it is possible to replace the Password Center logo. This can be achieved by following the steps below.

Replace the Logo on the Password Center Portal

  1. Go to the following directory:
    C:\Program Files\Imanami\GroupID 11.0\GroupIDPortal\Inetpub\<Your Portal Name>\Web\wwwroot\Content\Images\vector
  2. Locate the file named PCFullLogoTransaprent.svg. This file is used to render the logo on the portal.

  1. Convert your logo to the .svg format and copy it to the same directory.
  2. Rename the existing PCFullLogoTransaprent.svg file to PCFullLogoTransaprentbackup.svg for backup purposes.
  3. Rename your new logo file to PCFullLogoTransaprent.svg.
  4. Open the Password Center portal in incognito mode to verify that the new logo appears.

NOTE: Adjust the dimensions of your logo as needed to fit the portal layout.
