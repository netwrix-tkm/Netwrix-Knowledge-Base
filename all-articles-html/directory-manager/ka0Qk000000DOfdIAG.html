How to limit users to search only for User objects

Applies To:

GroupID 11

Business Requirement:

You would like to limit your users to search only for User objects in the User portal's�Find dialog box.

Solution:

You can launch the Find dialog box from multiple User portal pages to search for objects to designate as owners, managers, additional owners, members, and more.

By default, the Find dialog box searches for all types of objects, including users, contacts, and groups. Use this setting to limit the Find feature to specific object types. For example, select the Users check box to limit users to search for the User objects only.

Steps:

  1. In GroupID Admin Center, go to Applications. On the application/portal where you want to implement this setting click on the 3 dots and select Settings.

  1. On the next page, click on�Advanced Settings. On the right side under the tab named Portal & Search, look for the option named�Find Dialogue / Look For. From there uncheck Groups and Contacts.

  1. Scroll down the page and click on the Save button to save your changes.

Impact on the portal:

By default, the Find dialog box searches for all types of objects, including Users, Contacts, and Groups.

After changing the value only to Users as described above, the Find dialog box will only display the User object type in�searches.
