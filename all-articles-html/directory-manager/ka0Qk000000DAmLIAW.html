How To: Display the Groups that users own as Additional Owners

Applies To:

GroupID 11

Business Requirement:

By default, the �My Groups� tab in the Application portal displays the groups for which the logged-in user is the primary owner. We have a requirement to display those groups for which the logged-in user is an additional owner.

Methodology:

We can change the settings of the Application portal from the Admin portal to display the groups that the logged-in user owns as an additional owner too.

Users can still choose to view the groups they own as primary owners only. Individual users can adjust the settings accordingly from within the portal.

Steps:-

Follow these steps to display additional ownership on the My Groups page in the application portal:

  1. From the Admin portal, navigate to Applications -> Your application portal -> Click on the three-dot buttons at the top right corner -> Settings.

  1. In the settings, Navigate to Advance settings -> Listing Displays.

�

  1. Toggle the setting for �Display Groups in My Groups�.

�

  1. Once done, click on save and okay buttons in sequence and log in to the application portal. You will now see those groups in the My groups page as well for which the logged in User is also an additional owner.
