How To: Copy criteria of a Smart Group to another Smart Group

Applies To:

Directory Manager 11

Business Case:

You have a complex criterion defined for a Smart Group and you want to apply this criterion as it is to another Smart Group.

Methodology:

You can simply use the import/export feature of the Query Designer in the application portal to export the query of a Smart Group to a JSON file, and then import that file for another group. In this way, you can apply the same query to multiple groups.

Steps:

  1. In the application portal of Directory Manager, click �Groups > All Groups > Smart Groups.
  2. Open the properties of the smart group whose criteria you want to copy.
  3. Navigate to Smart group tab -> Query Designer.

  1. In the query designer window, click on 3 dots button at the top right corner and click on Export query. Depending on the download location, it will download the JSON file to that location.

  1. Now open the properties of the smart group in which you want to copy the criteria.
  2. Navigate to Smart group tab -> Query Designer.
  3. In the query designer window, click on the 3 dots button at the top right corner and click on Import query.
  4. Once done, you can check the Preview and confirm if the query is fetching correct results.
  5. Complete the rest of the steps of the smart group wizard.
