How To: Customize Directory Manager's Notifications

Applies To:

Directory Manager 11

More Information:

GroupID generates a variety of notifications for different events, such as group management, workflows, and profile validation. These notifications are sent to administrators, object owners, and other specified recipients.

Templates for these email notifications are predefined in GroupID. These templates are highly customizable, enabling you to change the following for each template:

  • Logo
  • Subject line
  • Message (content and formatting)
