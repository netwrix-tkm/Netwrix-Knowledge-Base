HTTP/HTTPS Error 500.19: Cannot add duplicate collection entry of type 'add' with unique key attribute 'name' set to 'X-Frame-Options'

Problem and Cause

This error occurs when sometimes upgrading Change Tracker to the latest version from a much older version. The "X-Frame-Options" collection entry has a duplicate in another location, and needs to be removed from the web config file to avoid IIS Conflicts. The below information will show you how to do this.

Resolution

  1. Open up IIS, and stop the IIS Server
  2. Navigate to C:\inetpub\wwwroot\Change Tracker Generation 7 WebUI\html\
