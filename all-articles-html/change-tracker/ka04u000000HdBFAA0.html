Support for TLS 1.2 in the .NET Framework 3.5.1

The .NET Framework version 3.5.1 and earlier versions did not support for applications to utilize the TLS 1.2 protocol. Microsoft has released an update that enables the use of TLS 1.2 in the .NET Framework 3.5.1.
