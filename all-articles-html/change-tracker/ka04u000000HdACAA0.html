How to Change the Online and Archive Data Retention settings in Log Tracker

There are two options that control the online and archive retention within Log Tracker. Log Tracker utilizes a web based menu and both can be accessed from the following menu:Messages > Config > Parmsand then select the "Edit >" button. There are two options, "Keep Online Data" and "Keep Archive Data". The first parameter specifies the duration for how long the Log Tracker event data is kept online. This value can be from 7 to 500 days. The second parameter specifies the duration for how long the Log Tracker archive data is kept in the system. This value can be from 0 to 5000 days. When your changes are complete, click the "Commit >" button to save the changes or "< Cancel" to not save the changes.
