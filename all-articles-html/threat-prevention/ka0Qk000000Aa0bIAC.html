Symptom

When performing an upgrade from 7.3.9 to 7.4, during the install stage the NTP installer generates the following error message:

Cause

These errors may be caused by any one of the following:

  • SQL Storage Disk drive is at capacity

  • dbo.Alerts table in the NvMonitorConfig database needs to be truncated

  • SQL Database Mirroring is enabled
