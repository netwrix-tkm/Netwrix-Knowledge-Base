Hotfix: SIWindowsAgent_7.3.5.159

Summary:
Allows filters for DCSync

Behavior before Fix:
No mechanism for turning on only DCSync-specific logging existed
Behavior after Fix:
DCSync-specific logging now can be enabled via logging.ini
Enabling:
  • Open SIWindowsAgent\logging.ini file
  • Add or update:
    • DebugTrace = Debug
    • Filter = DCSync
  • Restart SI Agent
