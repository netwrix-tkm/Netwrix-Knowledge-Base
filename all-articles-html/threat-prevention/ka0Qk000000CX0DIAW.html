Error: NTP Agent Lost Connection After Upgrading

Symptom:

NTP Agent listed as Lost Connection post upgrading/patching to new build.

Cause:

Within the NTP Console the host was listed without the Domain details this could have resulted from the host being replaced with a new server that was named to match the original.

Found within the NVMonitorConfig DB within the Server Table the host was listed multiple times with and without the Domain Prefix.
