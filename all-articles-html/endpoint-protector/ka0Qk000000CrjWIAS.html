Printing jobs not blocked/reported

In order for the EPP agent to properly report or block the printing jobs, the Advanced Printer and MTP Scanning option needs to be enabled in Device Control -> Global Settings.

�

In order for the setting to be properly enabled, please follow the below steps:

  1. Enable "Advanced Printer and MTP Scanning" in Device Control -> Global Settings.
  2. Click "Update Policies Now" on the EPP client icon.
  3. Reboot the machine.

�

    If the problem persists after following the steps above, please raise a ticket on the Support Portal.
