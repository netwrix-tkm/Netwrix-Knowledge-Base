[UPDATE]- EPP server UI only accessible with Safari browser - VM hosted only

If you can only access the EPP server UI in the Safari browser, while the EPP server is hosted on a VM, please follow the below steps:

  1. Please access the VM where the EPP server is hosted.
  2. Check the networking settings.
  3. Please make sure that the Gateway is configured/added.
  4. Save the settings.

    5. The EPP UI should now be accessible from any browser.
