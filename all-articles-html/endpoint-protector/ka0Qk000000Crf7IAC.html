[UPDATE]- How to monitor webmail

The webmail can be monitored by EPP using Deep Packet Inspection.
Note: Make sure to enable Deep Packet Inspection under Device Control > Global Settings.

In order to enable the webmail monitoring:
- Access Content Aware Protection > Deep Packet Inspection�
- Enable "Monitor webmail"
- Select Parser (Gmail, Outlook, Yahoo)
- Save.

After the Monitor webmail is enabled you will have to:
- Access Content Aware Protection > Content Aware Policies
- Create/edit a policy�
- Select the browsers from where you want to monitor webmail�
- Save.
