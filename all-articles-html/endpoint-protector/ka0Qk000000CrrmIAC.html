[UPDATE]- How to upgrade from Endpoint Protector Server v5.8.0.0 to v5.8.1.0 with a greyed-out "Apply Updates" button

Please follow these steps if the "Apply Updates" button is grayed out after selecting the available EPP Software updates:



Resolution:

Step 1: Uncheck the checkbox next to the patch update.



Step 2: Click on the checkbox again to activate the patch update once more.



Note:

  • To activate the "Apply Updates" button please click the checkbox directly.
  • Hosted customers need to use the "Apply All EPP Software Updates" button.



Step 3: Click the "Apply Updates" button to begin the download.

�

Step 4: Refresh the entire URL page so that on the bottom right corner, you will see the new version of the Server.
