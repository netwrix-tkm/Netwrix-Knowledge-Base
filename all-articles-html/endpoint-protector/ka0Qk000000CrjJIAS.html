CAP tab missing from the epp agent UI

Question:

Why is the CAP tab missing from the EPP agent UI?

Answer:

The CAP tab missing from the EPP agent UI usually means that the particular machine / user is not targeted in any Content Aware Protection policies.

�

If you determine from server UI that the machine or user is targeted in at least one Content Aware Protection policy, but the CAP tab is still missing from�the EPP�agent UI, then please make sure the EPP client is able to communicate with the EPP server(e.g. check "Last Seen" field from server UI > Device Control > Computers page).

�

This can be checked from the EPP client Settings tab by pressing on the keyboard:

  • on Windows: CTRL + ALT + i
  • on macOS: Command + Options + i
