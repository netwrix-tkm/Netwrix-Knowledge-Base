Source code detection can be diminished by the small volume of code that is present inside a file.

As mentioned in the user manual, Source Code Detection may encounter challenges when dealing with small code snippets. This can occur due to the potential overlap among various programming languages. It's important to consider these limitations when configuring and utilizing Source Code Detection for optimal results.

In conclusion, the detection rate is directly proportional with the amount of code lines that are added inside of the files (Shorter files, with less code, have less chances to be detected correctly).
