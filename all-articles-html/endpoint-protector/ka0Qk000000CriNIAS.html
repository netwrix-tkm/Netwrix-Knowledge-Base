What to do if you are not seeing the Destination URL in the Content Aware Reports

Starting with version Endpoint Protector Server version 5700, the Reporting V2 was introduced to the product.

In order to have the Destination details available in Content Aware Reports, the Reporting V2 needs to be enabled from System Configuration > System Settings, just as presented in the screenshot below:

After enabling Reporting V2, the Reports and Analysis > Content Aware Reports will contain an additional column that needs to be selected by the Administrator to be displayed in the Reports, from the Show /Hide Columns button, just as presented in the screenshot below:
