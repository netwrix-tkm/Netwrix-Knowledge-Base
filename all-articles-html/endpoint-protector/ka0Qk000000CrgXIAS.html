[UPDATE]- What do client integrity fail events mean ?

This type of event can have multiple causes, for example :

- Requires full disk access

- Certificate was not imported in Keychain

- User doesn't allow the network extension

- Installation file missing

- Bluetooth permissions not granted

If you want to see more information about what caused the Client Integrity Fail event, then please go to server UI > Reports and Analysis > Logs Reports > then enable columns 'File Name' and 'File Type' from 'Show/Hide columns' section :
