[UPDATE]- What to add for deploying EPP client on MAC OS through Intune for full disk access

What to add for deploying EPP client on MAC through Intune for full disk access:

Please check the user manual for Intune and along with following steps you will be able to deploy the client on MAC OS through Intune and benefit of Full disk access for the client:

https://helpcenter.netwrix.com/bundle/EndpointProtector_5.9.4/page/Content/EndpointProtector/Install/Agent/Intune/macOSDeployment.htm

Info for adding system extension into Intune:

Select Devices > macOS > Configuration profiles > Create profile
Select profile type as Templates and Device restrictions as template name
Complete the basics and go to configuration settings where we select privacy preferences
