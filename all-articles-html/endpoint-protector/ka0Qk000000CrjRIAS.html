Enforced Encryption(EasyLock) License Shows Expired When EPP Server Has Valid License

Question:

How do I reassign a valid Enforced Encryption (EasyLock) License from the Endpoint Protector Server to the USB Storage Device?

Answer:

To resolve the issue of "License Expired" for Enforced Encryption (EasyLock) please follow the below steps:

�

  1. Set the USB device on�allow�right under "Device Control" > "Devices" > "Right Click on the device > "Manage Rights"(be sure that the USB storage device is accessible). Alternatively you can connect the USB storage device to a computer which does not have Endpoint Protector installed.
  2. Navigate to hidden folder ".EasyLock Settings" and delete the file labeled "cs14c7el.data".
  3. Delete the executable file "EasyLock.exe" from the root of the USB storage device.
  4. Download the EasyLock utility executable from the following link: https://download.endpointprotector.com/Support_files/EasyLock_util.zip
  5. Extract the file "EasyLock_util.exe" to the root location of the USB Storage Device and run it.
  6. Download EasyLock package from the Endpoint Protector Server and copy it directly to the root of the selected USB storage device, extract all files and overwrite existing files.
  7. Open the EasyLock "EasyLock.exe" from the USB Storage device and enter password.
