[UPDATE]- ChatGPT is not working - how to allow it

In order to allow ChatGPT you will have to add the openai.com URL in Denylists and Allowlists -> Allowlists -> Deep Packet Inspection.

As an example, you can create a dictionary for allowed domains and add the domain using wildcards:

*openai.com*

Once the Allowlist dictionary is created you will have to include it in the Content Aware Policy.

Edit the Content Aware Policy -> Expand the Policy Allowlist section -> Deep Packet Inspection -> select the dictionary created above and save the policy.
