[UPDATE]- Destination Field missing info

This guide is trying to address the case of the "Destination Details" tab missing from the Reports and Analysis section as per below.
Reports and Analysis section can be found in the Netwrix Epp server UI.

if the Destination details, which lists the website to where a file monitored was uploaded, is missing, this can be enabled by turning on the reporting V2 setting, as shown in the next screenshot.

After this, the Destination Details filed should be selectable and viewable.

Also, this setting should always be enabled as it contains enriched features over the Reporting V1 counterpart.
