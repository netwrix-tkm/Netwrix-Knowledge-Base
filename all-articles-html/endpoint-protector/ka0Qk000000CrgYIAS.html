[UPDATE]- Content aware policies priorities

You can configure a priority for each content aware policy (CAP) from server UI.

However, please note that in the current EPP implementation, there is no guarantee in which order the 'Block' CAP policies will trigger as policies are evaluated at the same time (not sequentially) against a file.

The reasoning for the CAP policies priorities is that when a conflict is encountered (e.g. one policy only-reports a PII and the other blocks the PII) then EPP will apply the policy which has a higher priority.
