[UPDATE]- Websites not loading correctly with DPI enabled

If the EPP client is blocking some images, charts etc. from loading on websites that aren't blocked, or the website is not loading correctly, and DPI is turned on, you can check if somehow the CDNs get blocked and add them to an Allowlist, and assign that Allowlist to the Content Aware Policy which is used to block certain websites.
