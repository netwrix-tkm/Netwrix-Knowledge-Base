[UPDATE]- Is stealthy DPI driver in BETA?

We already consider the "Stealthy DPI driver" as stable while using the latest client versions.

It can be safely used in your production environment, and the BETA tag will be removed with server version 5.9.3.0

Please note that users trying to access websites with untrusted certificates will not be able to access these sites with "Stealthy DPI driver" unless "Enable peer certificate validation" is disabled or the site gets added to an allowlist in the Allowlist - Deep Packet Inspection section.
