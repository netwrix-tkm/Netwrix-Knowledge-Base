[UPDATED] - How to reset an EPP admin password

The reset of an Administrator password on the Endpoint Protector Server.


In case of a forgotten administrator password, the password can be reset via the methods described below.

NOTE: The information presented in this article applies to regular Endpoint Protector administrators. For the administrators imported from Active Directory the password should be the one from AD.
