[UPDATE]- Update the Epp server using offline patches

Before applying any update please be sure to create a VM snapshot in your environment.

Request the offline patches from support by mentioning the current EPP Server version you are on. The version can be seen on the UI in the lower right corner.

When applying the offline patch you will need to apply only one patch at a time to avoid issues.

To apply an offline patch you will have to access "Dashboard > Live Update > Offline Patch Uploader".

Here please browse for the appropriate offline patch and click "Upload Patch".

You will need to wait for the confirmation that the patch has been uploaded.

After the patch has been uploaded you will need to return "Live Update" to see the current status of the update. You will need to wait for the confirmation that the update is finished.

Once this is done you can refresh the web page and confirm the current EPP Server version.

You can repeat the steps above to apply the next offline patch.
