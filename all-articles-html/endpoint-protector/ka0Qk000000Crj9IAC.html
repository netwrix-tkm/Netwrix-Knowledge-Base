How do I uninstall EPP Linux clients older than 2.0.0.0?

Answer:

If you intend to uninstall your current EPP client from your machine please make sure to use the uninstall.sh file that you received from the same version.

If you no longer have the installer for that version you can raise a Support Ticket from the Support Portal and ask for the kit corresponding to that EPP client version. We will send you the link for it.

After uninstalling the old version it would be recommended to restart the computer.
