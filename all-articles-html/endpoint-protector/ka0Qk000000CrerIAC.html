[UPDATE]- Does "Uninstall Attempt" event confirm the agent was removed from the client machine ?

No, there is no confirmation of the final result of an attempted agent removal as there would be no agent present to communicate back to the EPP Server if removal is successful.

Note: If the agent still "PINGs" the server after an "Uninstall Attempt" event, it should be a good indicator that the agent removal process failed.
