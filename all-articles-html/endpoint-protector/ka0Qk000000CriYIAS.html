[UPDATE]- Migrating the Epp server to a new image

  1. Download the new VM from the link below.

https://www.endpointprotector.com/epp/virtual-app-email

  1. Import it into the environment, and check the version on the browser in the lower right corner.

  2. Please activate the trial license on the new server.

  3. Update both EPP servers up to the version of the new server using the Live Update or offline patches. If offline patches are needed, please let us know the current version and I will send you the patches.

- Before updating please create a snapshot of the VM to have a restore point in case anything goes wrong.
