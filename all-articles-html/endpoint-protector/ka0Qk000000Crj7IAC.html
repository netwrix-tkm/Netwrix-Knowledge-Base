[UPDATE]- Whitelist applications in EPP

Question:

How do I whitelist applications in EPP.

Answer:

Before applying the below steps, check if the setting called Advanced Printer and MTP Scanning(from Device Control -> Global Settings is Enabled.

If it is turned off, please turn it on, save and restart the machine. If the application works with this setting enabled, continue to follow the steps:

  1. Please go to System Parameters -> Advanced Scanning Exceptions.
  2. Under the first tab called Application Process Name please input the executable name as shown in the logs.
  3. Click Add to content so it will be listed under the second tab called "Whitelisted Applications".
  4. Check the checkbox next to the newly created entry and then hit generate.
  5. Update the policies.
