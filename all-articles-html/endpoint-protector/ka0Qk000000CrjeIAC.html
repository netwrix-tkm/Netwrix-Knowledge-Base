FireFox (especially on MAC)

Particularly on Mac, it is common behaviour that Firefox will not use the System certificates for validation, but rather the local Firefox profile (pre-cached) certificates.

To mitigate this behaviour, Firefox needs to have the security.enterprise_roots.enabled setting to be set to TRUE, as showcased in the above screenshot. This will make Firefox trust 3rd party certificates like ours.
