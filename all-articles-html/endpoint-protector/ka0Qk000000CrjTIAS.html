Create Exception for specific file extensions

In some cases, you do not wish to have a specific file extension traced, scanned, or shadowed.

ï¿½

In this case please follow the below steps:

  1. Please go to "Device Control > Global Settings", under "File Tracing and Shadowing".
  2. There is a field for each action tracing, scanning, or shadowing.
  3. In the respective field(Exclude Extensions) you can add the desired extension(e.g. .mp3;.exe;) you want to exclude.
