[UPDATE]- DPI certificate not available on the browser

Scenario:

Even if DPI is enabled on the computer and there aren't apparent errors, sometimes the DPI certificate is not available when checking it on a web browser.

Resolution:

Make sure there are Content Aware Policies (CAP) that apply to the computer. In order for the DPI certificate to be deployed and used it is necessary to have CAP policies that monitor browsers or other applications that are controlled using DPI.
