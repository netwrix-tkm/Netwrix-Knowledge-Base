[UPDATE]- Reports and Analysis logs not loading

If the Reports and Analysis logs section does not load on your end please check the below setting:

  1. Please go to Device Control -> Global Settings.
  2. Scroll down to the bottom of the page to the Display Settings.
  3. Make sure that "No. of records per report page:" is not set to "All".
  4. If it is set to "All" or a high value please select a lower value, between 10 and 100.
  5. Save.

Please check again the Reports and Analysis logs section and see if the logs are displayed now.

Reports and Analysis logs section can be found on the Netwrix Endpoint Protector server UI.
