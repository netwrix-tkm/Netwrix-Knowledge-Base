[UPDATE]- How to collect extended Deep Packet Inspection (DPI) logs for Windows

Question:

How do I collect extended Deep Packet Inspection (DPI) logs for Windows?

Answer:

Please find below the steps you will need to perform in order to collect extended Deep Packet Inspection for Windows:

  1. Download: https://download.endpointprotector.com/Support_files/epp_collect_dpi_tool.zip
  2. Copy the epp_collect_dpi_info.bat script file to the computer where you are experiencing DPI problems and run it as Administrator.
  3. Wait until the script finishes executing.
  4. At the end a Windows Explorer window will open where you can view the collected files.
  5. Please archive the logs as .zip and send them in the support ticket/thread opened regarding the problem you are experiencing.
