[UPDATE]- How to check Endpoint Protector administrators activity in the EPP console

If there is a need to check the activity of a certain administrator of the solution, please go to the Reports and Analysis - Admin Actions section in the UI.

Here you can find logs related to the administrators activity, such as adding custom rights for devices, AD Syncs, creating or modifying groups, etc.

The filters at the top of the page can be used to search for a specific administrator and a specific Activity/Operation/UI Section.
