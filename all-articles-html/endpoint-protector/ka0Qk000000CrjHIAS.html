Manually uninstalling the epp agent(uninstall password is required if configured)

Question:

How do I manually uninstall the epp agent(uninstall password is required if configured)?

Answer:

�

For�Windows OS:

  1. Go to Control Panel.
  2. Programs and Features.
  3. Select the Endpoint Protector from the list.
  4. Select Change(Uninstall).

�

For�MacOS:

  1. Inside the client installer package you will find a file called "remove-epp".
  2. This needs to be executed or run in Terminal.

�

For�Linux OS:

  1. Inside the client installer package, you will find a file called "uninstall.sh".
  2. This needs to be run from the terminal(i.e. bash uninstall.sh).

�

An EPP agent uninstall can also be triggered from server UI > Device Control > Computers > check computers, then click 'Uninstall' button found at the bottom of the page.
