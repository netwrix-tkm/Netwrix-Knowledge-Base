Adding a SSL Certificate

Question:

How do I add a SSL Certificate?

Answer:

The SSL Certificate will have to be generated either on your side, or by using a 3rd party application.

In order to add a SSL Certificate:

  1. Please navigate to the EPP UI > Appliance > Server Maintenance:
  2. Once a .pem SSL Certificate is generated(certificate.pem and certificate.key), their content can be pasted in the self signing certificate tabs and then saved.
  3. Allow 5-10 minutes for changes to apply, then close and re-open the browser completely.
