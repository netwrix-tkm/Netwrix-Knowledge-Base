[UPDATE]- Upgrade Linux EPP clients without uninstalling-reinstalling

The upgrade mechanism in Linux is actually an uninstall and reinstall process.

The following steps perform an upgrade without having to uninstall and reinstall:

-> Copy installer over to the Linux machine

-> Open Terminal

-> Export the variables from options.ini (from the older build)

export EPPCLIENT_WS_SERVER= ip.of.the.server

export EPPCLIENT_WS_PORT=443

export EPPCLIENT_DEPARTMENT_CODE=defdep

-> Unpack the archive: tar xvf [Filename.tar.xz]

-> Run sudo -E apt install ./pkgs/*.deb from the unpacked directory from the newer build

-> Open Endpoint Protector Notifier and confirm if the EPP client was upgraded.
