Check if the Client is communicating with the Server

Please open the EPP client and go to the Settings tab:

To see if the client is communicating with the EPP server, hit the following button combinations, depending on your operating system:

Windows: Control + Alt + I

MacOS: Option + Command + I

Linux: Control + Alt + I

If the client is communicating, it will show the time and date of the last server connection on the bottom right corner, the server IP and Port, if it"s connected, and the time and date when the policies were last received:
