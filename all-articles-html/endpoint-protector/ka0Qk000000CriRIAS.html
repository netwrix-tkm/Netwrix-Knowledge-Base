[UPDATE]- How to use the Utility tool for the Audit Log Backups

In order to have good performances of the Endpoint Protector Server we created a tool called Utility tool, that will basically recreate the exact view from the Reports menu.

By having this tool you will no longer need to store logs for long periods on the server, but instead you can create Audit Log Backups that will be opened with the help of the Utility tool, which download link can be found below:

https://download.endpointprotector.com/ReportTools/EndpointProtectorUtilityTools.zip

Procedure for using the EPP utility tool for the exported logs and shadows:ï¿½

open EPP Utility tool> Drag and drop Audit Log Backup files into the Utility tool

Wait until the Audit Log Backup is Loaded 100% and then you can use the tool to navigate through the different type of logs you would like to check with the help of the filters and you will also be able to export them exactly as from the Reports and Analysis menu.
