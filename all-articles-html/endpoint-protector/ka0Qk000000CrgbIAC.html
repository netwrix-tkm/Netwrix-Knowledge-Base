[UPDATE]- View Patch Notes of previously applied updates on the EPP server

In order to view all the Release Notes for all the EPP server versions, you can check the release history in the link below:

https://www.endpointprotector.com/support/endpoint-protector-release-history

If you would like to see theï¿½Release Notes, bug fixes, enhancement,s and new features for each EPP server version that was previously applied on your end, please follow the steps below:

  1. Please go to Dashboard -> Live Update:
  2. Please select View Applied EPP Software Updates.
  3. A list with all the EPP Software Updates should appear:
    1. Either right click the upgrade version you would like to see the Release Notes and select "View Release Notes" or select the icon from the right side column called "Action" for the desired version.
    2. The Release Notes should be displayed for that version.

The Dashboard can be found in the Netwrix Endpoint Protector server UI.
