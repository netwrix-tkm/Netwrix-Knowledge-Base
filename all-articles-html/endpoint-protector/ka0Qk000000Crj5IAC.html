Set rights for a specific device

Question:

How do I set rights for a specific device?

Answer:

In order to set a specific right for a single device for a computer/user or a group:

  1. You will have to access Device Control > Devices > Filter for the device in cause > Click on the actions button > Manage rights.
  2. Click on the "Add" button > Select the entity for which you are going to allow the device (User/Computer/Group) > Select the Device Right for the device in cause > Click on the "Next" button.
  3. From the list with Users/Computers or Groups you will have to select the entity for which you want to give the new right for the device in cause.
  4. Tick the check boxes on the left side of the entity > Click the "Save" button

�

In order to make sure that you set the correct right, you can access again Device Control > Devices > Filter for the device in cause > Click on the actions button > Manage rights.

�

In order to cancel/remove the device rights for a specific computer/user or a group:

  1. You will have to access Device Control > Devices > filter for the device in cause > Click on the actions button > Manage rights.
  2. From the list of entities where the device was given a specific right you will have to click on the actions button and remove the entities where you do not need the specific right anymore.

�

In order to check if the right was canceled you will have to access Device Control > Devices > filter for the device in cause > Click on the actions button > Manage rights.
