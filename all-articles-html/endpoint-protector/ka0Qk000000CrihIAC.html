[UPDATE]- How to set up Smart Groups

The Smart Groups can be used in order to add computers/users automatically to a specific group based on their name.

In order to set up the Smart Groups you will have to access System Configuration > System Settings >Enable Smart Groups.

Once the Smart Groups are enabled you will have to access Device Control > Groups > Mark the group as a smart groupï¿½> Name the group > Select the department from where you want to add the computers/users > Select the entities that you want to add to the group (Computers or Users).

In order to select which entity will be included in the group you should type a specific part of the entities name

Type the names of the computers which should be added to the group using the wildcard (*) based on the rule below:

XYZ* if the name starts with XYZ

*XYZ* if the name contains XYZ

*XYZ if the name ends with XYZ

If there are computers which should not be included in the same group, you can also exclude some names so the computers will not be included in the group.
