[UPDATE]- Is it possible to block an internal card reader on Mac with Endpoint Protector?

Endpoint Protector can block internal card readers on Mac computers as well. However, inside the Endpoint Protector interface, a card reader is recognised as a USB Storage device.

This means that, for example, if you set "Allow" rights to the USB Storage Device class, then, subsequently, the internal card readers will also be set to "Allow" and, therefore, the user will be able to access the information from the inserted cards.ï¿½

In case that the rights for the USB Storage Device class are set to "Deny", then the internal card reader will be blocked.
