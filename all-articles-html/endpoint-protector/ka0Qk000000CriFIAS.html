[UPDATE]- How to upgrade the EPP server with Live Update

In order to upgrade the EPP server via the Live Update functionality please follow the below steps:

  1. Please go to Dashboard -> Live Update and select the Check Now button:

    2. 2. Select the update that will appear below by ticking the checkbox in front of it.

    3. Select Apply Update.

  2. Wait a few minutes and a progress bar should appear that will track the EPP server upgrade.
  3. After the progress bar reaches 100% please wait 1-2 minutes and then refresh the browser webpage.
  4. The EPP server should be upgraded by 1 version. You can also check the current/new EPP server version in the bottom right corner of the EPP server UI:

    5. Note: The live Update upgrade process requires the EPP server to be connected to the internet. If the server is Offline, you can use Offline Patches in order to upgrade.
