Capturing Client Logs using the Support Tool-v2 on a Windows Machine

Question:

How do I Capture Client Logs using the Support Tool-v2 on a Windows Machine?

Answer:

Please follow the below guide:

  1. Download our Support Tool from here:ï¿½https://download.endpointprotector.com/Support_files/Support_Tool_V2.zip
  2. Extract the contents of the .ZIP archive in a new folder.
  3. Run the executable file "SupportTool_x64.exe" with administrator privileges.
  4. Click on the the drop down menu and select "Create Log File" and then click on "Execute".
  5. Click on the the drop down menu and select "Create DPI Log File" and then click on "Execute".
  6. Leave the support tool open and running in background and reproduce the issue or retest scenario of issue a few times.
  7. After the issue was reproduced / retested, return to our EPP Support Tool and click on "Export Files".
  8. From the dropdown menu please select Create Archive and click on "Execute".
  9. In the same folder where you extracted the Support Tool files you should have an archive containing all the log files .
  10. The next step would be to send the .ZIP archive to us.
