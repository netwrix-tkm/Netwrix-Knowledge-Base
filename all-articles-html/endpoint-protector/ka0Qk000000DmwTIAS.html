Why Smartphones Are Detected as Multiple Device Types

Question

How does Endpoint Protector identify smartphones, and why might multiple device types appear depending on manufacturer and configuration?

Answer

Endpoint Protector identifies smartphones based on how the operating system (OS) presents the device when it is connected. Modern smartphones function as full-featured computers with their own operating systems, and their connection behavior can vary by manufacturer and configuration.

For example:

  • Apple phones typically appear as a single device type when connected.
  • Some Android phones (such as Samsung devices) may appear as multiple device types.

Depending on the phoneï¿½s features and settings, the OS and Endpoint Protector may detect the following device types:

  • MTP Device: Most smartphones are detected as Media Transfer Protocol (MTP) devices in Device Manager.
  • USB Storage Device: If the phone uses an SD card and is configured to appear as a storage device, Endpoint Protector will detect it as a USB storage device.
  • USB Modem: If internet sharing (tethering) is enabled, the device may also be detected as a USB modem.

To manage access and apply the appropriate policies, review the device types detected in Endpoint Protector for each scenario.
