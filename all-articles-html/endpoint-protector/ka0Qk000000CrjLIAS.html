Set up Automatic License Realease

Question:

How do I setup Automatic License Release?

Answer:

In case you have licenses assigned to computers that are no longer active, and you want to make sure that the licenses are not used up unnecessarily, you can enable the Automatic License Release feature.

ï¿½

To set it up please log into the Endpoint Protector web console and follow the below steps:

  1. Go to System Configuration and System Licensing.
  2. In the Licensing windows please press View Licenses.
  3. Press the Automatic Release License button once you reach the View Licenses window.
  4. Toggle the Automatic Release License option, and in the Last Online drop-down, select either a pre-defined value or you can enter a custom value. The custom value can be set between 1 and 360 days:
  5. Once you press the Schedule button, all computers that were last seen longer than the defined value will have their license released.
