How can I exclude access events to the Calendar folder from the product reports?
Access events to the Calendar folder are excluded by default (this is set in the mailboxestoexclude.txt file located in C:\Program Files (x86)\Netwrix Auditor\Non-owner Mailbox Access Reporter for Exchange by default). However, the product only recognizes this file if agents are enabled, so make sure that the compression serviceï¿½option is enabled.

In order to exclude Contacts access from being reported please add the following line to the mailboxestoexclude.txt file:
*/Contacts*
