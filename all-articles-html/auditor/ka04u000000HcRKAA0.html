Error: "The following group policies could not be compared due to a snapshot reading error".

The Change Summary emails contain the following error message: "The following group policies could not be compared due to a snapshot reading error"
Netwrix Auditor uses native Windows tools like Microsoft Group Policy Management console during the data collection. When an issue appears in there it automatically shows up in Netwrix Auditor reports too. For example, some Group Policy objects can have an incorrect structure and can not be processed. To make sure that this issue relates to Group Policy objects themselves perform following steps:
  1. Click Start -> Run and type gpmc.msc on the computer where Netwrix Auditor is installed.
  2. Find there any problematic GPO that displayed in report.
  3. Switch to Settings tab, right-click and select Save Report.
  4. The error should occur during this process.
To troubleshoot this issue, please open problematic GPOs in Microsoft Group Policy Management console, review settings and fix the errors. In some cases, it is not possible to fix these incorrect GPOs and it is recommended to recreate them.
