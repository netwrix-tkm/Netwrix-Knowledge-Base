Error: "Unable to connect to the Database. Please check your database connection settings"

You get the following error when trying to upload data to the database using the DBImporter tool: Unable to connect to the Database. Please check your database connection settings
This error can happen when SQL settings for the DBImporter tool are misconfigured and different from the SQL setting in the Netwrix Auditor console.
In order to resolve this issue please perform the following steps: 1. Open the DB Importer tool (Start | Programs | Netwrix | File Server Change Reporter); 2. Click on the "Configure" button; 3. Then using the "Start" button open the Reports Configuration Wizard and step by step configure SQL settings. 4. Once completed you should be able to upload data to the database.
