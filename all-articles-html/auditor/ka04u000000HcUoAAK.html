404 Bad Request

The data collection returns the error message '404 Bad Request'.
Following to https://hostname:443 gives you a blank page or a malformed page.
Check if you have installed the following KB/Update/hotfix on Netwrix host: http://support.microsoft.com/kb/2661254 If you have installed the KB above, you need to run the following command: certutil -setreg chainminRSAPubKeyBitLength512 This will set your security length back down to 512 instead of 1024. Currently only 512-bit encryption is supported.
