Unable to Edit Existing Password Expiration Notifier Monitoring Plan

Symptoms

  • Administrators do not receive reports and password expiration notice emails for an existing Password Expiration Notifier monitoring plan.

  • Attempts to change any monitoring plan settings lead to the following error:

    The following error has occurred: The system cannot find the file specified.
(Exception from HRESULT: 0x80070002)

Cause

The Password Expiration Notifier monitoring plan is corrupted.

Solution

Recreate your Password Expiration Notifier monitoring plan. It is required to perform the setup manually since there's no option to copy the existing plan automatically.
