Restoring individual attributes with the Netwrix Auditor Object Restore for AD.

Does the Netwrix Auditor Object Restore for AD restore single attributes on single/multiple objects or does it restore just the whole object?
Netwrix Auditor Object Restore for Active Directory provides granular restore capabilities (object and attribute level). In other words, you can select to restore an object attribute rather than just restore the whole object to its original state. Note: Netwrix Auditor Object Restore for Active Directory is part of Netwrix Auditor for Active Directory and can be invoked with Start -> All Programs -> Netwrix -> Netwrix Auditor Object Restore for Active Directory.
