Event ID 1259 in Health Log

Symptom

The following error message under Event ID 1259 is prompted in Health Log for your SharePoint monitoring plan:

Unable to report changes correctly, as the previous configuration snapshot has been removed or is partially invalid

Cause

The data collection was launched manually while the initial data collection process has been already in progress after the monitoring plan creation. This has led to data collection failure.

Resolution

Restart the data collection ? in the main Auditor menu, select Monitoring Plans > select %SharePoint_plan% > Edit > click Update.
