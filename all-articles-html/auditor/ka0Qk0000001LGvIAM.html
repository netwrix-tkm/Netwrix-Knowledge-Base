Event ID 1255 in Health Log

Symptom

The following error message under Event ID 1255 is prompted in Health Log for your SharePoint monitoring plan:

Site collection %URL% cannot be found.

Causes

  1. An invalid site collection URL was specified during the monitoring plan creation.

  2. The affected site collection has the Locked ï¿½ No Access status.

Resolution

  • Cause #1 ? Navigate to Monitoring Plans > select the %affected_SP_monitoring_plan% and click Edit > click Edit item in the right pane. In the Changes tab, verify the affected collection URL under the Specific SharePoint objects section.

  • Cause #2 ? In SharePoint Central Administration, navigate to Application Management > Configure Quotas and Locks > %affected_collection%, and change the status to Not locked.
