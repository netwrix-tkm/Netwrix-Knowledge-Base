Security Log Overwrites Occurred on This DC

Symptom

The following error is prompted either in daily Activity Summary emails or Netwrix Auditor Health Log:

Security log overwrites occurred on this DC since the last data collection.
Please increase the maximum size of the Security event log

Causes

  • Misconfigured Security event log size.

  • Insufficient DC resources affecting the operation.

Resolution

  • To configure the Security event log size and retention settings, refer to the following article: Windows File Servers ? Configure Event Log Size and Retention Settings ? v10.6.

  • For additional information on sample deployment scenarios and corresponding hardware requirements, refer to the following article: Requirements ? Sample Deployment Scenarios ? v10.6.
