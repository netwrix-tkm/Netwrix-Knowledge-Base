Error: A transport-level error has occurred when sending the request to the server

The error message is displayed when assigning users read access permissions to view reports.
This error occurs because the Clear all database entries functionality has been applied. It allows you to clear all audit data and the information on user access permissions.
To resolve the issue, do the following:
  1. Click OK on the error message.
  2. Close the SQL Database Access dialog.
NOTE: If you proceed with the user assignment without closing the SQL Database Access dialog, the following error message will be displayed: The role 'NetWrix_ReportViewer_ReadOnly' does not exist in the current database. Changed database context to 'NetWrix_Windows_Server_Change_Reporter'.
  1. Click the Assign link and specify the users who can access the database.
