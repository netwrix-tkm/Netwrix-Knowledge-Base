Error : "Encountered an error while parsing. Incorrect document syntax. File <path to file>"

You get the "Encountered an error while parsing. Incorrect document syntax. File <path to file>" error in the summary reports and sessions for Group Policy auditing. For example: "Encountered an error while parsing. Incorrect document syntax. File <DC name>sysvol<domain name>PoliciesPolicyDefinitionsEnhancedStorage.admx, line 1, column 1"
Solution for Group Policy during the data collection process uses the native Windows tools like GP Management Console, and if any issues appears it will shows up in summary reports and sessions for Group Policy too.
Error means that files or administrative templates have incorrect structure and cannot be processed.
To troubleshoot this issue, please open these GPOs in Microsoft Group Policy Management Console, try to view their settings and fix the errors. In some cases it is not possible to fix it and it is recommended to recreate them from scratch.
