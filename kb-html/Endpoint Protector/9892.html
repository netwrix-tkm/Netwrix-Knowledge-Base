[UPDATE]- What is the file shadows format for documents sent to Printers?

When enabling the Advanced Printer and MTP Scanning feature, due to the high number of ways a document can be printed, the current version of the feature will save documents as a .txt.

This is because of the textual representation of the document itself and because Endpoint Protector inspects the confidential data defined within the Content Aware Protection and Device Control Policies.

There are however situations where the transferred file will be saved in its original format, such as a .pdf.
