How to Update to macOS 12 (Monterey) While Using Endpoint Protector Client and Deep Packet Inspection

   Instructions

 

  1. Update the Endpoint Protector server to version 5.9.4.2.
  2. Go to System Settings -> Server Certificate Stack -> Regenerate Server Certificate Stack and hit the Regenerate button in order to regenerate the server and the client CA certificate.
  3. Download the new CA certificate:

           * From System Configuration > System Settings > Deep Packet Inspection Certificate.

           * Toggle the option and confirm by selecting "Yes" in the pop-up window.

           * Click "Download Client CA Certificate," and the ClientCerts file will download to your local directory.

 

   4. On your macOS, open the Keychain Access application and go to System.

 

   5. Decompress the downloaded ClientCerts file.

   6. Select cacert.pem ?le and drag and drop it into System > Keychain Access.   

7. Double click the X from the newly added certi?cate and from the Trust section, select Always Trust.       

 

 8. Save the changes.

 9. Update the macOS Endpoint Protector Client to version 3.0.3.1009.

10. Restart the PC.

11. Update to Mac OS 12 (Monterey).
