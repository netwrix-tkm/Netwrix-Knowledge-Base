Installing or Uninstalling the Windows epp agent from the terminal / command prompt / sccm

Question:

How do I install or uninstall the Windows epp agent from the terminal / command prompt / sccm?

Answer:

 

For uninstalling the EPP agent please use the following syntax from the windows command line :

msiexec.exe /x "path_to_msi" ADMIN_PASSWORD_0="your_uninstall_password" REBOOT=ReallySuppress REMOVE_PROP=1 /qn

 

Note :

You will have to input your uninstall password in above command only if such a password was previously configured from server UI, otherwise you can include an empty string for field ADMIN_PASSWORD_0.

 

For installing the EPP agent please use the following syntax from the windows command line:

msiexec.exe /i "path_to_msi" WSIP="EPP_server_IP" WSPORT="443" DEPT_CODE=defdep /q REBOOT=ReallySuppress

 

E.g. :

msiexec.exe /i "C:\EPPClientSetup.5.9.3.0_x86_64" WSIP="192.168.7.108" WSPORT="443" DEPT_CODE=defdep /q REBOOT=ReallySuppress
