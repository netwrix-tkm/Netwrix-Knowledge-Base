[UPDATE]- End to End encryption -Whatsapp ,Telegram Webb and other applications

This is a known limitation. The epp client intercepts the files uploaded on the web browser to WhatsApp and Telegram, but the problem is that the files are encrypted (we cannot scan the original file, we will scan the encrypted file in which we will not find anything).

The files can be scanned/intercepted by epp if the DPI is disabled. Another suggested workaround is to block the mentioned domain "web.whatsapp.com" and "web.telegram.org".

Please afterwards use their desktop versions which is not monitored via DPI (and the files will be blocked).
