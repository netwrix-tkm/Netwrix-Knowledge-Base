[UPDATE]- How to enable EPP Easylock update and file shadowing for Removable drives

Easylock application is automatically updating once the following option is enable on the EPP web console:

Open EPP console, navigate to Enforced Encryption tab and under the Settings you will find the Update option.

To enable file shadowing for Removable drives, first you will have to navigate to the Global Settings from the EPP web console, then under the File Tracing submenu enable the File tracing and File Shadowing .

The second step will be to enable the

File tracing

option from the EPP web console under Enforced Encryption menu -> Easylock .
