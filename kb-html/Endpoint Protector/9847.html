Setting up a AD administrator sync

 

To bring your AD administrators to EPP you can set up an Active Directory Authentication feature.

 

Please follow the below steps:

  1. Please log into the EPP Web console and go to System Configuration -> System Settings and in the Active Directory Authentication section please fill in the required fields.
  2. Check the "Enable Active Directory Authentication" check box to enable the feature.
  3. Select the connection type depending on your AD server settings.
  4. Fill in the IP and port used by the server.
  5. In the Domain Name field, just add the domain controller without an OU (ex. DDC=domain,DC=local for "domain.local").
  6. The account suffix used by the administrator directory (ex. @domain.local). In some cases, the domain will be required before the username (ex. DOMAIN\User).

    Active Directory Administrators Group can contain any other groups of users except for the primary groups which are limited from this action by Microsoft. Only users that are part of this AD group will be synced and imported as Super Administrators for Endpoint Protector. Any additional administrators(with different access control levels) can be created manually from the System Administrators section.

  7. Next scroll to the bottom of the page and press the "Save button" to save all changes. This action will prompt a banner at the top of the page with a success message.
  8. Return to the Active Directory Authentication section and test the connection, and after confirming the connection is working you can press "Sync AD Administrators" to bring the administrator accounts into EPP.
