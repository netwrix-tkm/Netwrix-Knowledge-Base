[UPDATE]- Endpoint Protector Administration Interface session timeout

Overview
In Endpoint Protector Server releases before 5.7.0.0, the session timeout for the Management User Interface is fixed at 15 minutes. Starting with version 5.7.0.0, you can configure the session timeout between 5 and 60 minutes.

Instructions

  1. Navigate to System Configuration > System Settings in the Administration Interface.
  2. Scroll down to the Session Settings section.
  3. Set the Session Timeout (min): to a value between 5 and 60.
  4. Click Save to apply the changes.
