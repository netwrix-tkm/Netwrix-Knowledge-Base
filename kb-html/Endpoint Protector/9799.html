How to activate EPP Mail Add-in required for Domain Allowlist on Outlook for Windows

In order to use the Allowlist for Domains in Outlook for Windows, you will need to make sure that the Epp Mail Add-in is enabled in Outlook.

The first step you will need to pay attention to is that the Epp Mail Add-in is included when you download the client from the User Interface, as presented in the screenshot from below:

The second step would be to make sure that the EPP Mail Add-in is active. For this, please open your Outlook and check that the Add-in is present in the Active list of Add-ins from Outlook:

Outlook Options > Add-ins > list of Active addins.

If the Add-in is not active please activate it and then restart the Outlook in order to preserve the new settings.

Now the Domain Allowlist for Outlook should work on the Domains you added under Denylists and Allowlists > Allowlists > Email Domain.

Please make sure that the Email Domains are also selected inside the Content Aware Policy.

Edit the policy -> Expand the Policy Allowlists section -> E-mail Domain -> select the E-mail domain dictionary -> save.

Non-Windows Endpoints require Deep Packet Inspection to be enabled in order to use this Email Domain Allowlist.
