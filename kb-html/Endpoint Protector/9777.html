[UPDATE]- How to view Release Notes, bug fixes, enhancements and new features for each EPP server version

If you would like to see the Release Notes, bug fixes, enhancements and new features for each EPP server version, you can access the below link and see them for all the EPP server versions:

https://www.endpointprotector.com/support/endpoint-protector-release-history

If you would like to see which ones where applied on your EPP server version please follow the below steps:

  1. Please go to Dashboard -> Live Update:

2. Select View Applied EPP Software Updates:

  1. You can view the Release Notes and details regarding each individual EPP Server version that was applied on your EPP server by selecting the Actions icon corresponding to the version you would like view/check.
