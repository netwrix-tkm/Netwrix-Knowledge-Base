How can I whitelist a particular device, allowing access globally?

To globally allow a specific device, such as a USB storage device, within the Endpoint Protector interface, you can create a custom class and identify the device using criteria like Device Type, Device Class (VID, PID, etc.), or Unique Device Identifier (serial number).
 
Steps to Create a Custom Class:
 
1. Access Custom Classes:
- Navigate to Device Control ? Custom Classes and click Create.
 
2. Configure Custom Class:
- Provide a name for your custom class.
- Ensure the Custom Class Status is set to ON.
 
3. Add Device to Custom Class:
- Click Add in the Custom Class Device List.
 
4. Device Wizard (Step 1/2):
- Device Type Selection: Choose from a comprehensive list, which includes:
Unknown Device, USB Storage Device, Internal CD or DVD RW, Internal Card Reader, Internal Floppy Drive, Local Printers, Network Printers, Windows Portable Device (Media Transfer Protocol), Digital Camera, BlackBerry, iPhone, iPad, Bluetooth (with subcategories like Radio, Keyboard, Mouse, Smartphone, Headphones), iPod, VM USB Device, Serial ATA Controller, WiFi, FireWire Bus, Serial Port, PCMCIA Device, Card Reader Device (MTD, SCSI), ZIP Drive, Teensy Board, Thunderbolt, Infrared Dongle, Parallel Port (LPT), Thin Client Storage (RDP Storage), Additional Keyboard, USB Modem, Android Smartphone (Media Transfer Protocol), Chip Card Device, Audio Device.


- Device Right: Select "Allow Access" or "Deny Access" or as appropriate.

- Choose which Method for Adding Device then select Next


5. Finalize Setup Device Wizard (Step 2/2):
- "Existing Device (Wizard)": Use the check mark box to select one or more Device Types, then press Save.
- "New Device (VID, PID, Serial Number)": Enter the VID, PID, Serial Number, and Description, then press Save.
- "Device Serial Number Range": Enter the VID, PID, First Serial Number in range, Last Serial Number in range, and Description, then press Save.
- "Bulk List of Devices": Choose Enrollment Options, Type or Paste content, or Import content, then press Save.
