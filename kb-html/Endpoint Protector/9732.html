[UPDATED] - Optical Character Recognition (OCR) is not working

Optical Character Recognition (OCR) is not working

 

The Optical Character Recognition (OCR) feature can be found under "Device Control" > "Global / Groups / Computer Settings" page.

 

 

Please note that in order for OCR to work it has to be a good image quality, our recommendation is a minimum of 150 dpi (dots/pixels per inch) for the images in order for EPP to be able to scan the content. Good contrast also helps.

You can check the dpi of an image for example on Windows OS by using Paint application > Open image using Paint app > go to File > Properties > check Resolution field (in dpi).

Also make sure that you are using the latest Endpoint Protector client version available.
