[UPDATE]- How to use different Client Modes

Overview
The Endpoint Protector (EPP) agent can operate in six different client modes, each catering to specific user scenarios.

Client Modes

  1. Normal Mode

    • The EPP agent icon is displayed, and notifications are generated.

  2. Silent Mode

    • The EPP agent icon is displayed, but no notifications are generated.

  3. Hidden Icon

    • The EPP agent icon is hidden, and no notifications are generated.

Specific Modes

For more specific scenarios, the following modes are available:

  1. Transparent Mode
  2. Stealth Mode
  3. Panic Mode

Recommendations

  • The first three modes cover most typical use cases.
  • We strongly recommend contacting our support department before setting agents to Transparent, Stealth, or Panic modes, as these are intended for very specific situations.
