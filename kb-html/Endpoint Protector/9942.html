[UPDATE]- Why doesn't the WiFi connection work on a computer running Endpoint Protector Client, even if the WiFi rights are set to "Allow Access"?

In case that there is no local network connection on the client computer and the WiFi connection was disabled from the Endpoint Protector interface, by changing the WiFi rights to "Allow Access", your WiFi connection will still not function as the computer does not have an Internet connection enabled to allow the Server to communicate the WiFi Rights changes to the Endpoint Protector Client.
