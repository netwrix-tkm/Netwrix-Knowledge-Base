[UPDATE]- Security Backend Updates

You can use the Security Backend Updates section to check and apply different types of security updates, view information on recent updates checked or installed, and a list of updates available.
Note: The security update options will only be available for customer-hosted instances (e.g. AWS, Goggle, etc.) with the exception for Operating System and Kernel upgrades.
Important: Updates are not tested beforehand but are pulled from the official Linux repository.

Security Backend Updates can be found on the Epp server interface at "Dashboard > Live update".

To ensure the updates will not harm the system, follow these actions:

  1. Test the updates in a test environment first
  2. Make a VM snapshot
  3. Make a system backup from the System Maintenance, the System Backup v2 section

Select one of the security updates type available and then click Check Updates:

  1. Security this will update all security-related updates of installed packages (Critical and High)
  2. Other, this will download and apply any update available to 3rd party libraries, kernel, OS packages and MySQL database
  3. All Updates this will download and apply Informational and Optional/Unclassified updates

If there are updates available, click Apply Updates.

Important: Due to patching nature, some updates may automatically restart the Endpoint Protector server or other sub-services in the background.
If by any chance the updates do not apply, then please create a ticket in our Support Portal to investigate this matter further.
