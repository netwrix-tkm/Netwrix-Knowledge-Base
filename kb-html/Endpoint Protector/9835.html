Installing Linux client

Question:

How do I install the EPP Linux client?

Answer:

First, you will need to request the EPP client from us via a ticket raised in the Support Portal. Please mention the Linux flavor you need.

Â 

  1. After you download the client and extract it, please edit options.sh to remove # from the lines below(remove the # from all 6 lines).
  2. Also replace "set.this.to.the.endpointprotector.server.com" with your EPP server IP:

    #EPPCLIENT_WS_SERVER=set.this.to.the.endpointprotector.server.com

    #export EPPCLIENT_WS_SERVER

    #EPPCLIENT_WS_PORT=443

    #export EPPCLIENT_WS_PORT

    #EPPCLIENT_DEPARTMENT_CODE=defdep

    #export EPPCLIENT_DEPARTMENT_CODE

  3. Save the changed file.
  4. Install the EPP client using the install.sh file.
