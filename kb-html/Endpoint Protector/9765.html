[UPDATE]- Setting up SMTP - Gmail

To set up SMTP:

- Please open the EPP Web Console
- Go to System Configuration -> System Settings
- In the Email Server Settings, please choose SMTP from the Email Type field.
After that, please fill in all fields with the correct options, and make sure you use the correct port for the encryption type used.
 E-mail Type: SMTP
 Hostname: smtp.gmail.com
 SMTP Port: 587
 Require SMTP Authentication: (checked)
 Username: add your email address
 Password: add the app password generated in your Google Account as
previously detailed
 Encryption Type: TLS
 Send test email to my account: (checked)
 No-reply email address: Default


Select Custom in the custom no-reply e-mail address, and you can either leave the default email address or fill in with your own.

When using Gmail as your SMTP provider, please make sure you are using an App password instead of the account password.

To create and use app passwords, please follow the steps from this link:
https://security.google.com/settings/security/apppasswords

- Go to your Google Account.
- Select Security.
- Under "Signing in to Google," select 2-Step Verification.
- At the bottom of the page, select App passwords.
- Enter a name that helps you remember where you will use the app password.
- Select Generate.
- To enter the app password, follow the instructions on your screen. The app password is the 16-character code that is generated on your device.
- Select Done.

When all fields are filled, scroll to the bottom and press Save, then return and select "send test email to my account" and press Save again.
If all fields are correctly filled, after pressing sav,e you should see a green banner stating that a test email was sent
