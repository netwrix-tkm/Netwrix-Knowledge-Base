[UPDATE]- How to correctly install Endpoint Protector on macOS Monterey (12.0)

The steps to install the EPP client on macOS Monterey are as follows:

  1. Install the EPP client normally
  2. Go to Apple Menu > System Preferences.
  3. Go to Security & Privacy.
  4. Click on the Privacy Tab.
  5. Click Full Disk Access section in the sidebar.
  6. Click on the lock icon and enter the admin password.
  7. Add Endpoint Protector.
  8. Then click on Accessibility.
  9. Check EPP Notifier.
