How to Set an Uninstall Protection Password

Overview

This article explains how to configure an uninstall protection password in Endpoint Protector. Enabling this setting ensures that only authorized users can remove the client from managed devices.

Instructions

Follow the steps below to configure these settings:

  1. Access the Endpoint Protector Console.
  2. Navigate to System Configuration > System Security. 
  3. Enter the desired password in the designated field.
     
  4. Save the changes.
  5. After Endpoint Protector clients update their settings, users will be required to enter this password to uninstall the client.
