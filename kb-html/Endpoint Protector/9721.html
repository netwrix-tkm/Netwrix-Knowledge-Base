Exporting Event Logs from "Reports and Analysis"

 

If you would like to export selected event logs from the Endpoint Protector server under "Report and Analysis" then you can do so by adding a filter for the desired logs and then use the "Create Export" button at the bottom of the list.  See below screenshot as an example where the filter is applied for "Blocked" events.

 

This will generate a .csv or .xlsx file that you can download from "View Export List".
