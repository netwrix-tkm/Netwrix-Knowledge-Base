[UPDATE]- Restarting the EPP Client without restarting the computer

For each of the main OS there is a way to restart the services without being necessary to restart the computer.

Please see the point below:

For WindowsOS you can restart the 'Endpoint Protector' service from the Windows Task Manager.

For MacOS you should be able to restart the EPP process using the following commands in a Terminal :

sudo launchctl unload /Library/LaunchDaemons/com.cososys.eppclient.launchdaemon.plist

sudo launchctl load /Library/LaunchDaemons/com.cososys.eppclient.launchdaemon.plist

For LinuxOS you can restart the service by using the command in a Terminal:

sudo service epp-client-daemon restart
