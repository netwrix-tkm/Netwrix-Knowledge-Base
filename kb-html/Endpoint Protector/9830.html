Set up URL Allowlists and Denylists

Question:

How do I set up URL Allowlists and Denylists?

Answer:

In order to use the URL Allowlists and Denylists you will have to make sure that Deep Packet Inspection is enabled in Device Control > Global Settings.

 

URL Denylist

The URL Denylist is used in order to block the access on specific websites.

In order to set up the URL Denylist you will have to:

  1. Access Denylists and Allowlists > Denylists > Domain and URL
  2. Click the Add button in order to create a list with URLs that you want to block > Set a name for the list > Set a description for the list > add the URLs > Click on the "Save" button

Notes:

You can add the full URL: https://example.com or you can use the * which is a wildcard that stands for anything before or after the domain: *example*, this will prevent the users to access any website which has the word "example" in the URL.

 

In order to apply the Denylist to the users or computers you will have to include the Denylist in the Content aware protection policy which is assigned to the specific users or computers.

 

You will have to access Content Aware Protection > Content Aware Policies > edit the policy > make sure that the web browsers used by the users are included in the Policy Exit Points > Scroll to the policy Denylists > click on the Domain and URL tab > tick the check box next to the Denylist that you want to use > click on the "Save" button.

 

URL Allowlist

The URL Allowlist is used in order to allow the upload on specific websites.

In order to set up the URL Allowlist you will have to:

  1. Access Denylists and Allowlists > Allowlist > Deep Packet Inspection.
  2. Click the Add button in order to create a list with URLs that you want to allow the upload on > Set a name for the list > Set a description for the list > add the URLs > Click on the "Save" button

Notes:

You can add the full URL https://example.com or you can use the * which is a wildcard that stands for anything before or after the domain *example* this will allow the users to upload anything on any website which has the word "example" in the URL.

 

In order to apply the Allowlist to the users or computers you will have to include the Allowlist in the Content aware protection policy which is assigned to the specific users or computers.

 

You will have to access Content Aware Protection > Content Aware Policies > edit the policy > make sure that the web browsers used by the users are included in the policy exit points > Scroll to the policy Allowlists > click on the "Deep Packet Inspection" tab > tick the check box next to the Allowlist that you want to use > click on the "Save" button.
