[UPDATE]- Logs - How to view all entries when the number exceeds the records limit

Netwrix Endpoint Protector server offers the possibility to set a maximum number of 10 000 entries to be displayed for each report.
In case the number of available logs exceeds the records limit in place, the entries that are not visible on the UI can still be viewed by using one of the below methods:

Option 1: Using filters to narrow down the search results.

On any table from "Reports and Analysis":
- click on Filter,
- add the details you need,
- click Apply.
The server will show the logs based on the filter.

Option 2: Using the Create Export option The exported file can be saved afterwards from the Export Log List available by going to View Export List.
- at the bottom of the table click "Create Export",
- select the file type you want (.csv. or .xlsx),
- wait for the notification that the export was created,
- click on view export list,
- under actions click on the download button.
