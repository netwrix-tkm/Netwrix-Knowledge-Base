[UPDATE]- How to enable sensitive data protection on EPP Server

Security Password for Sensitive Data Protection

From this section, you can set a password to protect sensitive data to provide additional security.

This security feature protects the following sections of Endpoint Protector with a security password:

Reports and Analysis > Logs Report, File Tracing, Content Aware Report, Admin Actions, Published Computers, Published Users, Published Devices, Statistics.

System Maintenance >File Maintenance, System Backup.

To set a security password for Sensitive Data Protection follow the below steps.

Navigate to "System Configuration" > " System Security" under the section "Additional Security Password for Sensitive Data Protection" enter a strong password in the box of "New Password" and then re-enter the chosen password in the box of "New Password (confirm)"

Click on "Save"
