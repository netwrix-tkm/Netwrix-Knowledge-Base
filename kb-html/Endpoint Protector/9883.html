[UPDATE]- How is File Copy event working on Linux distributions?

Starting with the 1.7.1.2 client version, File Copy event is supported on all Linux distributions for which clients are available, such as Ubuntu 18.04 LTS, Ubuntu 20.04 LTS, Rhel 8.1, Rhel 8.3. The event is made despite the kernel version used by the distribution, so even if the kernel is lower than 5.1 or higher than 5.1, the event will be created.

At the moment, File Copy event is made only when transferring from PC to USB. In logs we report only the file that was copied, the source and the destination paths are not displayed.
