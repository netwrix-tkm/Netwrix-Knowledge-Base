How do I upgrade to the latest version of Endpoint Protector?

Overview

To stay informed about future version releases of Endpoint Protector software, please visit the Netwrix Community Endpoint Protector link and click the Subscribe button:

Netwrix Community Endpoint Protector


Instructions

1. Go to the "Announcements" section and select the latest Endpoint Protector Version Release.

2. On that page, locate the banner "Download Netwrix Endpoint Protector x.x.x.x here!" and double-click it to download the offline upgrade installation package to your local directory.

3. Create a snapshot of your current Endpoint Protector Appliance VM.

4. Open a web browser and access the console's web interface.

5. Log in to the console's web interface with an account that has administrator privileges.

6. In the Dashboard section, choose Live Update..Â 

7. Under EPP Software Update, click Offline Patch Uploader.

8. Navigate to and select the downloaded update file.

9. Select the Upload Patch button.

10. A green banner stating that Patch applied successfully! will appear above the Offline Patch Uploader section. Select Back and wait for the installation bar to fill.

11. Refresh your browser and verify the update by checking the bottom right corner for the current Endpoint Protector version number.
