[UPDATE]- OTP device code compatibility

Overview
This article outlines the recent improvements to the Offline Temporary Password (OTP) feature and the implementation of 8-digit device codes, aimed at facilitating multiple device usage per company.

Availability
The 8-digit device code feature is available with Endpoint Protector agents version 5.7.2.4 for Windows and 2.5.1.3 for macOS.

Behavior Based on Endpoint Protector Agent Version

  1. For Versions 5.7.2.4 (Windows) or 2.5.1.3 (macOS) and Newer:

    • The user interface will display an 8-digit device code.
    • OTP codes will be generated based on the 8-digit device code.

  2. For Versions Older Than 5.7.2.4 (Windows) or 2.5.1.3 (macOS):

    • The user interface will display a 4-digit device code.
    • OTP codes will be generated based on the 4-digit device code.

Compatibility

  • Codes generated from the user interface can be used with older versions.
  • The Endpoint Protector Server will re-compute the OTP device codes and display them in the appropriate format according to the agent version.
