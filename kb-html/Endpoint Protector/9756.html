[UPDATE]- How to enable Two-Factor Authentication for System Administrators

Please follow the below steps to enable two-factor authentication for a system administrator.

  1. Edit the system administrator which you want to enable two-factor authentication for in edit mode and then toggle the switch labeled "Two Factor Authenticator" to "On".
    - The System administrator can be found on the server under "System Configuration > System Administrators".
  2. Enter the code provided into / or scan the QR Code with the Google Authenticator app. Once the account is imported, take the authentication code and then enter it into Endpoint Protector's "Google 2FA Validation" then click on "Validate" Save the changes for the  system administrator. You will now see the notification "Two-Factor Authentication activated successfully!" highlighted in green and "Yes" will now be shown under the column "2FA" next to the system administrator(s) which have two-factor authentication enabled.
  3. Log out and then log in with the system administrator which two-factor authentication was enabled for. After the password is entered and authenticated, Endpoint Protector will now request that the authentication code be entered before the system administrator can proceed to log into the Server interface.
