[UPDATE]- How to find out if there are any computer duplicates in Endpoint Protector

In case you are having all of your licenses occupied, it is always good to check if you have any endpoint duplicates that are occupying some of the licenses, so that you can later on release those licenses by eliminating the doubles.

In order to do this, you will need to go to Device Control menu > Computers .

From the Filters, select Duplicates > Yes > Apply (just as presented in the screenshot below).

This will revert all computer duplicates. You can then filter for Licensed and Offline and delete the unwanted doubles, in order to gain free licenses.
