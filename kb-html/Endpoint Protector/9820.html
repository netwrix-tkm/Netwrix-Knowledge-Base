[UPDATE]- Create System Backup V2

The purpose of this guide is to create a backup of all settings, rights, policies, and integrations that you might be using, for cases of emergencies or for migrating to a new appliance.

The first step is to open the console and navigate to System Maintenance -> System Backup V2

On that page, please click on Create to begin

Please type in a Backup name and a description for this backup and then hit Save

After pressing the save button you will see a pop-up containing the Backup key. Please write that down somewhere as it will be mandatory if you need to use the backup for importing.

Once you confirm that you saved the system backup key, you will see the backup running in the List. Once it is done you can download the backup locally for safekeeping.
