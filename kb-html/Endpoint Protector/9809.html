[UPDATE]- How to generate logs for Mac EPP agents via terminal

It is possible to create the log file and DPI log file for the EPP Mac Client via terminal using the following commands:

sudo /bin/launchctl unload /Library/LaunchDaemons/com.cososys.eppclient.launchdaemon.plist   - this command will terminate the EPP Client

sudo touch /private/var/log/eppclient.log   - this command will create the EPP Client log file

sudo touch /private/var/log/eppsslsplit.log   - this command will create the DPI log file

sudo /bin/launchctl load /Library/LaunchDaemons/com.cososys.eppclient.launchdaemon.plist   - this command will restart the EPP Client
