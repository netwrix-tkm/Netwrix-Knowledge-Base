How to Use Client Update Mechanism

Overview

This article explains how to use the Client Update Mechanism to update Endpoint Protector clients. The Client Update Mechanism allows the Endpoint Protector Administrator to update up to 100 computers per hour.

Instructions

  1. In the Endpoint Protector Console, navigate to the System Configuration > System Settings page.
  2. In the Use Custom Hostname field, enter the desired hostname. For example: epp.hosted.endpointprotector.com.
  3. If the UI port is different from the default, enter the desired port in the Use Custom Port field. For example: 8455.
