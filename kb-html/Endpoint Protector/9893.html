Endpoint Protector Server can be installed as a Virtual Appliance or as a Hardware Appliance. For the complete list of supported server virtualization environments and client operating systems please follow this link: https://helpcenter.netwrix.com/bundle/EndpointProtector_5.9.4.2/page/Content/EndpointProtector/Install/Overview.htm

We cover a range of deployment methods, including:

  1. Physical and Virtual Appliances: Detailed instructions for configuration, deployment, and management.
  2. Cloud Platforms: Deployment strategies for AWS, GCP, and Azure, assuming existing cloud accounts and basic platform knowledge.
  3. Active Directory Integration: Leveraging Group Policy Objects for efficient client deployment.
  4. Third-party Management Tools: Utilizing JAMF and Microsoft Intune for streamlined deployment.
