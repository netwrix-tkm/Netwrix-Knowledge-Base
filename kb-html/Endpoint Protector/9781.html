What rights the colours represent in Endpoint Protector Server and Client

Server side:

  • Red means that the device is blocked in the system;
  • Green means that the device is allowed on computers or users;
  • Yellow means that device is allowed on some users or computers with restrictions.

Client side:

  • Red means that the device is blocked;
  • Green means that the device is allowed;
  • Yellow means that the device is set on Trusted Device Level 1 otherwise read only;
  • Orange means that the device is set on Read-Only;
  • Blue means that the device is setTD 1, TD 2, TD3, TD 1+ (all TD)
  • Turquoise means that the iPhone is set to Allow Charging
  • Grey means that the device was disconnected on Mac OS and Linux
