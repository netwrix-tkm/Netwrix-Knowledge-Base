[UPDATE]- Will Content Aware Protection detect and block documents with sensitive content (credit card numbers, SSN etc.) if transferred as archives?

Overview
The Content Aware agent can detect and block documents with sensitive content if they are archived as ZIP files. To ensure comprehensive protection against data leaks through other archive formats, follow the recommendation below.

Instructions

  1. Use the available File Type Filter settings in the Content Aware agent.
  2. Select all archive types to prevent leaks of sensitive data through archive files other than ZIP.
