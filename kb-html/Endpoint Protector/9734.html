[UPDATE]- Installing the agent when using Proxy on Linux

If you are using a Proxy server, please follow the below instructions to install the EPP agent to work with your proxy.

After downloading the installer provided by Support, please unarchive it and edit options.sh.

Please look for the Proxy settings section.

Just uncomment the last two lines and add the proxy settings as described in the first 2 comment lines.

The proxy address also works with a DNS or FQDN

After that please run the install.sh script and once the installation is complete, wait for a few minutes and check in the EPP Web Console in the Device Control -> Computers section to see the Computer added to the list.
