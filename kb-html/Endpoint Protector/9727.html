[UPDATED] - Can the Easylock app be opened on a computer without the Epp agent installed?

Can the Easylock app be opened on a computer without the Epp agent installed?

 

There is an option to set whether you want it to open only in the presence of the Epp agent or if it can freely be opened.

You can find it under "Device Control > "Global Settings" > "Easylock Settings" toggle the switch next to "Endpoint Protector Client presence required".
