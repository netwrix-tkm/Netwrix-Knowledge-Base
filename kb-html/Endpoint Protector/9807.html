[UPDATE]- How to check that EPP network extension is enabled on macOS endpoint via CLI and User Interface

To check via command line if the Endpoint Protector network extension is enabled and activated you will have to run the following command in terminal: 

"systemextensionsctl list"

The endpoint protector network extension should show both "Enabled" and "activated"

 

You can also check via the User Interface by navigating to:

"System Settings" > "Network" > "Filters" > "Filters & Proxies" > check if the Endpoint Protector network extension is present and that the status shows "Enabled"
