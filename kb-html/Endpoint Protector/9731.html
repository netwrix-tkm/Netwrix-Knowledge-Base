What to do if multiple licenses are assigned to the same Computer?

 

1. To view the license ID's associated with every machine, please go over to "System Configuration" > "System Licensing" > "View Licenses" as per the below image.

 

 

2. If duplicates exist whilst checking the licenses assigned to step 1 - Which means having more than one license associated with the same machine name, please go over to "System Configuration" - "System Settings" and enable the option called "Virtual Desktop Clones Support" as per the below image.

 

This setting is meant to to allow the Endpoint Protector server to identify the virtual desktop clone and interact accordingly with the Endpoint Protector client.
