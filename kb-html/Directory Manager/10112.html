How To: Change the support email address for the Application portal

Applies To:

Directory Manager 11

Business Requirement:

The application portal of Directory Manager has a Contact link at the bottom of its pages. This link launches an email application with a default email address in the To box, to send an email to admin or helpdesk for inquiries, support requests, or feedback.

We have a requirement to change this support email address, so that users can contact the local IT team in case of queries or issues with the portal.

Methodology:

We can change the default email address bound to the Contact link from the portal settings in Directory Manager Management Console.

Steps:

  1. In Directory Manager Admin Center, select Application -> Your required portal -> Settings.

  1. Click the Directory Manager Support tab.
  2. In the Support group/administrator's email address box, enter the email address for the group, user, or contact that will be responsible for responding to requests or inquiries from portal users. 

  1. Click Save button.

Now click the Contact link in the portal to verify that your specified email address is displayed in the To box in the default mailing application.
