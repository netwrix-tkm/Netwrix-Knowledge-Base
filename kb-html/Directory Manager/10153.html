How To Add attributes to the list in Export option in the Application portal

Applies To:

Directory Manager 11

Business Scenario:

We have a members export option set up in our environment. On clicking Export, only the default set of attributes appears to be available to select. We would like to add other attributes to the list of attributes available to be exported.

Methodology:

Adding more attributes to the list in Export in the application portal is achievable and requires changes in the design file of the portal. Follow the steps to add more attributes to the Export list.

By Default, the Export option provides the following attributes to export from the application portal. We will be adding the Common Name (CN) attribute to this list as an example.

Steps:

  1. Log in to the Admin portal of GroupID and navigate to the Applications tab.

  2. Open the settings of the Application in which you want to add the field and click on the Identity store name. 

 

  1. Navigate to the Import/Export tab.

  2. Click on (+) button and from here, select the field you want to add and give it a friendly display name.

 

  1. Once done, save your changes, and your field should be visible in the Export wizard of the Application portal.
