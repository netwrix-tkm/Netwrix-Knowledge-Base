How To: Convert a static group to a Smart Group and vice versa.

Applies To:

Directory Manager 11

Business Requirement:

You have a requirement to convert an existing Smart Group to a regular static group or you might want to convert an existing static group to a Smart Group.

Solution:

In the application portal of Directory Manager, you can simply clear the LDAP criterion of a Smart Group to convert it to a static group. Similarly, you can upgrade a static group to a Smart Group by applying an LDAP criterion to it.

Steps to convert a Smart Group to a Static Group:

  1. Log in to the application portal of Directory Manager.
  2. Open the properties of the Smart group which you want to convert into a static group.
  3. On the properties page, navigate to Smart Group tab.
  4. Click on Clear button in front of Smart Group query:-

 

  1. On clicking the Clear button, click on Clear query text to confirm and this will convert the group into a static group with making any changes in the already present membership.

 

Steps to convert a Static Group to a Smart Group:

  1. Log in to the application portal of Directory Manager.
  2. Open the properties of the static group which you want to convert into a smart group.
  3. On the properties page, click on the Upgrade to button. Two options will be shown. You can either convert the group to a smart group or a dynasty. Select Smart group here.

  1. On confirming this action, the query designer will open and you can specify the LDAP query as per your requirement.
  2. Once the query is confirmed, then it will process and convert the group into smart group.
