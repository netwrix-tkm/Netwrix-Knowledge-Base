How to specify Dell EMC Unity server as a monitored item

To audit Dell EMC Unity storage system, Netwrix Auditor needs to access its registry data. For that, FQDN of the Unity storage system is required. Enter it when adding a Dell EMC Unity storage as a monitored item in the Netwrix Auditor monitoring plan.

Alternatively, you can enter the IP address. In this case, consider that if your Netwrix Auditor server is located in another domain (untrusted domain or workgroup), then you should ensure any of the following:

  • A reverse lookup zone is configured on the DNS server for the Unity system - to resolve its IP address into FQDN.

-OR-

  •  Netwrix Auditor data collecting account has Read permissions for the OU=EMC NAS servers node in Active Directory.  


Otherwise, data collection will fail, reporting an error caused by network path not found.
