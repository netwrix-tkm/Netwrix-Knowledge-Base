Error: Invalid Login Response in VMware Monitoring Plan

Symptom

Netwrix Auditor prompts the following error for your VMware monitoring plan in Health Log:

There are errors while creating snapshot. No web api connection available. Reason: Invalid login response from %URL%

Cause

This was a known issue fixed in Auditor v10.6.12359 and later versions.

Resolution

Upgrade your Auditor instance to the latest version. Download the latest version of Netwrix Auditor from the Customer Portal · Netwrix 🡥. Refer to the following article to learn more about the upgrade recommendations: Installation  Upgrade to the Latest Version  v10.6.
