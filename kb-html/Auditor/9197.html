Specifying the SQL Server Instance Name

Question

What SQL Server instance name should you specify in the Audit Database settings in Netwrix Auditor?

Answer

Depending on the SQL Server setup, you should specify the instance differently. Refer to the following steps to establish the correct instance name:

  • In environments with a single default SQL Server instance (i.e., MSSQLSERVER), specify the server FQDN or NetBIOS name to connect to the SQL Server.

  • In environments with multiple SQL Server instances and named instances, specify the instance in the Server_Name\Instance_Name format. Refer to the following example:

    SQLMachine\SQLExpress

Learn more about establishing the instance name in Connecting to the Database Engine  Determine the name of the instance of the Database Engine · Microsoft 🡥.
