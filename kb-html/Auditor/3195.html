How to track Network interface changes on a server?

How do you track Network Interface changes on a server and what types of changes are tracked?
In order to track these types of changes you will want to use Netwrix Auditor for Windows Servers. When choosing Monitored System Components you will want to ensure that Hardware is checked (even if it is virtual).

The types of changes that are tracked are as follows:
  • Adapter Type
  • Configuration Manager Error Code
  • Default IP Gateway
  • DHCP Enabled
  • DNS Server Search Order
  • IP Address
  • Last Error Code
  • MAC Address
  • Network Connection Name
  • Network Connection Status
  • Service Name
  • Status
Please refer to the following guides for installation and configuration of Netwrix Auditor for Windows Servers:
