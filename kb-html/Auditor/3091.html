Can I configure administrator reports and user notifications to be in a language other than English?

Yes, you can modify reports and notifications by editing the corresponding template. To do this, perform the following steps:

1. Run NetWrix Enterprise Management Console.
2. Expand your Managed Object node and select Password Expiration Notifier. The settings for the module will be displayed in the right pane.
3. Click the Customize button next to the template that you want to edit.
4. Enter your text in any language you need and click OK to save the changes.

NOTE: The word SUBJECT, COLUMNHEADERS, COLUMNVALUES, HTML tags and value names in braces CAN NOT BE CHANGED.
