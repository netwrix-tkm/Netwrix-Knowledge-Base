Scenario: Cannot execute Search-AdminAuditLog after applying the Exchange Cumulative Update.   Cause: The update disables TLS 1.1 and older and then enables TLS 1.2. The .NET framework remains configured to use 1.1 or older, leading to the inability to run the Search-AdminAuditLog cmdlet.   Resolution: This Microsoft article explains how to update .NET Framework to use TLS 1.2 and disable TLS 1.1 and older.   If the issue is not resolved, we suggest reviewing this article for configuration validation and contextual information.
