List of all Netwrix Auditor .exe's

In order to compile a comprehensive list of current .exe's pertaining to the installation of Netwrix Auditor that you have deployed, continue with the following steps:

  • Launch an Administrative PowerShell session
  • Execute cd 'C:\Program Files (x86)\Netwrix Auditor'
  • Execute Get-ChildItem -Recurse -File *.exe |FT Name | Out-File exe_Names.txt
    • This will compile a list of all .exe's found in the Netwrix Auditor directory and export a .txt file to C:Program Files (x86)Netwrix Auditor
    • Replace *.exe with any other file extension you wish to look for

Note: If you are held accountable for keeping this list up-to-date for a company policy or compliance, be sure to run this again after upgrading, as more .exe's may be included or removed in future releases.
