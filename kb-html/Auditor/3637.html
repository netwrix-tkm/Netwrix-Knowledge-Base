Event ID 1225 in Health Log

Symptom

The following error message under Event ID 1225 is prompted in Health Log for your SharePoint monitoring plan:

Unable to make the configuration snapshot of the audited SharePoint farm

Causes

Netwrix Auditor is unable to collect farm configuration changes due to network connection problems, web service issues or configuration issues at the time of the scheduled or manual Activity Summary generation. Refer to the following list of possible errors prompted in Health Log for the affected monitoring plan:

  • Event ID 1204 ? Unable to establish connection to the remote WebService..

  • Event ID 1203 ? The following unexpected error has occurred: %error%.
