How to omit User Account Locked Out events from reports?
In order to exclude all "User Account Locked Out" please perform the following steps:
  • Navigate to the product installation folder
  • Open the unomitproplist.txt file
  • Remove the *.lockoutTime line OR just add # in the beginning of the line (#*.lockoutTime)
  • Save the file
