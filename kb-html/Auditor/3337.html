Advanced Reporting XML Parsing Error: not well-formed location

The product generates the summary report and delivers it via the email. Clicking the URL at the top of the email to retrieve more reports generates the following error in the browser window:
'XML Parsing Error: not well-formed
Location:
http://servername/Reports/Pages/Folder.aspx?ItemPath=%2fNetWrix+AD+Change+Reporter
Line Number 1, Column 2:<%@ Page language="c#"
Codebehind="Folder.aspx.cs" AutoEventWireup="false"
Inherits="Microsoft.ReportingServices.UI.FolderPage"
EnableEventValidation="false" %>
This error indicates that Report Manager installation is corrupted.
Try reinstalling the SQL Server. As a workaround, you can access it using the report server URL (e.g. http://servername/ReportServer).
