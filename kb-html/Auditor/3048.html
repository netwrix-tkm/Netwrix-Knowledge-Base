Collecting events for the workstations/servers from other domain

How can I collect events for the workstations/servers from other domain?
To collect events for the workstations/servers from other domain, do the following:
  1. Install NetWrix Event Log Manager on a computer with SQL Server and run it under local account.
  2. Make this account a member of the local Administrators group and grant it the required permissions for SQL Server.
  3. On all managed computers, create local accounts with the same account name and password.

NOTE: This method does not require trusts, but it cannot be applied to collecting events for domain controllers.
