Netrwrix Auditor Doesn't Work with Microsoft Exchange Server 2010

When trying to collect information from Microsoft Exchange Server 2010 (or earlier), Netwrix Auditor shows no activity records. There are no errors or warnings in Netwrix Health Log.

Cause

Starting with Netwrix Auditor 10.0, Auditing Exchange for Microsoft Exchange Server 2010 is no longer supported. This decision was made in accordance with the decision from Microsoft side to end support for Exchange 2010.

Resolution

In order to fix this issue we strongly recommend updating Exchange server to a newer version. Microsoft Exchange Server 2010 might contain unknown vulnerabilities and pose a threat to the enviroment, as Microsoft doesn't maintain it any longer. If you have Netwrix Auditor 9.96 (Build 8412) or older, you will be able to monitor Microsoft Exchange Server 2010, however in this case we would also recommend upgrading Netwrix Auditor after Exchange server has been upgraded.
