Warning: "Group Policy Management Console (GPMC) is not installed. Information on Group Policy changes will not be available..."

You are getting the following Warning in summary reports:
  • Windows Server 2003 XP:
Group Policy Management Console (GPMC) is not installed. Information on Group Policy changes will not be available. Group Policy Management Console (GPMC) can be downloaded from the following link: http://go.microsoft.com/fwlink/?LinkId=46570
  • Windows Server 2008 2008 R2 2012:
Group Policy Management Console (GPMC) is not installed. Information on Group Policy changes will not be available. Enable the Group Policy Management Windows feature.
  • Windows 7 8 Vista:
"Group Policy Management Console (GPMC) is not installed. Information on Group Policy changes will not be available. Group Policy Management Console (GPMC) can be installed as a part of Remote Server Administrative Tools.
During data collection Netwrix Auditor uses native Windows tools like GP Management Console. The error you are getting indicates that it is not installed.
To resolve this issue you need to install Group Policy Management Console (GPMC) on your Netwrix server. To do this perform the following steps depending on the OS:
  • For Windows Server 2003Windows XP:
Use the following link to download and install GPMC : http://go.microsoft.com/fwlink/?LinkId=46570
  • For Windows Server 20082008 R22012: The Group Policy Management Console (GPMC) is included with Windows Server 2008 and above. However, this feature is not installed with the operating system. Use the following Microsoft article to install it: http://technet.microsoft.com/en-us/library/cc725932.aspx
  • For Windows Vista7:
    1. Download and install the Windows Remote Server Administration Tools from http://www.microsoft.com/download/en/details.aspx?displaylang=en&id=7887. Once the download and installation of the toolkit is completed go through the following guidelines to enable it.
    2. Click Start -> Run and type Turn Windows features on or off.
    3. Navigate to Remote Server Administration Tools and enable the Group Policy Management Tools.
  • For Windows 8:
  1. Download and install the Windows Remote Server Administration Tools from http://www.microsoft.com/ru-ru/download/details.aspx?id=28972 . Once the download and installation of the toolkit is completed go through the following guidelines to enable it.
  2. Click Start -> Settings -> Control Panel -> Programs -> Turn Windows features on or off.
  3. Click Settings and then
  4. Navigate to Remote Server Administration Tools and enable the Group Policy Management Tools.
