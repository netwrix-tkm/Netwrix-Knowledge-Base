How to Identify Whether Auditor Server Can Receive Data from Meraki API?

! This article is currently under review. Some of the information below may not be up to date.

Symptom

When configuring Meraki Dasboard data source, the following error appears:

 
        Source: Network Devices Audit Service
        Event ID: 6023
        Unable to process item: The network path was not found.

No data was found despite troubleshooting efforts, leading to no progress.

Troubleshooting

To identify whether Netwrix server can receive data from Meraki API, open the command prompt as administrator and execute the following command:

curl https://api.meraki.com/api/v1 > 1.html
