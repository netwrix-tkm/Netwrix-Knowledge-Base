Expired Certificate in Azure App for Microsoft 365 Tenant in Netwrix Auditor

Question

A certificate for a Microsoft 365 (Office 365) tenant monitored by Netwrix Auditor is expiring shortly. Should a certificate be reissued manually?

Answer

No, a certificate will be reissued automatically by Netwrix Auditor as long as the application secret is valid. Once the certificate is expired, Netwrix Auditor will use the valid application secret to reissue it. If your application secret has expired, a certificate won't be issued.

NOTE: If you'd like to force the certificate reissue, you can delete the certificate from the corresponding Entra ID (Azure AD) app.
