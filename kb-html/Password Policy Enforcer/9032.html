Enable Extended (Debug) Logs

Question

How to enable extended (debug) logs in a Netwrix Password Policy Enforcer client or server installation?

Answer

  1. Create a .reg file and paste the following lines:

    Windows Registry Editor Version 5.00
            
[HKEY_LOCAL_MACHINE\SOFTWARE\Policies\ANIXIS\Password Policy Enforcer]
"DebugModeEnable"=dword:00000001

    The .reg file will later be used to create a DWORD DebugModeEnable entry with the value of 1 in the Password Policy Enforcer subkey.

  2. Open the resulting .reg file to add the new entry to the Password Policy Enforcer subkey. Confirm the addition by clicking Yes.

  3. Reboot the server for changes to apply.

  4. The ppclt_debug.txt log file will be created in C:\.

IMPORTANT: Once the troubleshooting phase is over, make sure to delete the DebugModeEnable entry in HKEY_LOCAL_MACHINE\SOFTWARE\Policies\ANIXIS\Password Policy Enforcer. Otherwise, the extended log may take up a significant space in your hard drive.
