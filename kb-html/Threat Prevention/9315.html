Troubleshooting Netwrix Threat Prevention (NTP) / StealthINTERCEPT Server (SI) Agent LSASS Blocked.

Symptom:

No Events Received due to Agent unable to hook into LSASS.

 

Example output is found in the log: LoaderTrace2024-10-01 17_24_07.log 

2024-10-01 17:24:07.269| Debug   | 0x09ac| executing NVInject, result: 6

Cause:

3rd party Security tools not allowing the Agent to Hook into LSASS.

Resolution:

Correct the needed End point protection (EPP) exclusion.

EPP / Antivirus Software Considerations


Alternatively this could have also been Sl Agent Safe Mode if enabled could prevent the LSASS hook to mitigate host rebooting.

To check for this setting take a look a the 'AD Agent' column.


 

This can be enabled or disabled with the Agent Update Settings > Set Options Window.

Netwrix StealthlNTERCEPT v7.3 > Administration > Policy Center > Agents Interface > Agents Interface Right-Click Menu > Update Agent Settings
