Duplicate Threat Prevention Windows agent entries in the Threat Prevention Admin Console

Summary: Basic steps to remove duplicate Threat Prevention Windows agent  entries

 

Issue: Remove duplicate Threat Prevention Windows agent entries in Threat Prevention Admin Console

 

Instructions:

Threat Prevention Admin Console duplicate Threat Prevention Windows agent entries steps -

  • Open the Threat Prevention Admin console
  • Verify duplicate Threat Prevention Windows agent entries in the Threat Prevention Admin console
    • As needed, use the data grid in the agents view to filter the duplicate entries
    • As needed, take reference screen shot of the duplicate entries
  • Try to delete duplicate entry using Remove Server from List right-click menu option. If this fails then perform the following steps:
    • Exit the Threat Prevention Admin console
    • Exit any additional Threat Prevention Admin console sessions
    • Temporarily stop the Threat Prevention Scheduler service
      • Stopping the Threat Prevention Scheduler service will also stop the Threat Prevention Reporting service
    • Temporarily stop the Threat Prevention Enterprise Manager service
    • Query the server table in the nvmonitorconfig database to list all entries
    • Referencing duplicate server entries, usually the Name and LastMgrContact columns, remove duplicate entries in the server table in the nvmonitorconfig database
    • Query the server table in the nvmontiorconfig database, and verify duplicate entries are deleted
    • Start the Threat Prevention Enterprise Manager service
    • Check log entries in the Threat Prevention Enterprise Manager service logs for any errors
      • If there are any errors captured, compress and send the Threat Prevention Enterprise Manager log folder
    • Open the Threat Prevention Admin console, and check for duplicate Threat Prevention Windows agent entries 
    • Start the Threat Prevention Reporting service
      • Starting the Threat Prevention Reporting service will also start the Threat Prevention Scheduler service

Note:  Threat Prevention services need to be stopped to avoid any caching of Threat Prevention Windows agent entries

 

Affected Versions: Threat Prevention 4.0, Threat Prevention 4.1, Threat Prevention 5.0, Threat Prevention 5.1, Threat Prevention 5.2, Threat Prevention 6.0, Threat Prevention 6.1, Threat Prevention 7.0
