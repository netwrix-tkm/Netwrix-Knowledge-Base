Hotfix: SIWindowsAgent_7.0.1.776_7.1.0.521_7.2.0.316_7.3.0.225_7.3.5.148

Summary:
  1. 85414: SUVP 22-02 support 
  2. 69558: 00329310 - No AD Activity from hosts 
  3. 58540: 00327075 - SI agent causes GPO file corruption while configuring folder redirection policy 
  4. 84991: MS 'Preview' updates, released January 25, 2022 

Behavior before Fix:
  1. 2012 pre-R2 Server:
Kerberos events may be lost or incomplete after applying KB5010392
  1. SI might not notify SAM when pending mode detected
  2. GPO files handling was implemented as reading & parsing; GPO files copying was implemented via standard C# API
  3. Applying KB5009608 for 2022 Server:
  • Events will be lost or incomplete for the following:
    • AD Changes and Lockdown SI policies.
    • LDAP Monitoring and Lockdown SI policies.
  • SI AD and LDAP Lockdown policies may not work correctly.

Behavior after Fix:
  1. Signatures updated; SI Agents must be fully functional with KB5010392
  2. SI should notify SAM when pending mode detected
  3. GPO files handling now done as reading into memory (in shared mode) and then parsing from memory; GPO files copying now done as reading into memory (again in shared mode) and then writing new file using in-memory stream. This way time of interaction with GPO file is minimized thus minimizing potential locking issues.
  4. Signatures updated; SI Agents must be fully functional with KB5009608
