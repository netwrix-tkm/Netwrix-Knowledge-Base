Installing SAM via Command Line (to a different directory)

Summary: Procedure to install SAM via Command line to a different directory.

Issue: Switches to install SAM via command line to different directory.
 

Instructions:

  1. Load up Command Prompt as administrator.  
  2. Change Directory to install media location (must be a local directory). Ie CD C:\InstallMedia
  3. Run the following command msiexec.exe /a "C:\InstallMedia\ SBActivityMonitorSetup" INSTALLDIR="E:\" /qn (Replace C:\Installmedia\XXX with the location and Name of the SAM installer)
  4. Wait for the installer to run (there will no output however task manager will show "Windows installer" running with CPU usage.
  5. Navigate to the install folder to verify that it has installed
 
NB the switches listed above,
/a = Administrator  
/q Sets the UI Level
/n No UI (used in conjunction with /q)
