Hotfix: SIWebReports_4.0.0.653_SI-3220

Summary: SI Web Reports allows you to delete a Public Report

Issue: Summary:  13196 - SI Web Reports allows you to delete a Public Report
Before fix: Public Reports can be deleted by the Non-Owner
After fix: Public Reports cannot be deleted by the Non-Owner
 

Instructions: 1. Stop services:
                - SIReportingService
                - SISchedulerService
2. Extract the attached file to
C:\Program Files (x86)\STEALTHbits\StealthINTERCEPT\SIReports\ReportingServices
3. Start services:
                - SIReportingService
                - SISchedulerService
