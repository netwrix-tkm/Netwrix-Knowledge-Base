Hotfix: SA_11.0_005

Summary:

SMPC-10903 \ Escalations 56790 / 56842: CSV downloaded as .zip File

Issue/Request: The csv data from a report that was configured with 'Export table data as CSV' was being saved as a zip file containing a zip file.

Fix/Enhancement: Issue was corrected - the zip file (for large data sets) contains a csv file that opens up correctly. Works from the generated report in Access Analyzer, and from the Access Analyzer Web Console.
