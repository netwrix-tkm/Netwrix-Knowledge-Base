Hotfix Patch 2.6.211 Threat Manager 2.6

Description:

This hotfix addresses or implements the following:

  • Threat Manager will can infinitely queue events, leading to a variety of disk and memory issues due to backpressure in the event message service. This build introduces additional safeguards to prevent failure of the server under extreme backpressure.
  • Resolved issue with previous (removed) release 2.6.209, which could cause loss of TGT events

 

Submitted by:

Lee Berg
