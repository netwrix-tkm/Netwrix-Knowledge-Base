How To: Add the Organizational Unit field to the My Groups page

Applies To:

Directory Manager 11  Directory Manager Portal

More Information:

The Directory Manager Portal enables us to customize the portal based on our requirements. We can add a wide range of functionalities to the portal using Directory Manager Portal.

In this article, we will be looking into the steps to add the Organizational Unit field on the My Groups page in the portal. The My Groups page displays the groups that the logged-in user owns. It also displays search results when a user performs a group search in the portal.

By default, the My Groups page is as:

 

 

Steps:

Follow these steps to add the Organizational Unit column on the My Groups page.

1. In Directory Manager Admin Console, select Application> Directory Manager Portals > [required portal]

2. Click on the three dots and go to Settings>Design Settings>Select the identity store> Search Forms Tab>Search Results select Groups in the Name list and click Edit.

 

 

3. On the Edit Search Results, click Add Field.

 

 

4. On the Add Field dialog box, enter the following information:

    • From the Field list, select DistinguishedName.
    • In the Display Name box, enter Organizational Unit or OU; this text will appear as the field name in the portal.
    • In the Tooltip box, keep the default text, i.e. WEB_Type_changes.
    • From the Display Type list, select Text.

5. Click OK.

 

 

6. On the Edit Search Results, you can drag the fields to change the order of the fields and click OK.

7. Save the changes in the design settings.
