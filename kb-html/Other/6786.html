Missing GatesProxyIDs for resources picked up in FSAC scans

Summary: A bug was found that would result for resources that were picked in a FSAC scan to have a null GatesProxyID, even though its parent had one

Issue: Summary:
A bug was found that would result for resources that were picked in a FSAC scan to have a null GatesProxyID, even though its parent had one
 

Instructions: This bug has been addressed in FSAA 8.1.998. However, if a customer has already ran into this issue, a script has be created to remedy the existing data.

https://downloads.stealthbits.com/access/files/StealthTOOLS/AUDIT/FixMissingGatesProxyIDs.zip

Once the script has completed successfully, delete the StrucMap in tier 2 for the affected hosts. StrucMap will rebuild with the correct data in tier 1 when the next bulk import is ran.
 
Dependencies
THIS FIX REQUIRES AT LEAST THE FOLLOWING VERSIONS OF FSAA:
For SA 8.1, FSAA version 8.1.998 or later
For SA 8.2, FSAA version 8.2.1051 or later
