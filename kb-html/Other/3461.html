Custom Branding in a two server install configuration

When adding a custom logo on the Branding tab in the Password Manager admin portal, the self-service portal does not show this logo.
When the Self-Service portal and the Admin portal are on two different web servers, the image does not exist in the self-service portals web folder.
On the computer which hosts the Self-Service portal navigate to %install drive%Program Files(x86)Netwrix Password ManagerWeb_SSImages and copy the logo into this folder to replace the existing logo.gif. After refreshing the Self-Service portal, the new logo should show up.
