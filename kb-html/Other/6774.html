Issue: Title:  SIEntManager_4.1.0.603_SI-3361_SI-3363
Summary:  
SI-3361 - Windows 32bit console will not open.
SI-3363 - Enterprise Manager crashed while processing events
Before fix:
SI-3361
Big amount of concurrent registering agents with delays in db connection can lead to spawning big amount of threads in thread pool, as every registration performs a lot of db operations from ServerCollection classAWS security group to allow access.
SI-3363
Possible crash due to unhandled null reference exception inside timer callback due to incomplete agent registration with null client id inside active agents collection.
After fix:
SI-3361
Refactor ServerCollection class to use on thread for all db operations related to updating changes in ServerConfigs, thus registration is not slowed down by Server table updates and reads.
SI-3363
Put only initialized agent registrations inside active agents collection. Add exception handling to the timer callback.
