Instructions:
When a StealthAUDIT job is executied, during runtime only, a table named #sa_job_hosts is created, and populated with a list of all of the hosts selected from the host lists node.  

The table definition is:
CREATE TABLE [dbo].[#sa_job_hosts](
    [HostName] [VARCHAR](256) NOT NULL
) ON [PRIMARY]

For testing a SQL script prior to executing in SA, the easiest way is probably to add the following code to the top of the script:  
Obviously, you would could replace the host names, and all you need is the first two lines to add a single host.  
Since temp tables are per session, this shouldn't interfere with SA at all.  

SELECT 'sbnjeng30' AS [HostName] 
INTO #sa_job_hosts
UNION ALL SELECT 'sbslppsa01'
UNION ALL SELECT 'sbslamsa01'
UNION ALL SELECT 'sbslkdsa01'
