Parse CSV / Custom Delimiters

Summary: How to parse a CSV file or similar using the PowerShell data collector, specifying the delimiter.

Issue: Sometimes we need to parse a file using something other than commas. Text search has limited capability here, and PowerShell can parse a file much quicker.

Instructions: How to parse a CSV file or similar using the PowerShell data collector, specifying the delimiter: 

This job will allow SA to Parse a data file with a Custom Delimiter using PowerShell DC
 
The path is hard coded in the example, and the delimiter in this example job is a pipe "|".  
Change it to a comma or whatever is appropriate for the file you'd like to parse.  
This should be much faster and simpler to use than the text search data collector.  

http://downloads.stealthbits.com/access/files/Utilities/Jobs/JOB_customDelimiterParse.zip

PowerShell code is: 

$resultObj = import-csv "c:\dattest\pipedelim.dat" -delimiter "|"
$resultObj
