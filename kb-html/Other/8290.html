Support Process - Vulnerability Tracker Subscription Key Export/Import

Team - We just had an instance where it was necessary to switch subscription keys on a GSM Appliance so if you find yourself needing to do this please follow the below (we may need to switch some evaluation users running on our trial key to a personal key when they purchase)

Export/Import Subscription Key

Export Subscription Key

Enable SSH via Setup  Services  SSH (you will be prompted to change the default Admin password if not already done so  save if changed)

You can then putty to the appliance  find the IP address by using Setup  Network  IP

After logging in, this time if you go to the Advanced  Subscription, you can see the key onscreen but copy it out just by highlighting the key.

You can then paste the key into a text file and save it as key.txt

Import Subscription Key

To upload a key you can then go to Setup  Feed  Key (HTTP) which gives you a temporary upload URL to use for the key.txt file, or use the Key (editor) option to paste in the copied key directly.
