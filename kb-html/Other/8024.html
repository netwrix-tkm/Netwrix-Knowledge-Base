Summary: DFS long name / short name filtering

Issue: Summary: DFS long name / short name filtering
Issue/Request: The FSDFS System Scans job in the File System solution set can return both short and long hostnames causing duplicate scans. This can cause the activity and permissions for a single host to be split among two (one short name one long name) in the AIC, as well as other reports and so on.
Fix/Enhancement: The job's analysis SQL script was updated to convert all the target servers to short names.
