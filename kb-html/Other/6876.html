Hotfix: SIEntManager_SIWindowsAgent_4.1.0.706_5.0.0.739

Summary: Threat Prevention collects attributes values for ms-mcs-admpwd

Issue: Summary:  Threat Prevention collects attributes values for ms-mcs-admpwd
Before fix: Threat Prevention collects attributes values for attribute name ms-mcs-admpwd which are passwords in clear text.
After fix:
1. Changeset introduces 'permanent' rules to filter.json. Such a rules doesn't depend on any UI configuration, they will be applied immediately after EM startup & agents registering.
2. Changeset also extended filters.json syntax by introducing 'HideAttributeValues' action. It allows to hide values of the attribute matching condition. Values will be presented as ''/''
