Hotfix: SA_7.1_SP1_016

Summary: Enormous transaction logs: When attempting to remove old activity during FSAC bulk import, FSAA would remove the activity in one big transaction which would cause the transaction log to dramatically expand in size. Changes have been made so that this transaction is now chunked into smaller transactions, to limit the impact on the transaction log.
