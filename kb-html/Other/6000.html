Hotfix: SIWindowsAgent_7.0.1.766_7.1.0.513_7.2.0.308_7.3.0.217_7.3.5.128

Summary:

SBTSERVICE consuming memory

Behavior before Fix:

SBTservice using high memory usage after GPO Changes

Behavior after Fix:

Changes were made to sbtservice.exe used for file activity monitoring.  A combination of a high volume of GPO file activity and code in sbtservice.exe used to detect potential file copy operations resulted in certain internal buffers not being discarded based on expiration time.  As a result these buffers grew without limit.
