Hotfix: AIC_9.0_001

Summary: SMPC-7497: 39214 - AIC Errors During Ad Hoc Group Member Change by Group Owner . Issue/Current behavior: Changing membership to an owned group by adding a user whose name has a space and parenthesis (e.g. Domain\User (user)) was failing to commit. . Fix/Enhancement: Now users as the described above can be added to an owned group by using the Change Access button.

Issue: Summary:
SMPC-7497: 39214 - AIC Errors During Ad Hoc Group Member Change by Group Owner

  • Fix/Enhancement: Now users as the described above can be added to an owned group by using the Change Access button.
