Hotfix: SFAM_2.4_006

Summary: If monitored events are configured on Netapp running in 7-MODE, they will be reset by the SFAM agent to 'all' once monitoring is enabled. This update allows the administrator to specify which events are to be monitored.

Issue: Modules Included

  • SFAM-255: SI File Activity
  • SFAM-256: inconsistent information in the AIC for activity stats and activity details
  • SFAM-325: EMC / Isilon Activity Disconnect

Instructions:

  1. Run msi file on host that has the SBFileMonitor installed. (No need to uninstall previous version, msi will upgrade the monitor). Upgrade all agents.
  2. Update all agent's INI files. In each 7-mode netapp section. Add the following parameters set to ON or OFF depending if you need the specific operation enabled or disabled in fpolicy:
RECORD_READS=ON/OFF (Already exists)
 
RECORD_CREATES=ON/OFF
RECORD_DIR_CREATES=ON/OFF
RECORD_DELETES=ON/OFF
RECORD_DIR_DELETES=ON/OFF
RECORD_RENAMES=ON/OFF
RECORD_DIR_RENAMES=ON/OFF
RECORD_UPDATES=ON/OFF
RECORD_PERM_CHANGES=ON/OFF
RECORD_DIR_PERM_CHANGES=ON/OFF
