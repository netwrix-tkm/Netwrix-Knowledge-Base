Hotfix: SA_11.0_056

Summary:

Exchange Metrics collection Invalid file error on 1 server

Submitted by:

Justin Giemza

Affected Versions:

11.0

Affected Module:

SA  DC  ExchangeMetrics

 

Dev Ticket:

SAT-9607

Escalation #:

61410

Resolved in Version:

DC\DCExchangeMetrics.dll v. 11.0.173.0

DC\DCExchangeMetricsSrv.dll v. 11.0.173.0

 

KB Type:

               Hotfix

 

Instructions:

  1. Unblock the hotfix zip file in the windows property dialog if an unblock button exists there.
  2. Close all instances of StealthAUDIT (check task manager under processes for all users)
  3. Exit StealthAUDIT console
  4. Copy and replace files from the folders in this hotfix to the corresponding folders in SA Install dir.
  5. If running the Exchange Metrics Applet remotely it maybe neccesary to rerun the applet deployment or manually update the file DCExchangeMetricsSrv.dll on the exchange server.

It should be located:

%Program Files%\Microsoft\Exchange Server\V15\TransportRoles\Logs\MessageTracking\SA_ExchangeMetricsData.

 

(V15 in the file path is dependent on Exchange version)
