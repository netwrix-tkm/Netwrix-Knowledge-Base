Error 401 in the Enrollment wizard

I get the Error 401 (Not authorized) message when starting Enrollment wizard
To resolve the issue, please add the Password Manager Self-service portal URL to the Local Intranet Zone.
  1. Launch Internet Options from Control Panel
  2. Go to Security tab
  3. Select Local intranet, click Sites
  4. Click Advanced
  5. Type the URL and click Add
  6. Click Close, then Ok, and again Ok
As a workaround, there is an option to force the browser to send credential to the Internet zone
  1. Launch Internet Options from Control Panel
  2. Go to Security tab
  3. Select Internet, click Custom level
  4. In Security Settings window that opens sroll to the bottom
  5. Select Automatic logon with current username and password radio-button under User authentication - Logon
  6. Click Ok than Ok again
These settings can also be applied via Group Policy.
