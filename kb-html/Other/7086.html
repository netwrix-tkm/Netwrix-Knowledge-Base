Issue: Summary:
Exchange | add support for MS .Net updates, Released 02/25/2020
Behavior before Fix:
Exchange policies stop working after applying the following MS updates for .Net:

  • 2012 R2 Server - KB4538158 for .Net 4.7.2 - 4.8
  • 2016 Server - KB4537477 for .Net 4.8
  • 2019 Server - KB4538156 for .Net 4.7.2 and 4.8
Affected:
  • SI: 5.x - 7.0.x
  • Exchange: 2013 - 2019
SI behavior after applying MS updates:
  • SI Exchange policies stop working after applying .Net updates, released 02/25/2020
  • No additional performance or stability impact
Behavior after Fix:
Extend signature file. SI Exchange policies work as expected.
