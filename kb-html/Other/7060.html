Issue: Summary: SI Console| EPE report does not work
Behavior before Fix:
This issue appeared as a result of fixing SI-4042 where we had to change datatype for two field in DB views from nvarchar(max) to ntext. And this caused errors in some operations with these fields in three EPE queries
Behavior after Fix:
 To fix this problem we changed EPE queries to use tables instead of views.
