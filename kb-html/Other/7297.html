Dynamic Permission Values in File System Action

Summary: What values does it accept?

 

Instructions: The action module, in 8.1, will accept the following values.  
(Currently Case Sensitive)

Folder 
"Full Control", "Modify", "Read & Execute", "List Folder Contents / Read Data", "Read", "Write" 

Share 
"Full Control", "Change", "Read"


you can map these to the AllowMaskDescription values from an FSAA permissions view

except you'll need to replace list folder contents and special permissions with "List Folder Contents / Read Data", or whatever the next best option is.
