Hotfix Patch 2.6.207 Threat Manager 2.6

Description:

This hotfix addresses or implements the following:

  • Issues with the File Table taking up an abnormal amount of Storage space
  • Issues with Investigation Threat false positives when using Attribute Before or Attribute After

  • Various AD Sync Fixes and Improvements
  • Improved Python Job Logging
  • Issues with Threat History not showing the correct user in some circumstances
  • Isuses with User Access to Threat Manager
