Instructions to apply hotfix:

  1. Unblock the hotfix zip file in the Windows property dialog, if an unblock button exists there
  2. Close all instances of Access Analyzer (check task manager under processes for all users)
  3. Exit Access Analyzer console
  4. Stop the 'Access Analyzer Web Server' Service.
  5. Open Command Prompt as administrator.
  6. Change directory to the %SAInstallDir%
  7. Run UnregisterAll.bat.
  8. Replace the provided files in the appropriate directories.
  9. Run RegisterAll.bat
  10. Press enter to continue and close the cmd
  11. Restart the 'Access Analyzer Web Server' Service.
  12. Start Access Analyzer.
