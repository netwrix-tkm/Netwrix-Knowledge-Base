Is Data Still Collected when the License is Expired for Access Analyzer / Netwrix Activity Monitor?

Summary: What is the product behavior when the license is expired for Netwrix Access Analyzer/Stealthbits Activity Monitor (SAM)? Is data sill collected when the license is expired for Access Analyzer/SAM?

Issue: Is data sill collected when the license is expired for Access Analyzer/SAM?

Instructions: Access Analyzer 8.1+: When the license is expired, the scheduled jobs no longer run.  The interface will also not open, simply giving an error that the license is expired and a new one should be obtained.   Thus, even when a new license is applied, there will be no data from the time that the license was expired (but see below for SAM interaction).
 
Netwrix Activity Monitor 3.0+:  File activity continues in SAM when the license is expired.  SAM will give a message that the license is expired, but querying for activity data during the expired timeframe will show that the program is still collecting activity.  If Access Analyzer license is updated, data collected by SAM should be imported as normal once Access Analyzer job to do so runs.
