Can Change Tracker Run Agentless?

Yes, if we are pressed about the issue, we would prefer to run with an agent for various reasons- Real Time tracking, resource usage, who made the change. However, Change Tracker can run agentless, and has no issues doing so. If the prospect is agent-adverse, we are covered here.
