SbPAM: Applying a new license

Summary: How to add a new license to the SbPAM Console

Issue: If the SbPAM license is expired and no longer valid than this KB will assist an Administrator of the console in adding a new license.

Instructions:

Stealthbits Privileged Access Manager comes with a temporary 30-day license. Follow the steps to import a new license key file.

  • Login to the SbPAM console as the built-in Admin.
    • Username: admin
    • Domain: <none>
  • Click the signed in Users name in the console header and select About SbPAM from the drop-down menu. The About Page will show the version number and current license information.
  • In the Upload License File Section, click Import License.
  • Navigate to the license file and click Open to import the license.

 

The License file name is displayed at the bottom left and the organizations license information is updated.

Additionally, if a new license file is required than the organizations account manager can assist in providing a new one.
