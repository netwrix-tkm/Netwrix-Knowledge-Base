Hotfix Patch 2.7.1175 for Threat Manager 2.7 SP1 (HF 8)

Summary:

In this hotfix we fixed various issues relating to StealthDefend 2.7 SP1:

  • Fixed an issue where Threat Manager created Forged Ticket threats for groups that Threat Manager shows as on the user object.
  • Fixed an issue where access was being deneied when using the powershell action step that required elevated privileges.
  • Fixed an issue where activity statistics showed 0 events on the Threat Manager dashboard
  • Fixed an issue where StealthDefend was not sending events to SIEM
  • Created a tool for changing the StealthDefend PostgreSQL password

Submitted by:

Greg Connors

Affected Versions:

Prior versions of 2.7 SP1
