Hotfix Patch 2.2.216 for Threat Manager 2.2

Summary: This Hotfix addresses a variety of issues with UBA Threats failing to send email notifications and Investigations with a Tag in the query rule builder.

Issue: Summary:
This Hotfix addresses a variety of issues with UBA Threats failing to send email notifications and Investigations with a Tag in the query rule builder.
Issues Addressed

  • UBA Threats could fail to send alert emails in some instances due to invalid SIEM settings.
  • Custom Threats based on a Tag would fail to evaluate Threats.
Hotfix Application Note
Custom Threats based on a Tag criteria specified in the investigation rule builder will likely need to be Re-Saved in the console. This will resolve the tag investigation issue.
