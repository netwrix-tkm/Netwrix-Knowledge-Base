Updates to Cmdlets

SkipCertificateCheck

All cmdlets that make HTTPS calls now accept the SkipCertificateCheck parameter.

Start-SbPAMActivitySession

The Start-SbPAMActivitySession cmdlet has the following changes:

 The ManagedAccountId parameter has been removed, the value is derived from the token.

ActivityId and ActivityName are now mutually exclusive

ManagedResourceId replaced with ResourceName parameter

AccessPolicyId , StartTime and EndTime parameters are now optional

Get-SbPAMUserPolicy

The Get-SbPAMUserPolicy cmdlet has the following changes:

 JwtObj parameter has been removed, the value is derived from the token.
