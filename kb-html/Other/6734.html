Issue: Summary:  StealthINTERCEPT fails to send events to LogRythym SIEM
Before fix: Enterprise Manager stops to send events to SIEM
After fix: Enterprise Manager continues to send events to SIEM and traces invalid event info in log file.
Note for customer:
If such problem will happen then new Error-level entries in the EnterpriseManager log file will appear. The first entry which indicates the problem will start with "processSyslogMessage() Attributes can't be parsed:" text and then there will be more entries which show details about the event.
