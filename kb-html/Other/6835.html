Summary: password enforcer fails to block setting password

Issue: Summary: password enforcer fails to block setting password
Behavior before Fix:
Unicode pwd values were converted to ASCII format on saving or restoring pwds list in WinConsole. Due to this agent did not block some values from the pwd list.
Behavior after Fix:
Unicode pwd values should be blocked by the agent after saving or restoring or loading from file a pwd list in WinConsole.
