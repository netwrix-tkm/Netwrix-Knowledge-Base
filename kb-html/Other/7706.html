Hotfix: SA_8.2_068

Summary: if the report is generated with footer section enabled, The Emailed report PDF also has footer included. Before Fix: when the custom report that includes a footer, the generated Report that sent on email did not include Footer section. After Fix : when the custom report includes a footer , the emailed PDF should also have Footer section.
