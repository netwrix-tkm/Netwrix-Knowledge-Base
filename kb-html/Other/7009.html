THIS HOTFIX REQUIRES AT LEAST FSAA HOTFIX SA_8.1_006 TO BE INSTALLED IF USING FSAA.

Instructions:

  1. Unblock the hotfix zip file in the windows property dialog, if an unblock button exists there. 
  2. Close all instances of StealthAUDIT (check task manager under processes for all users)
  3. Uninstall previous version of SDD Add-on from local SA Console (if applicable).
  4. Install SensitiveDataAddon.msi on your local SA Console from x86 folder.
  5. Update all SensitiveData files in your %SAInstallDir%SensitiveData folder
  6. Launch StealthAUDIT. Run the 'Update Proxy Service' query under the Maintenance tab in the FSAA data collector against your proxy servers (if running proxy service).
  7. Uninstall previous version of SDD Add-on from target SharePoint farm (if applicable)
  8. Install SensitiveDataAddon.msi on the target SharePoint application server from folder x64 (if applicable)
