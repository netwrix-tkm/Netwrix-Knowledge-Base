THIS HOTFIX HAS A PREREQUISITE OF HOTFIX SA_9.0_018. PLEASE APPLY THAT HOTFIX PRIOR TO APPLYING THIS ONE. SEE KB FOR THAT HOTFIX FOR MORE DETAILS
Boldon James tags can now be applied thru the FS Action Module UI. Select the 'Boldon James' radio option on the Parameters page if you want to apply/remove a Boldon James tag.
NOTE: If choosing the option to overwrite tags, the action module will clear out both normal tags and Boldon James tags and then proceed to apply whatever tags you have configured to overwrite.
NOTE: If choosing the option to remove all tags, the action module will clear out both normal tags and Boldon James tags.
Instructions:
  1. Unblock the hotfix zip file in the windows property dialog, if an unblock button exists there. 
  2. Close all instances of StealthAUDIT (check task manager under processes for all users)
  3. Update all Action files in your %SAInstallDir%Actions folder
  4. Update all PrivateAssemblies in your %SAInstallDir%PrivateAssemblies folder.
