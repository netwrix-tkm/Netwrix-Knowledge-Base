Hotfix: SAM_5.0_004

Summary:

  1. Search takes 8 minutes to complete
  2. SAM Permissions "Failed Add" events when browsing folder
  3. SAM Search displays ALT_HOST Values but doesn't allow searching against that value
  4. Request to Confirm Support for EMC CEE 8.7.7

Behavior Before Fix:

  1. SAM Search was slow and took a while to complete.
  2. False Positve events were being reported (caused by Symantec EP).
  3. Search didn't allow searching against ALT_HOST values.
  4. SAM didn't support latest CEE 8.7.7 with VCAPS.

Behavior After Fix:

  1. Added improvements to search that should result in faster search.
  2. Added improvements that should not report false positive events.
  3. Updated search to allow searching against ALT_HOST values.
  4. Updated support for latest CEE 8.7.7 (which also includes VCAPS).
