Hotfix: SA_11.0_037

Summary:

- SMPC-11918 59530- Help needed with custom papersize for generated PDFs

Description:
        -  Issue/Request: There was no control over PDF page width.

         -  Fix/Enhancement: "LIMITPDFPAGEWIDTH" xml attribute in GlobalOptions.xml will be created on SAConsole's initialization. This will be set to true for A4 page width and false for no limit on page width.
