How to track if the "Stop Strongpoint Scripts" has been used and by who.

In order to view if the scripts were stopped through our menu at any point in time you can do the following:

  1. In NetSuite, go to Reports > New Search
  2. Pick Users & License Manager from the list
  3. In the Criteria tab add the following filter:
    • Select System Notes Fields... then choose Field and select StopScripts. It will look like this:
  4. Now go to the Results tab and add the following columns, you can select System Notes Fields here and choose Add Multiple to simplify the task:
    • System Notes Field
    • System Notes Date
    • System Notes Old Value
    • System Notes New Value
    • System Notes Set By
    •  
  5. Click Submit

If the search returns empty that means no one ever stopped the scripts by using our menu feature. Instead if there are any results they will display as in the screenshot below showing values for the stop scripts flag changing from F to T (false to true) when they where stopped and T to F (true to false) when they were resumed in case they were after.
