Hotfix: SIEntManager_6.1.0.206_SI-3930

Summary: Update SIEM template options

Issue: Summary: Update SIEM template options
Behavior before Fix:
OBJECT_RETURN, QUERY_FILTER, and BASE_DN_CLASS are not available in the Splunk events template.
Behavior after Fix:
For BASE_DN_CLASS may be used existing %CLASS_NAME% replacement token.
There are two new replacement token for SIEM templates available now:
  %LDAP_QUERY% - Query of LDAP Search request
  %LDAP_OBJECTSRETURNED% - Number of objects returned by LDAP Search
