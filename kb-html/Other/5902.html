How to Log Activity of PPE Mailer

This article will be useful if you need to have a reporting on the activity of the PPE Mailer e.g. you want to know whom has PPE Mailer sent email to.
In order to do this follow these steps:
  1. Stop the "PPE Mailer" service and set the startup for the service to disabled
  2. in Windows Task Scheduler, create a task to be run once a day with these settings: "ppemail.exe /send >> c:\logs\ppemail.txt" It will create a log file in c:\logs\ folder and will write the activity there once a day.
NOTE: The task can be set to be ran more often than once a day, however it will generate a lot of activity and cause unnecessary load on the server.
