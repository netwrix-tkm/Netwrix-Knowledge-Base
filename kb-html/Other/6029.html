Hotfix Patch 2.7.1101 for Threat Manager 2.7 SP1

Summary:

In this hotfix we fixed various issues relating to StealthDefend 2.7 SP1:

  • Fixed a ransomware job issue that caused PostgreSQL database issues
  • Fixed an issue where you could not upload custom threats to StealthDefend
  • Fixed an issue where Non-Trial licenses were displaying as Trial
