Instructions: Steps to apply the patch:
 
1. Open SI Console
2. Select 'Templates' node
3. Remove the following templates:
 - Templates\Microsoft\Active Directory\Authentication\Administrative Accounts\"AD: Failed Administrator Account Authentications"
 - Templates\Microsoft\Active Directory\Authentication\"AD: Failed Account Authentications"
 - Templates\Microsoft\Active Directory\Authentication\Service Accounts\"AD: Failed Service Account Authentications"
4. Extract attached "FixedTemplates.xml" to temp folder
5. Select Tools\Import in SI Console
6. Select extracted "FIxedTemplates.xml" from temp folder for Import
7. Run Import
 - !!! Select 'Apply to All' and then 'Skip' for existed Collections during import
8. Create new SI Policies using mentioned above SI Templates
