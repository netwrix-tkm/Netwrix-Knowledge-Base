Hotfix: SD_1.2_007

Summary: Hotfix to resolve issues with permissions given to members of nested child AD Groups, Unicode issues in Jobs, and 1.X to 1.2 License upgrade issues.

Issue: Summary:
Hotfix to resolve issues with permissions given to members of nested child AD Groups, Unicode issues in Jobs, and 1.X to 1.2 License upgrade issues.
 
Behavior before Fix:

  • 1.2 Installs would often not property recognize 1.1 or older DEFEND license files.
  • If and AD Group was scoped access in DEFEND console access, members of child groups in the parent groups would still not get access.
  • Some Unicode characters in process names, hostnames, and other fields could still potentially cause some of the various jobs to fail.
 
Behavior after Fix:
  • 1.2 Installs will take additional steps to utilize older license files.
  • If and AD Group is scoped access in DEFEND console access, members of child groups in the parent groups will receive the access via a new recursive permission check.
  • Updated Unicode handling in all python jobs add an additional layer of protection for Unicode issues.
