How To: Hide distribution list from the portal.

Applies To:

Directory Manager 10 and above.

Business Scenario:

We want to hide the distribution list from end-users on the Directory Manager portal.

Solution:

We can define LDAP criteria in the user security role to hide the distribution list for the end-users in the self-service portal. 

1) Please go to the properties of your identity store in the Directory Manager management console. 

2) Now, click on the security roles tab and edit the user security role. 

3) Click on the Polices tab and then go to Search. 

4) In the LDAP filter, apply the filter as below: 

sAMAccountType    is Not   268435457
sAMAccountType    is Not   536870913

Use the AND operator between the above two criteria. 

 

 

5) Save the settings in the security roles. After that, sign out from the Directory Manager portal and Sign in again with an account that is part of the user security role, and you will see the change. 


Note:

For Universal and Global distribution groups, the samAccountType value is 268435457
For Domain local distribution groups the samAccountType value is 536870913
