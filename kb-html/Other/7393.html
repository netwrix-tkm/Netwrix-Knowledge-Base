Reports Only Mode

Summary: How to use StealthAUDIT in reports only mode.

Issue: If you need to clear an entire report index, due to problems with duplicate reports or otherwise corrupted report indexes,  you can use this mode to generate reports easily without executing data collection, analysis, or actions.  

Instructions: 1) Navigate to the StealthAUDIT installation directory in a command prompt. 

the easiest way to do this is to
a) navigate to its parent folder (STEALTHbits folder)
b) shift-right click the StealthAUDIT folder and choose "Open command window here"

2) Run the command

stealthaudit.exe /reportsonly

3) Run any jobs and it will generate all of the reports in the jobs, but will skip all data collection, analysis, actions and ONLY generate reports.  


TIP:  If you have multiple instances of StealthAUDIT open, you'll be able to see the reports only mode in the title bar.  

WARNING:  Make sure you close the application after you're finished using this mode.  Otherwise, you may find yourself wondering why your data collections aren't running.
