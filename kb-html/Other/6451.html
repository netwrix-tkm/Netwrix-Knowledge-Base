Hotfix: SIWindowsAgent_7.0.1.658_SI-4116

Summary:

High File Monitor CPU utilization

Behavior before Fix:

Rule Engine Path evaluation is slow in case of big number of monitored GPO

Behavior after Fix:

There is new configuration parameter in the SIWindowsAgent.exe.config file: gpoFoldersSearchLimit

Default value is "30"

If we will see more than 30 folders there then instead if listing individual folders we will put higher level folder.
