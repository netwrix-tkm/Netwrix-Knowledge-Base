Issue:
SR-5661 - Improve Default PostgreSQL Config Calculations

  • Previously, on systems with lots of resources, the calculation for setting PgSQL defaults would exceed maximum values for certain settings, causing PgSQL to default to a basic setting.  The installer now correctly limits certain values to maximum thresholds.
SR-5685 - 41110 - StealthRECOVER: 1.5 fails to install with C++ 2017 is installed
  • The Microsoft Visual C++ Redistributables share the same product ID, so previously we would detect VC++2015 as being installed when only VC++2017 was present.  The checks have been expanded and this scenario is now handled correctly and the product installs as expected.
