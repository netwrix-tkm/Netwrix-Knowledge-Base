FORWARD EVENTS FROM SI to Threat Manager

Summary: When only SI agents are Deployed and Threat Manager is in Play

Issue: SAM agents are not deployed only SI agents and File System events need to be sent to Threat Manager
 

Instructions:

  1. Configure SIEM settings in SI and point Threat Manager via IP @ port TCP 10000
 
  1. You will now need to create and load custom SIEM template for Threat Manager formatting should be like below
 
%SYSLOG_DATE%|Threat Manager|%HOST%|%PRODUCT%|%PRODUCT_VERSION%|cat=%EVENT_NAME% setting=%SETTING_NAME% iotype=Filesystem domain=%EVENT_SOURCE_NAME% time=%TIME_STAMP_UTC% username=%USERNAME% process=%ORIGINATING_CLIENT% filename=%DN% sourcehost=%ORIGINATING_SERVER% success=%SUCCESS% blocked=%BLOCKED_EVENT% attrname=%ATTRIBUTE_NAME% attrnewvalue=%ATTRIBUTE_VALUE% attroldvalue=%OLD_ATTRIBUTE_VALUE%
