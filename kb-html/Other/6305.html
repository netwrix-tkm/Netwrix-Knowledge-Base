Hotfix Patch 2.6.209 for Threat Manager 2.6

Description:

This hotfix addresses or implements the following:

  • Threat Manager will currently infinitely queue events, leading to a variety of disk and memory issues due to backpressure in the event message service. This build introduces additional safeguards to prevent failure of the server under extreme backpressure.

 

Submitted by:

Lee Berg

 

Affected Versions:

Prior versions of 2.6
