Summary: This Hotfix addresses a variety of issues relating to the Investigation UI as well as some fixes for Installation and Reliability of the StealthDEFEND services.

Issue: Summary:
This Hotfix addresses a variety of issues relating to the Investigation UI as well as some fixes for Installation and Reliability of the StealthDEFEND services.
 
Behavior before Fix:

  • Issues when trying to use investigations with Perpetrator & Target Hosts
  • Finish button is not active on last page of Investigation Wizard
  • $PrimaryEventPath Variable for Action Steps only resolved for files, now supports Folders
  • Custom Threats created via the investigation did not rollup events properly.
Behavior after Fix:
  • Correct Query results when utilizing Perpetrator and Target Host in Investigation Queries
  • Finish button will show active on the Investigation Wizard
  • $PrimaryEventPath Variable for Action Steps support files and folder paths
  • Custom Threats created via the investigation will now "rollup" by user and host for a 24-hour period.
