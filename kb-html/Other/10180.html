How To: Generate a Report on 'Enabled Users without Managers along with DL memberships'

Applies To:

GroupID 11 - Reports

Business Scenario:

You need a report to view a list of all enabled users who are without a manager, along with their DL memberships in the domain.

Methodology:

The GroupID Reports module is a free reporting tool designed to run reports on Active Directory and Microsoft Exchange/Office 365.

Steps to generate the report:

  1. In the GroupID portal, select Reports from the left navigation bar. The Reports portal will open in a new Browser tab

  1. Click the User Reports button on the Navigation bar.
  2. Once the Users category is opened, in the search bar, type: Users multiple DL memberships.

  1. Click the report once it is visible after the search. A new window will appear.
  2. Click on Create Report to initiate the report creation wizard.

  1. On the Step ½ page, provide a Friendly name to the report, choose the search scope within the directory for the report to search the results from, and provide a filter criterion

By default, the wizard searches the Global Catalog for generating the report. You can limit this scope to a particular container. To do this:

    1. Click Browse to launch the Select Container dialog box and select the required source container.
    2. Select the Include sub-containers check box if you want to include its sub-containers when reporting.
    3. In the Filter Criteria box, modify the default LDAP filter. This filter is used for selecting items from the containers. The default filter for this report will generate a list of all the Users in the Domain along with their group memberships.
    4. Add the manager Not Present to fetch a list of all the Enabled Users without Managers, along with DL Memberships.

  1. Click Next.

  1. The Edit Report Fields page displays the fields that will be included in the report.
    1. To add more fields, click Add. This displays the Add a Field to the Report dialog box, where you can select a source field and specify a display name for it.
    2. You can also remove a field from the report by selecting it and clicking Remove.
    3. You can change the order of these fields by using the Double Bar buttons.
  1. Click Finish.
  1. A report will be generated. You can download the report in your desired format or pin the report to the Reports portal Dashboard.
