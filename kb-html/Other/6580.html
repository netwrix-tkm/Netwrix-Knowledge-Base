Create Host list parent container

Summary: Create Host list parent container with existing or newly created empty child host list

Issue: Currently in the StealthAUDIT console GUI there is no way to create a custom container to use to organize host lists under Host Management. Fortunately this can be achieved using SQL scripting.

Instructions: In its simplest form, a parent can be added as follows:

UPDATE HostListsTbl
set PlacementPath='My New Parent List'
WHERE Name='My Child List' AND SA_Node='SAHOSTNAME'
