Manual Removal of Threat Prevention Windows Agent

Summary: How to manually remove a Threat Prevention Windows Agent

Issue: steps to manually remove Threat Prevention Windows agent installation when the Windows agent entry is not in add / remove programs or program features

Instructions:  
To manually remove the Threat Prevention Windows agent:

  • step 1 -  open an administrator command prompt and execute:
    • sc delete SIWindowsAgent
    • sc delete SIGPOSvc
    • sc delete SBTService
  • step 2 - verify in the operating system's service list Windows agent, Windows File Monitoring, and Group Policy services are removed
  • step 3 - delete Netwrix folder at :\program files(x86)\STEALTHbits
  • step 4 - remove registry keys
    • remove registry keys at regedit hkey_local_machine | software | Netwrix
    • remove registry keys at regedit hkey_local_machine | software | wow6432node | Threat Prevention
