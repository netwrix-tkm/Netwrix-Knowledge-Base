Hotfix: SA_9.0_040

Summary: Address flaws with modern auth scans

Issue: Summary:  Address flaws with modern auth scans
Before Fix: During scans the authentication token can expire between the connection and query which will result in us failing to capture information around whatever we were querying at that time, resulting in 401 access denied errors.
After fix: If token expires mid-scan, SPAA will re-authenticate to avoid getting access denied errors
