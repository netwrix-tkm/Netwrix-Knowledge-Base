How to enable advanced Logging

IMPORTANT: This is a work-in-progress article. The product R&D team has yet to review some or all of the information below.

Question

How to enable advanced Logging in PingCastle?

Answer

To enable advanced Logging, please follow the below steps:

  • Navigate to the PingCastle directory and update the logging section in the app.settings.json with the following configuration:

"Logging": {
   "IncludeScopes": false,
   "LogLevel": {
     "Default": "Debug",
     "System": "Information",
     "Microsoft": "Information"
   }
}

  • Update the web.config XML so the aspNetCore tag has stdoutLogEnabled=true

  • Perform an IIS reset to update the configuration

In the PingCastle directory, you will find a folder named "logs," which will contain a TXT file with the following details.

IMPORTANT: The file will appear once any action will be performed in PingCastle.
