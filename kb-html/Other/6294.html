Hotfix: SIWinConsole_SIEntManager_SIWindowsAgent_7.2.0.222

Summary:

NTDS file lockdown behavior change

Behavior before Fix:

SI has no Option to Disable or Enable logging or blocking of NTDS.dit reads by System account

Behavior after Fix:

New global setting 'Ignore SYSTEM account for NTDS.DIT file' for 'File monitor settings' dialog in Configuration menu:

  • Disabled: default behavior
  • Enabled: do not block (or log) operations for this file
