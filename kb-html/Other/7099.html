Hotfix: SIEntManager_SIWinConsole_4.1.0.672

Summary: Exchange rule invalid if policy has empty collection

Issue: Summary:  Exchange rule invalid if policy has empty collection
Before fix:  Error   | 0x7700| Failed to initialize rule engineFailed to parse the When condition of the rule 'v2_block personal mailbox delegation_186'. Error parsing token ). No parameters specified
After fix:
Gather information about problems during rule generation.
 If problems detected report warning in winconsole and em
 Remove invalid entries from final exchange rules
Note:
UI warning in console appears during policy save. Rule generation during save is based on cached collection that might be empty if EM was recently restarted, thus warning UI warning can be not shown during save. But in EM logs during agent policy update warning should be written if some data is not correct.
