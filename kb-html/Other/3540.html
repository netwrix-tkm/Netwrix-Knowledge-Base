Automatic user enrollment failed: Not found (404)

The following error occurs after installation of Password Manager.
The error occurs because the client cannot connect to the Password Manager portals by the specified URL. The 404 error code means that there are no portals at the specified URL.
Reconfigure the client to connect to the server via the correct URL (should point to Password Manager Self-service portal) The URL is determined by the PRM_server registry value located in the following registry keys (by priority)
  1. HKLMSoftware[Wow6432Node]PoliciesNetWrixPassword Manager
  2. HKLMSoftware[Wow6432Node]NetWrixPassword Manager Self-Service client (is used if there is no value in the 1st key)
