Migrating NRAD v2.6

IMPORTANT: This is a work-in-progress article. The product R&D team has yet to review some or all of the information below.

Question

How do I migrate Netwrix Recover for Active Directory v2.6 to a new server?

Answer

To migrate NRAD v2.6 to a new server, please do the following:

  1. Get the location of the NRAD Database (SQLSERVER & DB name)
    • OPTIONAL: Take screenshots of each configuration tabs in the Web Console
  2. Download and install NRAD v2.6 on the new server (verify the file is unblocked before installing)
  3. During the install, connect to the previous NRAD Database
  4. After the reinstall is complete, open the Web Console
    • OPTIONAL: If screenshots were taken, compare them to the new configuration tabs
