How to learn which of devices were audited on which days (Gen 7 R2)

Question

How does Gen 7 R2 event record work?

Answer

Netwrix Change Tracker Gen 7 R2 records an Audit Event for every day a device is active.
Each new event is incremented by 1  in other words, for every agent seen active in any day period, an event will be recorded with one added to the Licensed Day Usage count.

Examples

  • System with 1 agent only, active every day for a 30-day period, will record 30 Licensed Day Usage Audit Events, numbered 1 to 30.
  • Alternatively, a system with 25 devices, where all devices have been active in each day period for a ten-day period, will record 10 Licensed Day Usage events for each device, the last of these events will therefore be numbered 250.
To assess the total number of Licensed Day Usage events in any period, run a query for the period, with a text search for Licensed for Audit Events.
The actual consumption of Licensed Days during the period will the difference between the last and first events number.
