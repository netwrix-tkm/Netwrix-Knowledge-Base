How to enable Baseline Events on monitored folders and what they are?

In this example, I will be using the FIM - File Integrity Monitoring Tracker. All paths listed as a tracked folder within a configuration template will be "baselined". 

Enabling the Send baseline events option on the FIM tracker (or any tracker which supports it) will effectively send back an event from the agent to the hub when a baseline scan is complete and it will allow you to see a list of the files and folders that have been captured as part of the baseline and the state of the file/folder i.e. its attributes.
