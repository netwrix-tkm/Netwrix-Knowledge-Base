How To Uninstall Change Tracker Agent via PowerShell Script

Question

This article explains how to uninstall Change Tracker Agent via PowerShell Script.

Answer

To uninstal Change Tracker Agent via PowerShell script. do the following, depending on your installation type:

  • For .NetCore-based installation, navigate to C:\Program Files\NNT Change Tracker Suite\Gen7Agent (NetCore)\unins000.exe

  • For non. NetCore installations the script is located here: C:\Program Files\NNT Change Tracker Suite\Gen7Agent\unins000.exe.

The script itself would be as follows:

.\unins000.exe /SP-- /VERYSILENT /SUPPRESSMSGBOXES

NOTE: Log Tracker has no Silent Uninstall feature, as of yet.
