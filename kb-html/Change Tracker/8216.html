HOTFIX - Hub Build .56 no longer allows me to schedule planned changes or put proxy devices in diagnostic mode

Upon upgrading to 7.0.1.56 - some functionality of the Planned Changes tab and Agents and Devices tab in the Settings Tile do not work properly. This article will show you how to implement a simple hot-fix in order to restore functionality.

Step-by-step guide

Here is what you need to do

  1. Stop IIS
  2. Stop the MongoDB Service
  3. Navigate toC:\inetpub\wwwroot\Change Tracker Generation 7 WebUI\html\pages\admin

  4. Copy and Paste the files below, replacing the files in the folder
     

    File 1 -devices-admin.html
    File 2 -planned-changes.html
  5. Navigate toC:\inetpub\wwwroot\Change Tracker Generation 7 WebUI\html\js
  6. Copy and Paste the file below, replacing the files in the folder
    File 3 -nnt.changetracker.v7.js
  7. Do not proceed until all 3 files have been replaced
  8. Start the MongoDB Service
  9. Start IIS
  10. Clear your browser cache
