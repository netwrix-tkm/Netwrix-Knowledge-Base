AIC Cannot Connect to SQL Database

Symptom: AIC users other than the "admin" user even with the correct password, it seems as though they cannot access AIC.

Cause: The AIC > "Home" > "Configure Console" > "Database" > DB password has expired or is not correct.

Resolution:
1) Assumption: Customer has changed the password in Access Analyzer (new DB password) to access the Access Analyzer (Access Analyzer) database.Â 
2) Use the default "admin" user and password to log into AIC.
3) Navigate to "Home" > "Configure Console" > "Database" and type in your new DB password for your SQL account and click the "Save" button.
4) Now AIC is working again.
5) Note: Other users (other than the "admin" user) with the correct password, it seems as though they cannot access AIC when the DB password is not current or correct.
6) At this point in time it is a best practice to login to AIC as the "admin" user in order to set a new DB password.
