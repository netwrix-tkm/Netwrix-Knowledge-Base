SAM and CEE Compatibility

Summary: Some CEE Versions are not compatible with SAM.

Issue: Sometimes EMC/Dell releases a version of CEE which is not compatible with SAM. This is to highlight supported and unsupported CEE versions.

 

CEE HTTP protocol support was recently added to 5.0, 6.0.  This includes support for using remote Linux CEE Servers.
