Instructions: 

  1. Unblock the hotfix zip file in the windows property dialog, if an unblock button exists there.? 

  1. Close all instances of StealthAUDIT (check task manager under processes for all users) 

  1. Update all DC files in your %SAInstallDir%DC folder 

  1. Update all PrivateAssemblies in your %SAInstallDir%PrivateAssemblies folder. 

  1. Update all Agent files in your %SAInstallDir%Agent folder 

  1. If using Agent mode for scans, please install SPAA Agent version 11.6.0.20 or higher to avoid compatibility issues. (This is only necessary for SP On-Prem, not SharePoint Online)
