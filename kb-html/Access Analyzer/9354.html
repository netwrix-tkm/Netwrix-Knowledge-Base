Report Error: Sequence Contains More Than One Matching Element

Symptom

When running the a job in Netwrix Access Analyzer that generates a report, the report generation fails with error "Sequence contains more than one matching element".

Cause

Duplicate entries in %sainstalldir%Reports\report.xml lead to the report failing to populate as it could not be determined which field is correct.

Resolution

Remove all entries in the %sainstalldir%Reports\report.xml file for the job causing the error. Save the file and run the job to generate the report again.
