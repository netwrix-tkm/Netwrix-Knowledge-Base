Overview

After completing Steps 1-3, please verify the following:

Validating the Netwrix Access Analyzer Console Migration

  • The Job Tree contains all expected jobs.

  • Verify Tasks under Schedules have been carried over.

  • If using Role Based Access, verify the Access tab under Settings contains users.

  • Validate the Access Information Center launches.

  • Validate Published Reports web server launches.

  • Validate reports are published and available in the Published Reports site.

  • Open .Active Directory Inventory\1-AD_Scan\Status\JobStats in NAA and verify the SA_HOST column contains the new NAA Console server name.
