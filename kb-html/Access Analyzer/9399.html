1) It looks like you are licensed for the Netwrix Access Analyzer Windows Solution which should contain a job that can help you identify where IS Local Administrators in a member of the local admins group.

2) In the Access Analyzer Console, expand the following: Windows > Privileged Accounts > SG_LocalAdmins

Image_2025-01-08_13-56-56.png

 

3) By default, this targets the ALL WINDOWS SERVERS (NO DC) Host List

Image_2025-01-08_13-56-02.png

 



4) Run the job and the information you're looking for should be found under the Results node beneath the job.

Image_2025-01-08_13-58-30.png
