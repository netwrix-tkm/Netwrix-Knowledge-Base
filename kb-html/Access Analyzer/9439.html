activitymonitor-8.0.142

Title:

activitymonitor-8.0.142

Summary:

  1. Qumulo. Monitoring service consumes 99% of RAM
  2. SUVP 25-03 support

Behavior Before Fix:

  1. MonitorService.exe's memory usage increased to the point of exhausting all available RAM when monitoring Qumulo under a heavy workload. Disabling of the MS Office filtering helped to alleviate the problem.
  2.  
  • 2025 Server - KB5053598
    • Kerberos events will be lost or incomplete. 
    • Cases with jump-boxes will no longer be handled correctly
    • Logs: 
      • Couldn't resolve I_GetASTicket
      • Couldn't resolve CConnectionEx::InitializeClientData (4 param)

  • 2022 Server - KB5053603
    • LDAP Bind and Kerberos events will be lost or incomplete
    • Logs: 
      • Couldn't resolve LDAP_CONN::BindRequest
      • Couldn't resolve HandleTGSRequest
      • Couldn't resolve I_GetASTicket
  • 2019 Server - KB5053596
    • LDAP Bind and Kerberos events will be lost or incomplete
    • Logs: 
      • Couldn't resolve LDAP_CONN::BindRequest
      • Couldn't resolve I_GetASTicket
  • 2016 Server - KB5053594
    • No impact

  • 2012R2 Server - KB5053887
    • Logs: Couldn't resolve LDAP_CONN::BindRequest

Behavior After Fix:

  1. File system event processing has been updated to efficiently handle Qumulo-specific activity patterns both with and without the MS Office filtering. Office filtering has been optimized across all file systems. Temp file filtering has been moved from Path Filters to a more efficient internal process.
  2. AD Module updated. Product should report active directory events as expected.

Changes Since Last Hotfix:

  • 378042: Qumulo: Empty path for the Share Read operations over NFS
  • 376947: Qumulo: Wrong share path in Share Add event
  • 377244: Qumulo: %ORIGINATING_SERVER% macro does not report host name, %ORIGINATING_SERVER_IP% does not report node's IP
  • 376557: Qumulo: Syslog events are lost
  • 378189: Qumulo: Server in the UNC path does not match the Host name
  • 373970: OfficeFilter, Performance issue
  • 376043: Path Filters optimization (NAS)
