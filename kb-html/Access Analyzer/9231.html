Built-in Users Group Permissions

Question

During the security assessment in the Netwrix Enterprise Auditor environment, the use of the Users group (S-1-5-32-545) was noted for Enterprise Auditor services and folders. Does Enterprise Auditor require the use of the Users group?

Answer

The use of the Users group is not requiredEnterprise Auditor implements the Users group by default. You can modify the permissions for the installation folder to exclude the group. Refer to the following path for the default installation folder for Enterprise Auditor:

C:\Program Files (x86)\STEALTHbits\StealthAUDIT\

Refer to the following article for additional information on Role Based Access to the Enterprise Auditor console: Access  Role Based Access · v11.6.
