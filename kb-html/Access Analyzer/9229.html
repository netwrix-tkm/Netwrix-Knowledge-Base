Tracking Access Analyzer Console Logons

Question

Does Netwrix Access Analyzer track console logons and logoffs?

Answer

Access Analyzer logs the events in the dedicated Access Analyzer event log located in the following path:

%SystemRoot%\System32\Winevt\Logs\StealthAUDIT.evtx

Review the Event IDs 1000 and 1003 to learn more about accounts accessing the Access Analyzer console.
