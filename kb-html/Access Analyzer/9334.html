Title:

activitymonitor-7.1.221

Summary:

  1. NAM reporting wrong user domain
  2. Not Recording Events for NFS Mount  Several Other servers are working new one is not.

Behavior Before Fix:

  1. In some environments when netbios and fqdn domain name differ, we report the wrong user domain.
  2. For Linux samba / smb activity we were reporting the path incorrectly.

Behavior After Fix:

  1. Implemented fix to always report the netbios domain name.
  2. Implemented several improvements in the logic how we process the events from auditd in order to have a better caching of path names and reconstruct correct full path based on the information we get from auditd.  Reporting of samba / smb paths should work as expected now.
