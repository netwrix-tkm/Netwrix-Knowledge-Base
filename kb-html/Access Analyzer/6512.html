Issue: Explanation and fix for "I/O Error 112"

FSAA may throw "Exception Processing Message 0xc000000r Parameters ..." error as well

Instructions: In the past, especially in large environments, observed the following error message:
 
"Exception EInOutError in module STEALTHAUDIT.exe at 00096530 I/O error 112"
 
Or simply
 
"I/O error 112"
 
Screen shots are shown below.
 
When you encounter an error like this (a 1-4 digit error code), it's sometimes useful to consult the following articles:
http://msdn.microsoft.com/en-us/library/ms681382.aspx (Error codes 0-499)
http://msdn.microsoft.com/en-us/library/ms681388(v=VS.85).aspx (Error codes 500-999)
Further links to error codes are available through these links.
 
Examining the error 112, we find that the problem is
 
ERROR_DISK_FULL 112 (0x70) There is not enough space on the disk.
 
Freeing up more disk space seems to be the solution for this problem.
