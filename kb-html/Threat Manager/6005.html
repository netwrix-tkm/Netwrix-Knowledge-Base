Update Ransomware Extension List with No Internet

Question

How to update the ransomware extension list without internet connection in Netwrix Threat Manager (formerly StealthDEFEND)?

Answer

Netwrix Threat Manager v2.7

  1. Run the following PowerShell script: Ransomware.ps1.

  2. Copy the output resp.json file to the following directory:

    C:\Program Files\STEALTHbits\StealthDEFEND\JobService\Python\pydefend\pydefend\data

Netwrix Threat Manager v2.7 SP1

  1. Save or copy the raw data from the following link to a file named filters.json: Anti-Ransomware File System Resource Manager Lists 路 Experiant 🡥.

  2. Copy the filters.json file to the following directory:

    C:\Program Files\STEALTHbits\StealthDEFEND\JobService\Python\pydefend\pydefend\data

  3. Increment the version number in the filters.metadata file in pydefend\data for old files to be reprocessed.

Netwrix Threat Manager v2.8

  1. Copy the raw data from the following link to a file named filters.json: CryptoBlocker ? Known Extensions 路 GitHub 🡥.

  2. Copy the filters.json file to the following directory:

    C:\Program Files\STEALTHbits\StealthDEFEND\JobService\Python\pydefend\pydefend\data

  3. Increment the version number in the filters.metadata file in pydefend\data for old files to be reprocessed.
