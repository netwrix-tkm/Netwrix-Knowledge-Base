DEFEND: Installing a new License file

Summary: Installing a new License file for Threat Manager

 

Issue: How to install a new License file in Threat Manager 2.6+

 

Instructions:  

1. Check to see if the license file is blocked (Right click >Properties>check unblock if blocked).

2.  Then within the Threat Manager UI select System Settings.

3. Then select Licensing.

4. Then click Browse and select the location of the new license file.
